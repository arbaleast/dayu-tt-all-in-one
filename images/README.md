1. 打开视频
2. 暂停到关键画面
3. 按 `Ctrl+Shift+S` (Windows) 或 `Cmd+Shift+4` (Mac) 截图
4. 保存到对应目录

## 图片格式建议

| 用途 | 推荐格式 | 原因 |
|-----|---------|------|
| 照片 | JPG | 压缩率高，文件小 |
| 截图 | PNG | 保留清晰度 |
| 透明背景 | PNG | 支持透明 |
| 动画 | GIF | 展示过程 |

## 图片尺寸建议

- 文档插图：宽度800-1200px
- 缩略图：宽度300-400px
- 高清展示：原始分辨率
