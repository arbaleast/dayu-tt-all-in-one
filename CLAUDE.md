# 大鱼TT — AI 协作协议

## 协议目的

本协议规范 AI Agent 在大鱼TT项目中的协作方式，确保文档质量和一致性。

## 发起任务

1. 明确任务目标和验收标准
2. 遵循 SUPERPOWERS.md 工作流程
3. 重大变更需经 SPEC 文档审批

## 交付流程

1. 完成实现或文档更新
2. 更新对应 DOMAIN.md 索引
3. 提交 git commit
4. 通知相关 Agent

## 冲突处理

1. **同领域冲突**：领域负责人决策
2. **跨领域冲突**：相关领域负责人协商
3. **无法达成共识**：提交仲裁（提利昂·兰尼斯特）

## 文档质量标准

- 引用具体文档来源
- 量化指标需标注数据来源
- 配置项需标注适用条件
- 更正旧文档需注明变更原因
