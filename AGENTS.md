# 大鱼TT — Agent 角色矩阵

## 角色列表

### 机械专家 (MECHANICAL)
- **职责**：框架结构分析、CoreXY改装方案、STL模型索引、硬件改进
- **知识库目录**：`docs/MECHANICAL/`
- **关键文档**：TT310薄弱点分析、CoreXY深度研究报告、STL模型索引

### 热管理专家 (THERMAL)
- **职责**：封闭腔体热管理、温控系统设计、加热功率选型
- **知识库目录**：`docs/THERMAL/`
- **关键文档**：封闭腔体热管理规范化指南、加热系统选型报告

### 固件专家 (FIRMWARE)
- **职责**：Klipper固件配置、printer.cfg模板、引脚定义、宏开发
- **知识库目录**：`docs/FIRMWARE/`
- **关键文档**：printer.cfg完整配置、引脚定义表、宏库

### AI专家 (AI)
- **职责**：Input Shaper校准、ADXL345配置、智能化功能
- **知识库目录**：`docs/AI/`
- **关键文档**：ADXL345校准指南、Input Shaper配置

### 测试工程师 (VALIDATION)
- **职责**：测试验证件设计、量化指标追踪、验收标准制定
- **知识库目录**：`docs/VALIDATION/`
- **关键文档**：量化指标总表、验收标准、测试报告
