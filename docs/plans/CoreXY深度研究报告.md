# CoreXY 结构 3D 打印机深度学术研究报告

**目标机型：310×310×310mm | 打印材料：PLA/PETG/ABS**

**作者：AI 研究助手 | 日期：2026-03-29**

---

## 摘要

CoreXY 是一种广泛应用于高速 3D 打印机的皮带驱动笛卡尔运动架构。本报告对 310×310×310mm 打印体积的 CoreXY 打印机进行全面学术研究，涵盖运动学原理、皮带张力优化、框架刚性选型、振动控制、热管理、固件参数优化及最新改进方案七大领域。

---

## 一、CoreXY 结构原理与设计要点

### 1.1 运动学原理

CoreXY 运动系统是一种耦合式笛卡尔 X-Y 机构，其核心原理基于两条交叉皮带通过两个静止电机的协同运动来确定打印头的 X 和 Y 位置。

当两个电机同向旋转时，打印头沿 Y 轴移动；当两个电机反向旋转时，打印头沿 X 轴移动。固件将位置映射为 `A = X + Y` 和 `B = X - Y`。

### 1.2 CoreXY vs H-bot

CoreXY 与 H-bot 的关键区别在于电机位置。在 CoreXY 中，所有电机都是静止的，X 和 Y 轴由皮带耦合共同管理。H-bot 结构虽然也使用交叉皮带，但电机通常安装在框架两侧，皮带直接连接活动滑架。

### 1.3 设计要点（310mm 机型）

- **皮带路径：** 标准 CoreXY 皮带路由采用 45 度角交叉排列，产生 √2 的减速比
- **皮带选型：** GT2 皮带，齿距 2mm，宽度 6mm
- **滑架设计：** 最小移动质量，同时保持足够刚性和热稳定性
- **对角线耦合：** 固件必须正确配置电机方向

## 二、皮带张力计算与惰轮排列优化

### 2.1 皮带张力理论

CoreXY 皮带的张力调整目标是使 X 和 Y 轴相互垂直，而非两条皮带的张力相等。皮带张力测量通常使用频率法：通过敲击皮带段并测量其振动频率来推算张力。标准 GT2 皮带的推荐张力范围为 100-150 Hz 振动频率。

### 2.2 惰轮排列优化

**中段惰轮（Mid-Span Idlers）：** 在长皮带路径中点添加惰轮可以减少皮带颤振，提高高速运动稳定性。

**惰轮直径：** 惰轮直径应与驱动皮带轮直径匹配，以保持皮带段平行。

### 2.3 推荐惰轮排列（310mm 机型）

1. 驱动轮位于框架底部两侧，对称布置
2. 上部惰轮位于框架顶部，与驱动轮垂直对齐
3. 滑架皮带夹位于滑架中心，确保受力平衡
4. 可选配中段惰轮用于长皮带路径（>200mm）

## 三、框架刚性与铝型材选型

### 3.1 2020 vs 2040

| 型材 | 截面 | 弯曲刚度 | 适用位置 |
|------|------|----------|----------|
| 2020 | 20×20mm | 基准 | 顶部框架、Y轴支撑 |
| 2040 | 20×40mm | 2倍于2020 | 底部框架、Z轴支柱、X轴导轨 |

### 3.2 推荐配置（310mm 机型）

| 位置 | 推荐型材 | 理由 |
|------|---------|------|
| 底部框架 | 2040 | 承受热床和打印件重量 |
| 垂直支柱 | 2040 或 2020 | 高度>300mm建议2040 |
| 顶部框架 | 2020 | 主要是连接作用 |
| X轴导轨支撑 | 2040 | 抵抗运动惯性 |

## 四、高速打印振动控制与吸音设计

### 4.1 振动来源

1. **电机振动** — 步进电机高速启停时产生
2. **皮带颤振** — 皮带高速运动时的弹性振动
3. **框架共振** — 铝型材框架固有频率
4. **桌面传递** — 振动通过安装表面传递

### 4.2 振动控制策略

- **电机减振器：** 将步进电机安装在减振器上可显著降低振动传递
- **阻尼材料：** 在框架接触点使用邵氏硬度 A50-70 的橡胶或硅胶垫片
- **质量增加：** 增加打印机质量可以降低共振频率

### 4.3 吸音材料

| 材料 | 效果 | 成本 |
|------|------|------|
| Sonex 泡沫 | 优秀 | 高 |
| EVA 泡沫 | 良好 | 中 |
| 毛毡 | 中频效果好 | 低 |

### 4.4 抗振脚方案

- **Squash Ball 减振脚：** 壁球作为减振元件，高度约 30mm
- **TPU 柔性脚：** TPU 弹性材料打印，填充率 15-50%
- **弹簧减振：** 金属弹簧配合橡胶垫圈

## 五、热管理：BOSSAC 热床 + 散热风道

### 5.1 BOSSAC 热床

BOSSAC（Board on Spring Aluminum Ceramic）将 PCB 加热电路直接结合在铝基板上，优势：
- 热传导路径更短，效率更高
- 加热均匀性更好
- 厚度更薄，热容量更小

### 5.2 310mm 热床选型

| 参数 | 推荐值 |
|------|--------|
| 尺寸 | 310×310mm |
| 电压 | 24V |
| 功率密度 | 0.5-0.8 W/cm² |
| 功率 | 400-600W |
| 表面涂层 | 碳化硅陶瓷或 PEI |

### 5.3 热管理设计

**保温层：** 添加保温棉可缩短加热时间 30-50%，降低能耗。

**散热风道：**
- 进风口位于机器底部
- 出风口位于机器顶部
- 避免气流直吹打印件

**安全：** SSR 故障通常表现为"常开"，必须配置独立热失控保护电路。

## 六、Klipper 固件优化

### 6.1 加速度设置

| 应用 | 加速度（mm/s²）|
|------|---------------|
| 一般打印 | 2000-3000 |
| 高速打印 | 3000-5000 |
| 填充打印 | 4000-8000 |
| 空驶移动 | 可达 10000+ |

### 6.2 Square Corner Velocity

`square_corner_velocity`（SCV）控制打印头在拐角处的速度过渡。默认值 5 mm/s，建议将所有 jerk 值设为等于 printer.cfg 中的 SCV 值。

### 6.3 输入整形（Input Shaping）

**实施步骤：**
```bash
# 测量共振频率
TEST_RESONANCES AXIS=X OUTPUT=raw_data
TEST_RESONANCES AXIS=Y OUTPUT=raw_data
# 自动校准
SHAPER_CALIBRATE
```

**典型共振频率：**
- X 轴：35-50 Hz
- Y 轴：30-45 Hz

### 6.4 推荐 Klipper 参数（310mm CoreXY）

```ini
[input_shaper]
shaper_type_x: mzvd
shaper_freq_x: 45
shaper_type_y: mzvd
shaper_freq_y: 38

[velocity_limit]
max_velocity: 300
max_accel: 5000
max_accel_to_decel: 3500
square_corner_velocity: 5
```

## 九、改进优先级排序

### P0 - 关键基础（必须实施）

## 十二、风险提示

1. **AC 热床安全：** 必须配置独立热失控保护
2. **皮带初期伸长：** 新皮带使用 100 小时后需重新张紧
3. **输入整形过度：** 过激的加速度设置可能导致丢步
4. **热管理：** 封闭打印时需注意散热，防止内部温度过高
