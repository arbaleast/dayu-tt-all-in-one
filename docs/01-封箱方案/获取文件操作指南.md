# 获取开源文件操作指南

## 目标

联系UP主获取大鱼TT封箱的STL文件和其他开源资源。

## 操作步骤

### 方法一：B站评论区获取（推荐）

#### 步骤1：打开视频
1. 访问B站：https://www.bilibili.com
2. 搜索视频：《大鱼TT魔改+封箱》
3. UP主：汪汪队射手

#### 步骤2：查看评论区
1. 滚动到视频下方评论区
2. 查看**置顶评论**
3. 通常包含：
   - 百度网盘链接 + 提取码
   - 阿里云盘链接
   - GitHub仓库地址
   - QQ群号

#### 步骤3：下载文件
**如果是网盘链接：**
1. 复制链接和提取码
2. 打开对应网盘
3. 输入提取码
4. 下载所有文件到本地
5. 解压后分类整理

**如果是GitHub：**
1. 点击链接访问仓库
2. 点击绿色 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压到项目目录

### 方法二：私信UP主

#### 步骤1：进入UP主主页
1. 点击视频中的UP主头像
2. 或搜索 "汪汪队射手" 进入主页

#### 步骤2：发送私信
1. 点击 "私信" 按钮
2. 发送礼貌的请求：

```
模板消息：

您好！我是大鱼TT的用户，看了您的《大鱼TT魔改+封箱》
视频受益匪浅。想自己动手制作，能否分享一下STL文件
和材料清单？非常感谢！

我的邮箱：xxx@xxx.com
或者网盘分享都可以。

期待回复，谢谢！
```

#### 步骤3：等待回复
- 通常1-3天内回复
- 周末可能更快
- 如果没有回复，可以再次礼貌提醒

### 方法三：加入QQ群

#### 步骤1：获取群号
- 视频简介中查找
- 评论区查找
- 私信询问UP主

#### 步骤2：加入群组
1. 打开QQ
2. 搜索群号
3. 申请加入
4. 等待管理员通过

#### 步骤3：获取文件
- 查看群文件
- 向群友询问
- 通常有整理好的资源包

## 文件整理

### 下载后操作

#### 1. 创建目录结构
```bash
cd /vol1/1000/projects/3d-printing/dayu-tt-all-in-one
mkdir -p stls/汪汪队射手
mkdir -p stls/官方
mkdir -p models
```

#### 2. 分类存放
- **STL文件** → `stls/汪汪队射手/`
- **源文件** → `models/`
- **图纸** → `docs/`

#### 3. 文件命名
如果下载的文件命名混乱，建议重命名：

```
原文件名：1.stl, 2.stl, 3.stl...
重命名后：
- 固定座_X轴.stl
- 固定座_Y轴.stl
- 活动块.stl
- 张紧轮.stl
...
```

### 文件清单核对

下载后对照以下清单检查：

#### 汪汪队射手方案
- [ ] 皮带张紧器相关（4-6个STL）
- [ ] 风扇罩（1-2个STL）
- [ ] 线槽配件（2-3个STL）
- [ ] 其他小配件（若干）

#### 官方方案（如果获取到）
- [ ] 框架连接件
- [ ] 风扇罩
- [ ] 过滤系统
- [ ] 门配件

## 常见问题

### Q: 评论区没有链接？
A: 
- 链接可能被吞，尝试展开评论
- 查看UP主主页简介
- 直接私信询问

### Q: 网盘链接失效？
A:
- 在评论区提醒UP主
- 私信请求重新分享
- 在QQ群询问

### Q: 下载的文件打不开？
A:
- 检查文件完整性
- 使用正确的软件打开（如PrusaSlicer）
- 可能是压缩包，需要先解压

### Q: 缺少某些零件的STL？
A:
- 仔细查看所有文件夹
- 可能是通用件（如轴承）需要购买
- 向UP主确认
