## 装配要点

### 框架搭建
1. 先组装底部方形框架
2. 安装4根立柱
3. 安装顶部框架
4. 用角尺检查垂直度
5. 紧固所有螺丝

### 板材安装
1. 先安装背板和侧板
2. 安装顶板（可做成可开启）
3. 安装门板（带磁吸）
4. 内侧贴保温棉

### 电气安装
1. LED灯条安装在顶部
2. 风扇安装在背板或侧板
3. 温控器固定在方便操作的位置
4. 加热器安装在底部（不直接接触打印机）

## 温控设置

### ABS打印推荐参数
- 仓温：45-50°C
- 喷头：240-250°C
- 热床：100-110°C

### 温控器设置
- 目标温度：45°C
- 回差：2-3°C
- 报警温度：60°C

### 常见问题

**Q: 235和310版本怎么选？**
A: 空间允许建议310，打印空间更大；空间有限选235。

**Q: 封箱后温度控制不好？**
A: 检查保温棉是否完整，风扇是否常开，温控器参数设置。

**Q: 打印ABS还是翘边？**
A: 提高仓温到50°C，检查热床温度，使用 brim。
