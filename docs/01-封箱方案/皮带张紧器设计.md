# 大鱼TT 封箱 - 皮带张紧器设计

## 设计概述

皮带张紧器是汪汪队射手方案的核心改进，解决了大鱼TT原机皮带调节困难的问题。

## 问题分析

### 原机问题
1. 皮带张紧需要拆机
2. 调节空间狭小
3. 难以保持最佳张力
4. 长期使用后皮带松弛

### 解决方案
设计外置张紧器，无需拆机即可调节皮带张力。

## 工作原理

### 初始安装
1. 将固定座安装在铝型材上
2. 轴承装入张紧轮
3. 张紧轮安装在活动块上
4. 活动块装入固定座
5. 安装弹簧和调节螺丝

### 张力调节

#### 粗调
- 松开固定座与型材的连接
- 移动整个张紧器位置
- 锁紧固定

#### 细调
- 旋转调节螺丝
- 活动块移动，改变皮带张力
- 弹簧自动补偿张力变化

## 3D打印参数

### 固定座
- 材料：PETG（推荐）或PLA
- 填充：40-50%
- 壁厚：3-4层
- 支撑：需要

### 活动块
- 材料：PETG（推荐）或PLA
- 填充：30-40%
- 壁厚：3层
- 支撑：需要

### 张紧轮
- 材料：PETG或ABS
- 填充：50%
- 壁厚：4层
- 支撑：不需要

## 安装步骤

### 第一步：打印零件
1. 打印固定座（2个，X/Y各1）
2. 打印活动块（2个）
3. 打印张紧轮（2个）
4. 检查尺寸，清理支撑

### 第二步：组装张紧轮
1. 将625ZZ轴承压入张紧轮
2. 确保轴承安装到位
3. 检查转动顺畅

### 第三步：组装活动块
1. 将张紧轮装入活动块
2. 插入轴销（M3螺丝）
3. 确保张紧轮能自由转动

### 第四步：安装到固定座
1. 将活动块放入固定座导轨
2. 安装弹簧
3. 安装调节螺丝

### 第五步：安装到打印机
1. 确定安装位置（X/Y轴电机对面）
2. 将固定座固定在铝型材上
3. 绕过皮带
4. 调节张力

## 调节方法

### 初始张力设置
1. 用手按压皮带，应有轻微弹性
2. 皮带不应过松（跳齿）或过紧（增加阻力）
3. 打印测试，观察是否有层移

### 日常维护
- 每月检查一次皮带张力
- 发现松弛时旋转调节螺丝
- 弹簧会自动保持张力稳定

## 注意事项

1. **轴承选择**：必须使用625ZZ轴承，内径5mm
2. **材料强度**：建议使用PETG，强度和耐温性更好
3. **打印精度**：确保活动块在固定座内滑动顺畅
4. **皮带路径**：确保皮带绕过张紧轮的角度正确
5. **张力适中**：过紧会增加电机负载和轴承磨损

## 常见问题

### Q: 张紧轮不转或转动不畅？
A: 检查轴承是否安装到位，轴销是否过紧。

### Q: 调节螺丝旋不动？
A: 检查螺纹是否打印清晰，可适当润滑。

### Q: 皮带还是跳齿？
A: 增加张力，或检查皮带是否磨损需要更换。

### Q: 活动块松动？
A: 增加弹簧力度，或检查固定座导轨尺寸。

## 改进建议

1. **增加刻度**：在固定座上增加刻度，方便记录最佳位置
2. **快拆设计**：设计快拆结构，方便更换皮带
3. **指示器**：增加张力指示器，直观显示张力大小
