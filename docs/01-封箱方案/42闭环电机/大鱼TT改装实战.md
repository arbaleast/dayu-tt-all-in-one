## 改装方案设计

### 大鱼TT结构分析

```
CoreXY结构特点:
- X/Y轴共用两个电机
- 电机需要精确同步
- 皮带传动，有弹性变形
- 高速打印时易丢步

改装重点:
1. X/Y轴电机 → 闭环控制 (最重要)
2. Z轴电机 → 可选改装
3. E轴电机 → 可选改装
```

### 推荐改装方案

- S42B V2.0 × 2 (X/Y轴)

**成本**: ¥160-240

**效果**:

**适合**:

### 改装流程

#### 第一步: 备份和记录
```
1. 记录原电机线序
   - 拍照记录
   - 标记线色

2. 备份固件配置
   - printer.cfg (Klipper)
   - Configuration.h (Marlin)

3. 测试原机性能
   - 打印测试模型
   - 记录速度/质量
```

#### 第二步: 拆卸原驱动
```
X轴拆卸:
1. 断电
2. 拆下X轴电机驱动板 (A4988/TMC2209)
3. 记录驱动型号和跳线设置
4. 保留驱动板 (备用)

Y轴拆卸:
1. 同上步骤
2. 注意Y轴电机方向可能与X相反
```

#### 第三步: 安装S42B
```
X轴安装:
1. 将S42B插入X轴驱动插槽
2. 注意方向: EN引脚对齐
3. 连接电机线 (按记录线序)
4. 暂时不连接编码器线

Y轴安装:
1. 同上
2. 确认方向正确
```

#### 第四步: 安装编码器
```
磁环安装:
1. 拆下X轴电机风扇 (如有)
2. 清洁电机轴
3. 安装磁环到电机轴尾部
4. 确保水平，不偏心
5. 固定磁环 (胶水/螺丝)

编码器板安装:
1. 固定编码器板到电机
2. 调整间隙至1-3mm
3. 连接编码器线到S42B
4. 同样安装Y轴
```

#### 第五步: 初次上电测试
```
1. 检查所有连接
2. 通电 (不要连接主板)
3. S42B应显示启动画面
4. 进入菜单，执行校准
   - 选择 "Cal"
   - 观察电机正反转
   - 确认无抖动
5. 如有问题，断电检查
```

#### 第六步: 连接主板测试
```
1. 连接S42B到主板
2. 通电
3. 使用主板控制:
   - Klipper: STEPPER_BUZZ STEPPER=stepper_x
   - Marlin: 手动移动菜单
4. 观察电机运行
5. 确认方向正确
```

#### 第七步: 固件配置

**Klipper配置**:
```ini
# 基本不需要修改，S42B兼容标准驱动
[stepper_x]
step_pin: PB13
dir_pin: PB12
enable_pin: !PB14
rotation_distance: 40
microsteps: 16
full_steps_per_rotation: 200

# 可以优化参数
homing_speed: 60          # 可以提高归位速度
second_homing_speed: 10   # 第二次归位速度
```

**Marlin配置**:
```cpp
// 驱动类型
#define X_DRIVER_TYPE TMC2209_STANDALONE
#define Y_DRIVER_TYPE TMC2209_STANDALONE

// 电流设置
#define X_CURRENT 1000
#define Y_CURRENT 1000

// 细分
#define MICROSTEP_MODES {16, 16, 16, 16, 16, 16}
```

#### 第八步: 打印测试
```
测试模型:
1. 20mm立方体 - 检查尺寸精度
2. 螺旋花瓶 - 检查Z轴同步
3. 速度塔 - 测试最高速度
4. 悬垂测试 - 检查挤出稳定性

测试速度:
- 原速度: 60mm/s
- 测试速度: 80, 100, 120, 150mm/s
- 记录最佳速度
```

### 优化调整

#### 速度优化
```
逐步提高速度:
1. 从60mm/s开始
2. 每次增加20mm/s
3. 观察打印质量
4. 找到最佳速度点

大鱼TT推荐速度:
- 普通打印: 80-100mm/s
- 高质量: 60-80mm/s
- 快速打印: 120-150mm/s
```

#### 加速度优化
```
Klipper配置:
[printer]
max_accel: 3000          # 原机可能1500-2000
max_accel_to_decel: 1500

逐步测试:
1. 从2000开始
2. 每次增加500
3. 观察是否有丢步
4. 闭环电机可以承受更高加速度
```

#### 噪音优化
```
1. 使用32细分
   - S42B菜单设置
   - 或固件配置

2. 添加减震垫
   - 电机和框架之间
   - 减少振动传递

3. 优化电流
   - 不要设置过高
   - 够用即可

4. 检查机械部件
   - 皮带张力适中
   - 导轨润滑良好
```

## 常见问题解决

### 问题1: 改装后打印尺寸不准
```
原因: 细分设置改变

解决:
1. 检查S42B细分设置
2. 计算正确的rotation_distance
3. 重新校准挤出机

计算公式:
rotation_distance = (360° / 1.8°) / 微步数 × 皮带轮齿数 × 齿距
```

### 问题2: 电机运行时有异响
```
原因分析:
1. 磁环偏心 → 重新安装
2. 电流过高 → 降低电流
3. 细分过低 → 提高细分
4. 机械共振 → 添加减震

解决步骤:
1. 检查磁环安装
2. 调整电流至1.0-1.2A
3. 设置32细分
4. 添加电机减震垫
```

### 问题3: 偶尔出现层移
```
原因分析:
1. Z轴问题 (如果改装了Z轴)
2. 挤出机问题
3. 模型切片问题

解决:
1. 检查Z轴丝杆
2. 检查挤出机齿轮
3. 重新切片模型
```

### 问题4: S42B显示异常
```
现象: 花屏、不显示

解决:
1. 检查供电电压 (12-24V)
2. 检查电源功率是否足够
3. 检查接线是否松动
4. 重启S42B
```
