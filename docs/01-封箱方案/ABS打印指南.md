# ABS打印指南

## 为什么需要封箱打印ABS？

ABS材料在打印时容易翘边和开裂，主要原因是：
- 冷却过快导致收缩不均
- 层间结合力不足
- 需要恒温环境

封箱可以将仓温维持在40-50°C，有效解决这些问题。

## 推荐打印参数

### 温度设置

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| 喷头温度 | 240-260°C | 根据材料调整 |
| 热床温度 | 100-110°C | 必须高温 |
| 仓温 | 40-50°C | 封箱核心目的 |
| 首层温度 | +5-10°C | 提高附着力 |

### 速度设置

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| 打印速度 | 30-50mm/s | 比PLA慢 |
| 首层速度 | 15-20mm/s | 确保附着 |
| 空驶速度 | 100-150mm/s | 正常 |
| 回抽速度 | 30-40mm/s | 减少拉丝 |

### 风扇设置

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| 首层风扇 | 0% | 不关风扇 |
| 正常风扇 | 0-30% | 低风速或关闭 |
| 桥接风扇 | 50-100% | 需要时开启 |

### 其他参数

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| 层高 | 0.2-0.3mm | 标准层高 |
| 壁厚 | 3-4层 | 保证强度 |
| 填充密度 | 20-40% | 根据需求 |
| 回抽距离 | 4-6mm | 减少拉丝 |

## 首层附着技巧

### 1. 热床准备
- 清洁热床（酒精擦拭）
- 涂胶棒/发胶/ABS浆
- 调平要精确（0.1-0.15mm间隙）

### 2. 首层设置
- 速度降至15-20mm/s
- 温度提高5-10°C
- 关闭风扇
- 使用Brim（边缘）

### 3. Brim设置
- 圈数：8-12圈
- 距离模型：0mm
- 易于拆除

## 常见问题解决

### 翘边
**原因**：
- 热床温度不够
- 首层附着不良
- 冷却过快

**解决**：
- [ ] 提高热床温度到110°C
- [ ] 使用Brim或Raft
- [ ] 涂胶棒或ABS浆
- [ ] 检查调平
- [ ] 关闭风扇

### 层间开裂
**原因**：
- 喷头温度太低
- 层间冷却过快
- 打印速度太快

**解决**：
- [ ] 提高喷头温度到250-260°C
- [ ] 降低打印速度到30mm/s
- [ ] 提高仓温到50°C
- [ ] 增加层厚

### 拉丝/渗料
**原因**：
- 温度过高
- 回抽设置不当

**解决**：
- [ ] 降低温度5-10°C
- [ ] 增加回抽距离到5-6mm
- [ ] 增加回抽速度
- [ ] 开启滑行（coasting）

### 表面粗糙
**原因**：
- 温度波动
- 材料受潮

**解决**：
- [ ] 稳定仓温
- [ ] 烘干材料（80°C 4小时）
- [ ] 降低打印速度

## ABS浆制作方法

用于提高首层附着力：

### 材料
- ABS废料或支撑
- 丙酮（或乙酸乙酯）

### 制作步骤
1. 将ABS废料剪碎
2. 放入玻璃罐
3. 倒入丙酮（刚好淹没）
4. 密封静置24小时
5. 搅拌成糊状

### 使用方法
1. 热床加热到60°C
2. 用纸巾蘸取ABS浆
3. 均匀涂抹在热床上
4. 等待干燥（变白色）
5. 开始打印

## 材料烘干

ABS容易吸湿，打印前需要烘干：

| 方法 | 温度 | 时间 | 备注 |
|-----|------|------|------|
| 烘料机 | 80°C | 4-6小时 | 最佳 |
| 烤箱 | 80°C | 4小时 | 注意通风 |
| 热床 | 80°C |  overnight | 应急用 |

## 安全注意事项

### 通风
- ABS打印会产生气味
- 确保房间通风良好
- 建议安装活性炭过滤

### 高温
- 热床温度100°C+，小心烫伤
- 喷头温度240°C+，绝对不要触碰
- 打印完成后等待冷却

### 材料
- 丙酮易燃，远离火源
- 在通风处使用
- 密封保存

## 测试打印

首次使用ABS，建议先打印测试件：

1. **温度塔** - 找到最佳温度
2. **桥接测试** - 测试桥接性能
3. **悬垂测试** - 测试悬垂角度
4. **尺寸精度测试** - 校准尺寸
