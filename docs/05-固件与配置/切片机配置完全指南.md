## 二、大鱼TT 基础参数

### 2.1 机器规格

| 参数 | 235mm 版 | 310mm 版 |
|------|---------|---------|
| 结构 | CoreXY | CoreXY |
| 打印尺寸 | 235×235mm | 310×310mm |
| 热床螺丝 | M6 | M8 |
| Z 轴 | 单轴 或 三轴（改装）| 同左 |
| 喷嘴直径 | 0.4mm（默认）| 同左 |
| 最大热端温度 | 300°C | 同左 |
| 最大热床温度 | 120°C | 同左 |

### 2.2 OrcaSlicer 机器设置

1. 打开 OrcaSlicer → **打印机** → **添加打印机** → 选择 **Custom Printer**
2. 命名：`大鱼TT-235` 或 `大鱼TT-310`
3. 机器设置参数：

```ini
# 运动参数
旅行速度: 150 mm/s
打印速度参考值: 60 mm/s

# 归位
X/Y 归位: 左后角（min_x, min_y）
Z 归位: 顶部（max_z）

# 搭梯
启用: 是
搭梯宽度: 3 mm
搭梯高度: 1 mm

# 尺寸
床尺寸 X: 235 / Y: 235
打印高度: 250 / 310
起点 X: 0 / Y: 0
起点 Z: 0
喷嘴直径: 0.4 mm
```

## 三、OrcaSlicer 优化设置

### 3.1 内核速度配置

```ini
[速度]
默认打印速度: 60 mm/s
默认移动速度: 150 mm/s
默认归位速度: 20 mm/s
Z 抬升速度: 12 mm/s

[加速度]
默认打印加速度: 1500 mm/s²   # 大鱼TT 推荐范围 1500-2500
默认旅行加速度: 3000 mm/s²   # 空程可更高
默认归位加速度: 1000 mm/s²
```

> ⚠️ 使用 Klipper 时，加速度在切片软件和 printer.cfg 两处设置，以 printer.cfg 为最终生效值。切片软件的值仅作为 G-code 输出参考。

### 3.2 推荐开启的优化选项

| 选项 | 推荐 | 原因 |
|------|------|------|
| **间隙填充（Gap Fill）** | ✅ 开启 | 减少缩痕，提升表面质量 |
| **压力平滑（Pressure Equalization）** | ✅ 开启 | Klipper 机器推荐，开启后自动与 PA 协同 |
| **按层高度重新分段** | ❌ 关闭 | 开启会降低精度，除非你知道自己在做什么 |
| **有机支撑（Tree Support）** | ✅ 开启 | 相比柱状支撑更节省材料，易于拆除 |
| **减少过挤（Ironing）** | ✅ 开启 | PLA 建议 -1% 到 -2%，改善表面光泽 |
| **间隙闭合（ Brim ）** | ✅ 开启 | ABS 建议 Brim 宽度 8mm+ |

### 3.3 首层优化

首层是打印成功的基础：

```ini
首层:
  打印速度: 50%（正常速度的一半）
  首层高度: 0.2mm（或层高的 200%）
  首层收缩补偿: 开启

热床粘附:
  类型: Brim（边缘）— 大多数情况选这个
  Brim 宽度: 5-8mm（ABS 建议 8mm）
  Raft: 仅大面积 / ABS / ASA 使用
```

## 四、耗材参数模板

### 4.1 PLA 参数

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 喷嘴温度 | 200-215°C | 主流 PLA 通常 205°C 最佳 |
| 热床温度 | 50-60°C | 根据耗材品牌调整 |
| 打印速度 | 50-80 mm/s | 质量优先选 50-60，高速可选 80+ |
| 冷却风扇 | 100%（第二层起）| PLA 必须充分冷却 |
| 收缩补偿 | ~0.4%（XY）/ ~0.2%（Z）| 依实际测试调整 |
| 回抽 | 0.4-0.8mm（近程）/ 0.8-1.2mm（Bowden）| 近程建议极小值或关闭 |

### 4.2 PETG 参数

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 喷嘴温度 | 240-260°C | PETG 温度窗口较窄，262°C 是常见最优 |
| 热床温度 | 70-80°C | 过低会翘边，过高会粘连 |
| 打印速度 | 40-60 mm/s | 偏低速保证层间粘合 |
| 冷却风扇 | 30-50%（多层后逐步增加）| 首层关闭，中层开始逐步增加 |
| 回抽 | 3-5mm（Bowden）/ 1-2mm（近程）| PETG 流动性强，回抽要大 |
| 收缩补偿 | ~0.3%（XY）/ ~0.1%（Z）| 比 PLA 略小 |

### 4.3 ABS 参数

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 喷嘴温度 | 250-270°C | ABS 对温度敏感，建议 260°C 起步 |
| 热床温度 | 100-110°C | 110°C 可减少底角翘曲 |
| 打印速度 | 50-80 mm/s | 封箱后可达 80mm/s |
| 腔体温度 | 45-60°C（封箱后）| **必须封箱**，否则翘曲严重 |
| 冷却风扇 | 0%（前几层）→ 20%（后期）| 冷却过度会导致层间开裂 |
| 回抽 | 2-4mm | ABS 冷却后粘度高，回抽不足会拉丝 |
| 收缩补偿 | ~0.5%（XY）/ ~0.4%（Z）| ABS 收缩率最大 |

### 4.4 TPU（柔性耗材）

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 喷嘴温度 | 220-250°C | 根据耗材硬度调整，越软越低温 |
| 热床温度 | 50-60°C | 与 PLA 类似 |
| 打印速度 | 15-30 mm/s | **必须慢速**，否则会堵头 |
| 冷却风扇 | 50-100% | 充分冷却防止粘连 |
| 回抽 | 0.2-0.5mm 或关闭 | 关闭回抽效果往往更好 |
| 层高 | 0.16-0.24mm | 粗层厚更易打印 |

## 五、支撑材料与填充优化

### 5.1 支撑类型选择

| 类型 | 适用场景 | 设置 |
|------|---------|------|
| **有机树形（Tree）** | 复杂悬空结构、文物复制 | 支撑角度 45-50°，密度 10-15% |
| **柱状（Normal）** | 简单悬空、大平面 | 支撑角度 55-60°，密度 10-15% |
| **混合（Hybrid）** | 需要同时保证底部平整 | 支撑密度 15%，界面密度 20% |

**关键参数：**

```ini
支撑密度: 10-15%
支撑界面密度: 20-30%    # 支撑与模型的接触面
支撑与模型间距: 0.2mm  # 间距太小难拆除，太大影响精度
```

### 5.2 填充图案选择

| 图案 | 特点 | 推荐场景 |
|------|------|---------|
| **三角网格（Gyroid）** | 各向同性，强度均衡，排水性好 | 综合性最佳，**首选** |
| **蜂窝（Honeycomb）** | Z 轴强度高，材料效率高 | 需要高强度时 |
| **直线（Lines）** | 简单快速 | 快速原型 |
| **同心（Concentric）** | 外观好，边缘光滑 | 外观件 |

**填充密度参考：**
- 功能件：40-60%
- 一般模型：15-25%
- 展示模型：5-10%
- 校准测试：0%（全外壳）

### 5.3 悬空角度与底部层

```ini
悬空角度阈值: 50-55°  # 大于此角度自动加支撑
底层数: 3-5 层        # 薄壁件可减少到底部表面质量影响大
顶层厚度: 0.8mm       # 确保不漏底
```

## 六、速度与质量平衡

### 6.1 速度档位设置

OrcaSlicer 提供多档速度预设，可以在同一文件中管理质量和速度：

| 预设 | 层高 | 打印速度 | 适用场景 |
|------|------|---------|---------|
| **高质量** | 0.12mm | 40-50 mm/s | 外观件、复杂小件 |
| **标准** | 0.16mm | 60-80 mm/s | 日常打印 |
| **快速** | 0.2mm | 100-150 mm/s | 大尺寸、低细节模型 |
| **超高速** | 0.24mm | 200mm/s+ | 需要 Klipper + 输入整形深度优化 |

### 6.2 层高与精度关系

| 层高 | 精度 | 打印时间 | 推荐场景 |
|------|------|---------|---------|
| 0.08mm | 最高 | 最长 | 精密零件、牙科、珠宝 |
| 0.12mm | 高 | 较长 | 外观件、验证模型 |
| 0.16mm | 标准 | 中等 | **日常首选** |
| 0.2mm | 较低 | 较短 | 大尺寸模型、低细节原型 |
| 0.24mm+ | 低 | 短 | 超高速打印测试 |

### 6.3 OrcaSlicer 十大技巧

> 来源：B站 `3D打印从菜鸟到专家 OrcaSlicer十大问题`

1. **"按层高度重新分段"** ❌ 不要勾选，会影响精度
2. **"间隙填充"** ✅ 建议开启，减少缩痕
3. **压力平滑** ✅ 对 Klipper 机器很有帮助
4. **多色打印** 支持 Bambu AMS 格式，可配合多色改装
5. **有机支撑** 比柱形支撑更好用
6. **减少过挤** PLA 建议 -1% 到 -2%
7. **回抽优化** 近程挤出建议关闭或极小值
8. **封口层** 底部封口层数影响底板强度
9. **网床导入** 支持 Klipper bed_mesh 数据
10. **自定义 Gcode** 善用切片后处理脚本

## 七、网床补偿与 Klipper 联动

### 7.1 OrcaSlicer 启用网床补偿

在 OrcaSlicer 的**打印设置 → 高级**中：
- 勾选 **"使用网床补偿"**
- 选择 **"使用建模位置"** 或 **"使用当前坐标"**

OrcaSlicer 会读取 Klipper 生成的网床数据，在切片时自动补偿热床不平。

### 7.2 Klipper 网床校准流程

```bash
# SSH 中执行
BED_MESH_CALIBRATE
# 或在 Mainsail WebUI 中点击"网格校准"

# 保存
SAVE_CONFIG
```

校准完成后确保 OrcaSlicer 切片时选择正确的网格配置文件。
