# BOSSAC 热床与热管理

> 更新时间：2026-03-29
> 适用：大鱼TT / CoreXY 结构 / 封箱改装 / ABS ASA 打印
> 前提：有一定电路基础，需要 24V 电源

---

## 一、BOSSAC 热床原理

### 1.1 什么是 BOSSAC 热床？

**BOSSAC** = Board One Stupid Simple Advanced Controller，是一种基于 **STM32 单片机**的智能热床控制器。它的核心功能是将热床的**开环加热**升级为**闭环温控**，具备过温保护和精确温度维持能力。

**与普通热床的区别：**

| 特性 | 普通硅胶加热板 | BOSSAC 热床 |
|------|-------------|------------|
| 温控方式 | 开环（PWM 直接控温）| 闭环（STM32 精确控制）|
| 过温保护 | ❌ 无 | ✅ 可配置 |
| 温度稳定性 | 受电源波动影响大 | 稳定维持在 ±1°C |
| 温度传感器 | 通常无 | 标配 NTC 热敏电阻 |
| 调温响应 | 滞后明显 | 快速响应 |
| 智能化 | 无 | 支持状态上报 |

### 1.2 BOSSAC 工作原理

```
MCU 指令 → STM32 控制器 → 驱动 MOS 管 → 加热层
     ↑                                    ↓
  温度传感器 ←←←←←←←←←←←←←←←←←←←←←←←←←← 实际温度
```

BOSSAC 内置 PID 算法，根据热敏电阻反馈实时调整加热功率，相比普通热床直接 PWM 控制的粗糙方式，温控更精准、功耗更优化。

### 1.3 热床选型对比

| 类型 | 电压 | 功率密度 | 加热速度 | 推荐度 |
|------|------|---------|---------|--------|
| 磁吸软玻璃热床 | 24V | 0.8-1.0 W/cm² | 快 | ⭐⭐⭐⭐⭐ |
| 硅胶加热板 + 玻纤底板 | 24V | 0.6-0.8 W/cm² | 中等 | ⭐⭐⭐⭐ |
| BOSSAC 定制热床 | 24V | 0.8 W/cm² | 快 | ⭐⭐⭐⭐ |
| 玻纤板 + 加热片 DIY | 24V | 0.6 W/cm² | 中等 | ⭐⭐⭐ |

> 💡 **推荐磁吸软玻璃热床**：拆卸方便（打印完成后可直接取下玻璃换新），加热均匀性好，配合 BOSSAC 温控效果最佳。

## 二、固态继电器（SSR）控制

### 2.1 为什么要用 SSR？

BOSSAC 热床本身的驱动能力有限（通常只能控制小功率负载）。大功率热床（200W+）需要通过**固态继电器（SSR）** 来控制市电输入（或 24V 主回路）：

- **SSR** 接收 Klipper 主板的 PWM 信号，控制大电流回路
- 主板只需提供几毫安的触发电流，通过 SSR 驱动上百安培的热床

### 2.2 SSR 选型

| 参数 | 要求 |
|------|------|
| 类型 | DC-SSR（直流负载）或 AC-SSR（交流负载）|
| 电压 | ≥ 工作电压（24V 或 220V）|
| 电流 | ≥ 热床最大电流的 1.5 倍 |
| 控制电压 | 与主板输出匹配（通常 3.3-24V）|

**大鱼 TT 推荐配置：**
- 24V 热床 + 24V SSR（最简洁）
- 或 AC-SSR 控制 220V 硅胶加热板（功率更大）

### 2.3 接线方式

```
Klippper 主板                    SSR                    热床
 ┌────────────┐              ┌─────────┐            ┌────────┐
 │ HEATER_0   │─── 控制线 ──▶│ IN(+)   │            │        │
 │ (PWM 输出)  │              │ IN(-)   │            │  加热  │
 └────────────┘              └────┬────┘            │  元件  │
                               │                  │        │
                          ┌────┴────┐             │        │
                          │ OUT     │────────────▶│  +     │
                          │ (负载端) │            └────────┘
                          └─────────┘
```

**接线注意事项：**
- SSR 输入端注意正负极（IN+ 接信号，IN- 接 GND）
- 负载端（大电流）注意散热，SSR 工作时会产生热量
- 大功率应用建议安装散热片

### 2.4 Klipper 配置

```ini
[heater_bed]
# 热床配置
heater_pin: PC13              # 主板上连接 SSR 控制脚的引脚
sensor_type: NTC 100K         # 根据实际传感器选择
sensor_pin: arduino:analog 1  # 温度传感器引脚
min_temp: 0
max_temp: 120
control: pid                  # 使用 PID 温控
pid_Kp: 50.0
pid_Ki: 2.0
pid_Kd: 200.0
# 初始校准后 Klipper 会自动填写更精确的 PID 参数
```

> ⚠️ PID 参数需要通过 `PID_CALIBRATE` 宏自动测定，不要使用手动随意填写。

## 三、热管理：风道 + 风扇 + 温控

### 3.1 封箱散热的核心思路

封箱打印的核心是**维持稳定的腔体温度**，同时兼顾：
- **保温**：减少热量散失，维持高温环境
- **换气**：排出湿气和 ABS 气味，补充新鲜空气
- **循环**：消除腔内温度分层，避免局部过冷过热

### 3.2 风扇配置原则

> "风只需要在打印的时候不发生跳动即为合格" —— 社区经验

**三个层次的风扇配置：**

| 类型 | 规格 | 作用 |
|------|------|------|
| **腔体换气扇** | 6020 / 5015 涡轮，24V | 主动换气，排湿排味 |
| **腔体循环扇** | 4010 静音，24V | 消除温度分层 |
| **热端散热扇** | 4010 / 5015，24V | 冷却热端，防止热端升温 |

### 3.3 推荐方案：腔体过滤换气系统

参考 K2 Plus 封箱方案：

| 组件 | 规格 | 作用 |
|------|------|------|
| 初效滤网 | G3 过滤棉 | 过滤大颗粒 ABS 粉末 |
| HEPA 滤网 | H12 级 | 过滤 PM2.5 颗粒 |
| 活性炭滤芯 | 椰壳活性炭 | 吸附 ABS/ASA 打印气味 |
| 腔体排风扇 | 6020 24V | 主动换气（安装在出风口）|
| 腔体循环风扇 | 4010 24V | 腔内热风循环（安装在角落）|

**进风与排风设计：**
- 进风口：底部或后部，开孔加装滤网
- 排风口：顶部或侧部，连接排风管至室外或过滤器

### 3.4 温控逻辑（Klipper 宏）

```ini
# 腔体温度传感器
[temperature_sensor chamber]
sensor_type: NTC 100K
sensor_pin: arduino:analog 2
min_temp: 0
max_temp: 150
device_type: generic_heater

# 腔体换气风扇（温控自动启停）
[heater_fan chamber_exhaust_fan]
pin: PC14
max_power: 1.0
kick_start_time: 0.5
heater_temp: 45.0          # 腔温超过 45°C 启动
fan_speed: 0.5             # 半速运行
heater: heater_bed

# 腔体循环风扇（持续低俗运转）
[fan]
pin: PC15
max_power: 0.3             # 低速运转，减少气流扰动
```

### 3.5 高级温控宏

```ini
[gcode_macro AUTO_CHAMBER_CONTROL]
description: 自动腔体温控管理
gcode:
  {% set chamber_temp = printer['temperature_sensor chamber'].temperature %}
  {% set bed_temp = printer.heater_bed.temperature %}

  # 热床未达到 80°C 前，不启动换气
  {% if bed_temp < 80 %}
    SET_FAN_SPEED FAN=chamber_exhaust_fan SPEED=0
  {% elif chamber_temp > 55 %}
    SET_FAN_SPEED FAN=chamber_exhaust_fan SPEED=0.8
  {% elif chamber_temp > 45 %}
    SET_FAN_SPEED FAN=chamber_exhaust_fan SPEED=0.5
  {% else %}
    SET_FAN_SPEED FAN=chamber_exhaust_fan SPEED=0.2
  {% endif %}

# 绑定到 idle_timeout 实现自动控制
[idle_timeout]
timeout: 600
gcode: AUTO_CHAMBER_CONTROL
```

## 四、ABS 打印环境搭建

### 4.1 ABS 对环境的要求

ABS 打印对环境温度极为敏感，不封箱打印 ABS 几乎注定失败：

| 环境因素 | 要求 | 后果 |
|---------|------|------|
| 热床温度 | 100-110°C | 低于 95°C 底角必翘 |
| 腔体温度 | 45-60°C | 低于 40°C 模型开裂 |
| 环境温差 | < 5°C 全程 | 骤冷骤热导致层间分离 |
| 空气流动 | 最小化 | 风吹导致冷却不均 |

### 4.2 封箱 BOM 清单

**封箱框架：**
- 2020 / 2040 型材（框架）
- 角码连接件
- 3mm 透明亚克力（侧板）
- 3mm 玻璃（门板，方便观察）

**热床系统：**
- BOSSAC 热床 或 磁吸软玻璃 + 硅胶加热板
- 24V 电源（≥ 10A，建议 15A）
- 热床线束（耐高温硅胶线）
- SSR（固态继电器）

**散热过滤系统：**
- 6020 腔体排风扇 × 2
- 4010 腔体循环风扇 × 1
- G3 初效滤网
- H12 HEPA 滤网
- 椰壳活性炭滤芯

**温控配件：**
- NTC 100K 温度传感器 × 2（热床 + 腔体）
- 继电器 / 固态继电器
- 接线端子、硅胶线、热缩管

### 4.3 ABS 打印参数

```ini
热床温度: 100-110°C
腔体温度: 45-60°C
喷嘴温度: 250-260°C          # 262°C 是很多 ABS 的最优值
冷却风扇: 0%（前 3 层）→ 20%  # 缓慢增加，不可急冷
打印速度: 50-80 mm/s         # 封箱后可提高至 80mm/s
 Brim 宽度: 8mm 以上           # ABS 建议大 Brim 防翘曲
首层速度: 50%（30mm/s）
```

### 4.4 ABS 打印标准流程

1. **预热 10 分钟**：打印前先开启热床和腔体加热，充分热透床面和腔体
2. **大 Brim**：ABS 建议 Brim 8mm+，首层加粗
3. **封闭环境**：确认门窗密封，腔体无漏风
4. **缓慢冷却**：打印完成后**不要立即开箱**，让模型在腔内缓慢冷却至少 30 分钟
5. **取件技巧**：冷却后用刮刀轻轻撬动四角，避免暴力取件损坏模型或伤手

### 4.5 ABS 打印常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 底角翘曲 | 热床温度不足 / 腔温低 | 热床提至 110°C，腔温提至 50°C |
| 模型开裂 | 腔温过低或降温太快 | 确保腔温 45°C+，打印后缓冷 |
| 层间分离 | 温差大或冷却过快 | 降低风扇速度，封闭腔体 |
| 拉丝严重 | 腔温过高 | 降低腔温至 50°C 以下 |
| 有强烈异味 | 换气不足 | 加装排风扇 + 活性炭过滤 |
| 玻璃门起雾 | 内外温差大 | 使用防雾涂层玻璃 |

### 4.6 封箱结构材料对比

| 材料 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **铝合金 + 玻璃** | 耐用、密封好、保温佳 | 成本较高 | ⭐⭐⭐⭐⭐ |
| **亚克力 + 型材** | 便宜、易加工、透视好 | 不耐高温（ABS 打印时软化）| ⭐⭐⭐ |
| **MDF 板 + 薄膜** | 极便宜 | 寿命短，不防火 | ⭐⭐ |
| **PETG/ABS 打印件拼装** | 可自制，定制灵活 | 需要先有一台能正常打印的机器 | ⭐⭐⭐⭐ |
