## 一、热力学现状分析

### 1.1 发热源清单

| 热源 | 功率 (W) | 进入腔体 (W) | 位置 | 备注 |
|------|---------|-------------|------|------|
| 热床 (310×310mm) | 300 | 270 | 底部 | 主要热源，占比 77% |
| 热端 (Hotend) | 60 | 50 | 喷头 | 200-260°C 喷嘴加热 |
| X/Y 步进电机 ×2 | 10 | 8 | 框架左右 | Nema17 |
| Z 步进电机 | 8 | 6 | 后侧 | 移动时消耗 |
| 主板 + 驱动 | 15 | 12 | 框架底部 | KL768 + TMC2209 |
| 开关电源损耗 | 10 | 5 | 框架底部 | 约 5-10% |
| **合计** | **~403 W** | **~351 W** | | |

### 1.2 现有封箱热性能（V2 结构）

| 工况 | 腔温 | PLA 兼容性 | 说明 |
|------|------|-----------|------|
| 自然对流（无风扇）| **~46°C** | ❌ 禁用 | 接近 PLA 软化点 55°C |
| 强制排风（120mm 单风扇）| **~27°C** | ✅ 可用 | 温差约 19°C |
| 目标区间（ABS 打印）| **50-60°C** | ✅ ABS 专用 | 需要配合热床加热 |
| 安全上限 | **60°C** | ⚠️ 注意 | 超过会软化 PETG 打印件 |

**关键结论：**
- 当前无风扇腔温 ~46°C，已超过 PLA 安全温度上限（55°C），封箱内**严禁使用 PLA** 打印件固定任何东西
- 120mm 风扇强制排风可将腔温降至 ~27°C，所有常规材料均可使用
- ABS 打印需要热床主动加热至 100-110°C，配合封闭腔室自然升温至 50-60°C

### 1.3 机箱换气效率分析

|------|------|------|

| 总换热面积 | **2.086 m²** | 六面体内表面 |
| 120mm 风扇风量 | ~288 m³/h | Noctua NF-A12x25 典型值 |
| 换气频率 | **~24 次/分钟** | 理论值，实际因风阻降低 |
| 排风理论温升 | 3.7°C | 热空气排出带走的温升 |

### 1.4 面板变形风险（已采纳）

| 工况 | 3mm 侧板 | 4mm 侧板（V2 采纳）| 说明 |

## 二、进风/排风最优布局设计

### 2.1 风道设计原则

```
【热空气上升原理 + 强制对流结合】

    ┌─────────────────────────────────┐
    │  🔄 顶部排风扇（抽风）           │ ← 最高点排出热空气
    │     120mm PWM  × 1              │
    │         ↑ 上升气流               │
    │                                 │
    │  ┌───────────────────────────┐  │
    │  │  热端热气 → 向上扩散        │  │
    │  │  热床热量 → 向上辐射        │  │
    │  └───────────────────────────┘  │
    │         ↓ 冷空气补充             │
    │  🌀 底部进风口（滤网）            │ ← 最低点进入冷空气
    └─────────────────────────────────┘

黄金原则：
1. 进风口在底部前侧（冷区），排风口在顶部后侧（热区）
2. 进排风对角线布局，形成完整对流路径
3. 避免短路（进风直接吸入排风）
```

### 2.2 推荐布局方案

#### 方案 A：单排顶部 + 底部前侧进风（推荐 ✅）

```
俯视图（从上往下看）：

    │ (后板上部)                  │

    │  ←─进风滤网(底部前侧)──→     │ ← 进风口（冷区）

    └─────────────────────────────┘

    Z=760mm ──────────────────── [排风扇120mm]
    │
    │         上升热气流
    │              ↑
    │     ┌───────────────┐
    │     │  热端热气     │

    │     │  热床 310×310 │
    │     └───────────────┘
    │              ↓
    │  ─────────────────────────  ← [进风滤网]
```

## 三、温控闭环系统设计

### 3.1 系统架构

```
┌──────────────────────────────────────────────────────────┐
│                     Klipper 主机                         │
│  ┌────────────────┐    ┌──────────────────────────────┐ │
│  │ 温度传感器      │    │ PWM 风扇控制                  │ │
│  │ BME280 (I2C)   │──→ │ 120mm 排风扇                  │ │
│  │ 腔温 + 湿度     │    │ Noctua NF-A12x25 12V PWM     │ │
│  └───────┬────────┘    └──────────────┬───────────────┘ │
│          │                              │                │
│          ↓ 反馈                         ↓ PWM信号        │
│   腔温读数 ─────────────────────────→  风扇转速调节       │
│                                                           │
│   控制逻辑：                                               │
│   IF 腔温 > 35°C → 风扇提速                               │
│   IF 腔温 < 30°C → 风扇降速                               │
│   IF 腔温 > 60°C → 报警 + 风扇全速                        │
└──────────────────────────────────────────────────────────┘
```

### 3.2 传感器选型对比

| 参数 | **BME280** ✅ 推荐 | MAX6675 |
|------|-------------------|---------|
| 测量类型 | 温度 + 湿度 + 气压 | 温度（热电偶）|
| 温度范围 | -40~85°C | 0~700°C |
| 腔温适用性 | ✅ **完美** | ✅ 可用 |
| 接口 | I2C / SPI | SPI |
| 电压 | 3.3V | 3.3V / 5V |
| 精度 | ±0.5°C | ±1.5°C |
| 成本 | ~15元（模块）| ~10元（模块+热电偶）|
| 典型模块尺寸 | 15×10mm | 25×13mm（模块）|
| 适合安装位置 | **腔内中部** | 热端附近 |

**结论：选 BME280** — 同时测量腔温和湿度，湿度数据有助于判断 ABS 打印条件。

### 3.3 温控闭环参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 腔温安全上限 | **60°C** | 超过则 PETG 打印件软化 |
| 腔温报警阈值 | **55°C** | 提醒检查通风 |
| 风扇全速触发温度 | **35°C** | 高于这个温度风扇全力运转 |
| 风扇怠速转速 | **20-30%** | 保持空气流通，防止热气积聚 |
| 风扇最低转速 | **15%** | Noctua 最低稳定转速 |
| 腔温目标（PLA/PETG）| **25-30°C** | 配合排风扇主动降温 |
| 腔温目标（ABS）| **50-60°C** | 关闭风扇，利用热床自然加热 |

### 3.4 控制曲线

```
风扇转速 (%)
    100% ─ ┐
           │     全速排气（高温警告）
     75% ─ ┤
           │     中速排气
     50% ─ ┤
           │  ════════════════ 35°C 触发全速
     30% ─ ┼──────────────────────────────
           │  ════════════════ 30°C 启动温控
     15% ─ ┼───怠速维持──→
           └───────────────────────────────→ 腔温 °C
           25   30   35   40   45   50   55   60
```

### 3.5 安全保护策略

```python
# Klipper G-Code 宏中的安全逻辑
[gcode_macro ENCLOSURE_SAFE_CHECK]
gcode:
    {% set chamber_temp = printer.temperature_sensor.chamber.temperature %}
    {% if chamber_temp > 60 %}
        M112  # 紧急停止
        {action_respond_info("⚠️ 腔温 %.1f°C 超过60°C！已紧急停止！" % chamber_temp)}
    {% elif chamber_temp > 55 %}
        SET_FAN_SPEED FAN=chamber SPEED=1.0
        {action_respond_info("⚠️ 腔温 %.1f°C 过高！风扇全速！" % chamber_temp)}
    {% endif %}
```

### 4.4 安装步骤

1. **定位**：将 ThermalControllerMountV1 滑入左侧板对应的 2020 垂直型材槽中
2. **固定**：顺时针旋转，听到"咔"一声即到位（过盈配合）
3. **布线**：BME280 I2C 线（VCC/GND/SDA/SCL）通过侧板走线槽引出到主板
4. **盖上**：盖板对准缺口用力压入
5. **连接**：I2C 线接入主板预留接口（需 3.3V 供电）

## 五、Klipper 温控配置

### 5.1 接线方案

```
BME280 模块接线：
  红线 (VCC)  →  3.3V
  黑线 (GND)  →  GND
  白线 (SDA)  →  I2C SDA (PC3)
  黄线 (SCL)  →  I2C SCL (PC4)

120mm PWM 风扇接线：
  黄色 (PWM)  →  任意 PWM 引脚（如 EXP1_2）
  红色 (VCC)  →  12V 或 5V（参考 Noctua 规格）
  黑线 (GND)  →  GND

※ Noctua NF-A12x25 有三种调速模式：
   1. 4-pin PWM（推荐）：精准调速
   2. 3-pin DC：电压调速
   3. 二合一降速线：降为低噪音版本
```

### 5.2 printer.cfg 完整配置

```ini
# ─────────────────────────────────────────────────────────────
# 腔体温控系统配置
# 大鱼TT 310 | Klipper + BME280 + Noctua NF-A12x25
# ─────────────────────────────────────────────────────────────

# ── 腔温传感器（BME280 I2C）────────────────────────────────
[temperature_sensor chamber]
sensor_type: BME280
# I2C 引脚（根据实际接线调整）
i2c_address: 118  # 0x76（默认），0x77 为 A0 跳线接通
i2c_bus: i2c1
# 温度校准（若传感器位置偏差）
temp_offset: 0
min_temp: 0
max_temp: 100
gcode_id: T_chamber

# ── 排风扇（120mm Noctua PWM）─────────────────────────────
[fan]
chamber_fan:
    pin: EXP1_2          # PWM 引脚（根据实际接线调整）
    max_power: 1.0
    kick_start_time: 0.5
    off_below: 0.15      # Noctua 最低稳定转速 15%
    shutdown_speed: 0.2   # 关机时保持 20% 防止热积聚

# ── 腔温联动 PID 控制（controller_fan）────────────────────
[controller_fan ChamberCooling]
fan: chamber_fan
sensor: temperature_sensor chamber
idle_temperature: 30      # 30°C 以下关闭风扇
fan_speed: 0.20          # 怠速 20%
min_speed: 0.15          # 最低 15%
max_speed: 1.0           # 最高 100%
temperature_offset: 3.0   # 滞后补偿（避免震荡）
control_width: 0.5      # 控制周期（秒）

# ── 热端风扇联动（热端温度 → 腔温监测）────────────────────
[heater_fan hotend_fan]
fan: chamber_fan
heater: extruder
idle_speed: 0.20         # 待机时 20%
min_speed: 0.15
max_speed: 0.60          # 热端全速时腔扇上限 60%

# ─────────────────────────────────────────────────────────────
# G-Code 快捷宏
# ─────────────────────────────────────────────────────────────

[gcode_macro FAN_TEST]
gcode:
    SET_FAN_SPEED FAN=chamber_fan SPEED=0.5
    {action_respond_info("风扇测试中：50% 转速")}

[gcode_macro FAN_FULL]
gcode:
    SET_FAN_SPEED FAN=chamber_fan SPEED=1.0
    {action_respond_info("风扇全速：100%")}

[gcode_macro FAN_OFF]
gcode:
    SET_FAN_SPEED FAN=chamber_fan SPEED=0
    {action_respond_info("风扇已关闭")}

[gcode_macro CHECK_CHAMBER]
gcode:
    {% set t = printer.temperature_sensor.chamber.temperature %}
    {% set h = printer.temperature_sensor.chamber.humidity %}
    {action_respond_info("腔温: %.1f°C | 湿度: %.1f%%" % (t, h))}

[gcode_macro ENCLOSURE_PLA]
gcode:
    # PLA 打印：低温通风模式
    SET_FAN_SPEED FAN=chamber_fan SPEED=0.40
    {action_respond_info("PLA 模式：腔扇 40%")}

[gcode_macro ENCLOSURE_PETG]
gcode:
    # PETG 打印：中温模式
    SET_FAN_SPEED FAN=chamber_fan SPEED=0.35
    {action_respond_info("PETG 模式：腔扇 35%")}

[gcode_macro ENCLOSURE_ABS]
gcode:
    # ABS 高温打印：关闭腔扇，利用热床加热腔室
    SET_FAN_SPEED FAN=chamber_fan SPEED=0.0
    {action_respond_info("ABS 模式：腔扇关闭，腔室自然升温")}

[gcode_macro ENCLOSURE_SAFE_CHECK]
gcode:
    {% set ct = printer.temperature_sensor.chamber.temperature %}
    {% if ct > 60 %}
        M112
        {action_respond_info("⚠️ 紧急停止！腔温 %.1f°C 超过 60°C！" % ct)}
    {% elif ct > 55 %}
        SET_FAN_SPEED FAN=chamber_fan SPEED=1.0
        {action_respond_info("⚠️ 腔温 %.1f°C 过高！风扇全速！" % ct)}
    {% endif %}
```

### 5.3 温度传感器校准

BME280 安装在腔内时，由于热端附近热气上升，传感器读数可能比实际腔温高 2-5°C。建议：

```ini
# temp_offset 用于校准
# 若传感器显示 32°C 但实际腔温约 28°C，设置 temp_offset: -4
temp_offset: -3.0  # 根据实测调整
```

## 八、安装流程

### 第 1 步：安装排风扇（TopFanMountV2）

1. 将 TopFanMountV2.stl 滑入顶部后侧 2020 横梁
2. 安装 120mm Noctua 风扇（注意方向：风扇标签面朝上）
3. 固定 4× M3×8 螺丝
4. 安装 D型密封条

### 第 2 步：安装温控传感器（ThermalControllerMountV1）

1. 将 ThermalControllerMountV1 滑入左侧板 2020 垂直型材（位置：侧板中部偏上）
2. 连接 BME280 I2C 线到主板 I2C 接口
3. 确认 3.3V 供电正常
4. 盖上 Snap-fit 盖板

### 第 3 步：安装底部进风滤网

1. 裁剪过滤棉至 100×100×3mm
2. 插入 BottomIntakeFilterV1 框架
3. 安装到底板前侧门框下方

### 第 4 步：Klipper 配置

1. 将 printer.cfg 中的温控配置追加到现有配置
2. 重启 Klipper 服务
3. 运行 `FAN_TEST` 测试风扇
4. 运行 `CHECK_CHAMBER` 确认传感器读数
5. 如有必要，调整 `temp_offset` 校准

### 第 5 步：调参与验证

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 室温下运行腔扇 30% | 无共振噪音 |
| 2 | 打印 PLA，观察腔温 | 腔温 ≤ 35°C |
| 3 | 打印 PETG，观察腔温 | 腔温 ≤ 40°C |
| 4 | 模拟 ABS 参数（热床 110°C）| 腔温 50-60°C 范围内 |
| 5 | 检查密封效果 | 门关闭后腔温稳定 |
