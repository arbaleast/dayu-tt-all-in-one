## 一、设计背景与目标

### 1.1 问题分析

大鱼TT310 原机皮带调节存在以下痛点：
- 皮带张紧需拆机调节，操作繁琐
- 电机支架与同步带调节器之间缺乏集成
- 长期使用后皮带松弛难以快速恢复
- 调节精度差，易出现过紧或过松

### 1.2 设计目标

设计一款**集成式张紧电机座 V2**，实现：
- 免拆机调节皮带张力
- 兼容原机 Nema17 电机支架（4×M3，31mm间距）
- 支持 PETG 3D 打印，4mm 壁厚
- 提供两种方案供用户选择

## 三、方案A详解：弹簧臂式（偏心轮张紧）

### 3.1 核心原理

```
        调节螺丝 (M4)
            ↓
      ┌─────────────────┐
      │    张紧螺丝     │
      └────────┬────────┘
               │
    弹簧 ─────►│◄──── 臂
               │ (枢轴旋转)
         偏心轮压向皮带
              ╱╲
         ═════╱  ╲═════  ← 皮带
              ╲  ╱
               ╲╱
```

- **偏心轮**：安装在 624ZZ 轴承上，偏心距 2mm
- **弹簧**：提供恒定张紧力，自动补偿皮带伸长
- **调节螺丝**：推动臂改变初始张紧位置

## 四、方案B详解：滑轨调节式

### 4.1 核心原理

```
    电机支架 (铝合金)
    ┌─────────────────────────┐
    │    滑轨底座（TensionerB_Base）   │
    │  ════════════════════════│
    │  [滑块]──► [调节螺丝M4]        │
    │   ↑                        │
    │  惰轮（压向皮带）              │
    │  ════════════════════════ 皮带 │
    └─────────────────────────┘
```

- **滑块**：在底座导轨槽内线性滑动
- **惰轮**：624ZZ 轴承支撑，光轮（无偏心）
- **调节螺丝**：M4 螺丝精确推动滑块
- **波纹预压环**：弹簧替代品，简化装配


**安装位置建议**：
- **方案A（弹簧臂式）**：安装在 X 轴电机对侧（通常是右侧电机位置）
- **方案B（滑轨式）**：安装在皮带松弛侧或两电机中间位置

### 6.2 皮带张力判断标准

> **"皮带只需要在打印的时候不发生跳动即为合格"**

| 状态 | 手指按压下沉量 | 表现 | 处理 |
|------|-------------|------|------|
| 过松 | > 3mm | 打印跳齿、层纹 | 增加张力 |
| 合适 | 1-2mm | 无异响、打印正常 | 保持 |
| 过紧 | < 1mm | 轴承噪音、阻力大 | 减小张力 |

## 七、注意事项与维护

### 7.1 安装注意事项

1. **安装方向**：确保偏心轮/惰轮与皮带的接触面正确
2. **轴承方向**：有防尘盖的一面朝外（朝向皮带）
3. **螺丝扭矩**：M3 螺丝扭矩不超过 0.8 N·m，避免塑料崩牙
4. **PETG 特性**：打印后 24 小时内避免承重，让材料充分结晶

### 7.2 日常维护

- **每月检查**：检查皮带张力，必要时微调
- **每半年**：清理轴承，检查是否有异响
- **每年**：检查轴承转动是否顺畅，必要时更换

### 7.3 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| idler/惰轮转动不顺畅 | 轴承安装歪斜或异物 | 重新压装轴承，清理异物 |
| 调节螺丝旋不动 | 螺纹磨损或塑料变形 | 攻丝或更换零件 |
| 皮带跳齿 | 张力不足 | 顺时针旋转调节螺丝增加张力 |
| 打印尺寸偏小 | 皮带过松或过紧 | 校准皮带张力后重新切片 |
