## 皮带张紧器设计详解

### 问题背景

大鱼TT原机皮带张紧调节困难，需要拆机才能调节，且难以保持最佳张力。

### 解决方案

```
外置张紧器设计：
┌─────────────────┐
│   铝型材框架     │
│  ┌───────────┐  │
│  │  固定座    │  │ ← 安装在型材上
│  │   ┌───┐   │  │
│  │   │轴承│   │  │ ← 张紧轮
│  │   └───┘   │  │
│  │    弹簧    │  │ ← 自动补偿
│  │  调节螺丝   │  │ ← 手动微调
│  └───────────┘  │
└─────────────────┘
```

### 调节方式

- **粗调**：松开固定螺丝，移动张紧器位置
- **细调**：旋转调节螺丝，弹簧自动补偿张力变化

## 装配步骤

### 第一步：框架搭建
1. 切割铝型材到指定长度
2. 用角码组装底部框架
3. 安装立柱和顶部框架
4. 检查对角线，确保方正

### 第二步：安装皮带张紧器
1. 打印张紧器零件（2套，X/Y各1）
2. 安装轴承到张紧轮
3. 将张紧器固定在铝型材上
4. 安装皮带并调整张力

### 第三步：板材安装
1. 切割亚克力板到尺寸
2. 安装侧板、背板、顶板
3. 安装门板（带磁吸/合页）
4. 内侧贴保温棉

### 第四步：电气安装（可选）
1. 安装LED灯条
2. 安装通风风扇
3. 整理走线

### 第五步：调试测试
1. 检查所有螺丝紧固
2. 测试皮带张力
3. 测试ABS打印效果
4. 调整仓温到40-50°C

## 注意事项

1. **皮带张紧器**是此方案的核心，建议优先打印安装
2. **铝型材切割**要精确，误差控制在1mm以内
3. **亚克力板**建议本地购买，快递易损坏
4. **ABS打印**时仓温控制在40-50°C，过高会损坏打印机部件
5. **通风系统**建议安装，减少打印异味
