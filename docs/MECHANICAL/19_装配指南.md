# 装配指南

## 工具准备

### 必备工具
- [ ] 内六角扳手套装（2/2.5/3/4mm）
- [ ] 十字螺丝刀
- [ ] 美工刀/亚克力勾刀
- [ ] 卷尺/卡尺
- [ ] 电钻（3-4mm钻头）
- [ ] 锉刀
- [ ] 砂纸（200-400目）

### 可选工具
- [ ] 台钳
- [ ] 热风枪
- [ ] 水平仪
- [ ] 角尺

## 装配流程

### 第一阶段：框架搭建（预计2-3小时）

#### 步骤1：铝型材切割
- [ ] 根据尺寸表切割铝型材
- [ ] 切口去毛刺
- [ ] 检查长度精度（误差<1mm）

#### 步骤2：底部框架
- [ ] 组装底部方形框架
- [ ] 安装角码（每个角2个）
- [ ] 检查对角线长度（误差<2mm）
- [ ] 确保框架平整

#### 步骤3：安装立柱
- [ ] 4根立柱垂直安装
- [ ] 使用角尺检查垂直度
- [ ] 临时固定，不要锁死

#### 步骤4：顶部框架
- [ ] 安装顶部框架
- [ ] 检查整体方正
- [ ] 锁紧所有螺丝

#### 步骤5：安装打印机
- [ ] 将大鱼TT放入框架
- [ ] 固定打印机底座
- [ ] 检查打印机水平

### 第二阶段：皮带张紧器安装（预计1小时）

#### 步骤1：打印零件检查
- [ ] 检查固定座尺寸
- [ ] 检查活动块滑动顺畅
- [ ] 清理支撑和毛刺

#### 步骤2：组装张紧轮
- [ ] 轴承压入张紧轮
- [ ] 确保转动顺畅

#### 步骤3：安装张紧器
- [ ] 固定座安装到型材
- [ ] 装入活动块
- [ ] 安装弹簧和调节螺丝
- [ ] 绕过皮带

#### 步骤4：调节张力
- [ ] 粗调：移动张紧器位置
- [ ] 细调：旋转调节螺丝
- [ ] 测试皮带张力

### 第三阶段：板材安装（预计2-3小时）

#### 步骤1：板材切割
- [ ] 按尺寸切割亚克力板
- [ ] 边缘打磨光滑
- [ ] 清洁表面

#### 步骤2：背板安装
- [ ] 安装背板亚克力
- [ ] 使用T型螺母固定
- [ ] 内侧贴保温棉

#### 步骤3：侧板安装
- [ ] 安装左右侧板
- [ ] 预留风扇孔位
- [ ] 内侧贴保温棉

#### 步骤4：顶板安装
- [ ] 安装顶板
- [ ] 可做成可开启式（推荐）
- [ ] 安装LED灯条

#### 步骤5：门板安装
- [ ] 安装合页
- [ ] 安装磁吸条
- [ ] 安装门把手
- [ ] 测试开合

### 第四阶段：电气安装（预计1-2小时）

#### 步骤1：LED灯条
- [ ] 灯条固定在顶部
- [ ] 连接电源线
- [ ] 测试开关

#### 步骤2：风扇安装
- [ ] 安装进风风扇（底部或侧下）
- [ ] 安装排风风扇（顶部或侧上）
- [ ] 安装风扇罩
- [ ] 连接电源

#### 步骤3：温控系统（可选）
- [ ] 安装PTC加热器（底部）
- [ ] 安装温控器
- [ ] 连接温度探头
- [ ] 设置温控参数

#### 步骤4：走线整理
- [ ] 使用线槽整理电线
- [ ] 固定线束
- [ ] 检查无干涉

### 第五阶段：调试测试（预计1小时）

#### 步骤1：机械检查
- [ ] 检查所有螺丝紧固
- [ ] 检查皮带张力
- [ ] 检查门开合顺畅
- [ ] 检查风扇转动

#### 步骤2：电气测试
- [ ] 测试LED灯条
- [ ] 测试风扇
- [ ] 测试温控器（如有）
- [ ] 检查无短路

#### 步骤3：打印测试
- [ ] 打印PLA测试（不加热）
- [ ] 测试ABS打印（加热到45°C）
- [ ] 观察是否有层移
- [ ] 检查表面质量

## 安全注意事项

1. **电气安全**
   - 接线时断电操作
   - 使用绝缘工具
   - 检查无短路再上电

2. **机械安全**
   - 锋利边缘打磨
   - 风扇加防护罩
   - 加热器远离可燃物

3. **使用安全**
   - 高温表面贴警示
   - 保持通风
   - 人离开时关闭加热

## 故障排除

### 框架问题
| 问题 | 原因 | 解决 |
|-----|------|------|
| 框架晃动 | 螺丝松动 | 检查所有连接点 |
| 门不严 | 框架变形 | 调整框架方正度 |
| 板材松动 | 固定不牢 | 增加固定点 |

### 皮带问题
| 问题 | 原因 | 解决 |
|-----|------|------|
| 跳齿 | 皮带松 | 调紧张紧器 |
| 异响 | 轴承问题 | 检查/更换轴承 |
| 层移 | 张力不均 | 调整X/Y张力一致 |

### 温控问题
| 问题 | 原因 | 解决 |
|-----|------|------|
| 温度上不去 | 加热器功率不足 | 增加加热器或保温 |
| 温度波动大 | 温控器参数不对 | 调整PID参数 |
| 局部过热 | 风扇位置不对 | 调整风扇位置 |
