# S42B V2.0 闭环驱动板详细资料

## 产品概述

**BIGTREETECH S42B V2.0** 是一款集成式闭环步进电机驱动板，专为3D打印机设计，可以将普通42步进电机升级为闭环伺服系统。

## 技术规格

### 基本参数
| 参数 | 规格 |
|-----|------|
| 供电电压 | DC 12-24V |
| 最大电流 | 1.5A (峰值2A) |
| 编码器分辨率 | 1000PPR |
| 细分设置 | 1/2/4/8/16/32/64/128/256 |
| 控制接口 | 脉冲+方向 / UART |
| 电机类型 | 42步进电机 (NEMA17) |
| 显示屏 | 0.96寸 OLED |
| 尺寸 | 42mm × 42mm × 15mm |

### 接口定义

#### 电机接口 (J1)
| 引脚 | 功能 |
|-----|------|
| 1 | A+ |
| 2 | A- |
| 3 | B+ |
| 4 | B- |

#### 控制接口 (J2)
| 引脚 | 功能 | 说明 |
|-----|------|------|
| 1 | VCC | 12-24V供电 |
| 2 | GND | 地线 |
| 3 | STEP | 脉冲信号 |
| 4 | DIR | 方向信号 |
| 5 | EN | 使能信号 (低有效) |
| 6 | TX | UART发送 |
| 7 | RX | UART接收 |

#### 编码器接口 (J3)
| 引脚 | 功能 |
|-----|------|
| 1 | VCC (3.3V/5V) |
| 2 | GND |
| 3 | A相 |
| 4 | B相 |

## 硬件安装

### 配件清单
- S42B V2.0 驱动板 × 1
- 磁环 (带背胶) × 1
- 编码器板 × 1
- 连接线 × 1套
- 散热片 (选配) × 1
- 固定螺丝 M3×6 × 4

### 安装步骤

#### 1. 磁环安装
```
步骤:
1. 清洁电机轴
2. 撕下磁环背胶
3. 将磁环套入电机轴
4. 确保磁环与轴同心
5. 按压固定

注意事项:
- 磁环必须完全水平
- 不能有倾斜
- 不能被磁化物品靠近
```

#### 2. 编码器板安装
```
步骤:
1. 将编码器板固定在电机尾部
2. 使用M3螺丝固定
3. 调整编码器板位置
4. 确保磁环在编码器中心
5. 测量间隙 (1-3mm)

间隙调整:
- 间隙 < 1mm: 可能摩擦，调大
- 间隙 1-3mm: 正常范围
- 间隙 > 3mm: 信号弱，调小
```

#### 3. 驱动板安装
```
步骤:
1. 将S42B插入驱动插槽
2. 注意方向 (EN引脚对齐)
3. 连接电机线
4. 连接编码器线
5. 连接电源和控制线

线序检查:
- 电机线: A+/A-/B+/B- 对应电机绕组
- 编码器线: 按颜色对应连接
- 控制线: STEP/DIR/EN 对应主板
```

## 菜单操作

### 按键说明
- **短按**: 确认/进入
- **长按**: 返回/退出
- **双击**: 快捷功能

### 菜单结构

```
主菜单
├── Info (信息)
│   ├── 当前位置
│   ├── 目标位置
│   ├── 误差值
│   └── 温度
├── Cal (校准)
│   └── 自动校准电机
├── Set (设置)
│   ├── Current (电流)
│   ├── Microstep (细分)
│   ├── Mode (模式)
│   └── UART (串口设置)
├── Test (测试)
│   ├── 正转测试
│   ├── 反转测试
│   └── 往复测试
└── Save (保存)
    └── 保存设置到Flash
```

### 参数设置

#### 电流设置
```
路径: Set → Current
范围: 0.5A - 1.5A
建议:
- 普通42电机: 1.0A
- 高扭矩电机: 1.2-1.5A
- 注意: 超过1.5A需要加强散热
```

#### 细分设置
```
路径: Set → Microstep
选项: 1, 2, 4, 8, 16, 32, 64, 128, 256
建议:
- 3D打印: 16或32
- CNC: 8或16
- 高分辨率: 64或128
注意: 细分越高，扭矩越小
```

#### 工作模式
```
路径: Set → Mode

模式1: 开环模式 (Open Loop)
- 当作普通步进驱动使用
- 无闭环功能
- 用于测试

模式2: 闭环模式 (Closed Loop)
- 启用闭环控制
- 自动补偿丢步
- 正常工作模式

模式3: 静音模式 (Silent)
- 降低电流和速度
- 噪音最小
- 适合夜间使用
```

## 校准流程

### 自动校准
```
步骤:
1. 进入主菜单
2. 选择 "Cal"
3. 按确认键
4. 电机自动正转一圈
5. 电机自动反转一圈
6. 显示 "Cal OK"

注意事项:
- 确保电机空载
- 确保磁环安装正确
- 观察是否有抖动
- 如有异常，重新校准
```

### 手动微调
```
如果自动校准不理想:

1. 进入 Test 菜单
2. 选择正转测试
3. 观察电机运行
4. 如有抖动，检查:
   - 磁环是否松动
   - 间隙是否合适
   - 电机线序是否正确
5. 调整后重新校准
```

## 故障排除

### 常见问题

#### 1. 显示屏不亮
```
检查:
1. 供电是否正常 (12-24V)
2. 电源极性是否正确
3. 保险丝是否熔断
4. 连接线是否松动
```

#### 2. 电机不转
```
检查:
1. 使能信号是否正确 (EN低电平有效)
2. 脉冲信号是否正常
3. 电机线是否接好
4. 电流设置是否过低
```

#### 3. 校准失败
```
现象: 显示 "Cal Fail"

原因和解决:
1. 磁环未安装 → 安装磁环
2. 磁环松动 → 紧固磁环
3. 间隙不合适 → 调整至1-3mm
4. 编码器损坏 → 更换编码器
5. 电机阻力大 → 检查机械部分
```

#### 4. 运行抖动
```
原因:
1. 磁环偏心 → 重新安装
2. 电流过低 → 提高电流
3. 细分过高 → 降低细分
4. 机械共振 → 添加减震

解决步骤:
1. 检查磁环安装
2. 适当提高电流
3. 尝试16或32细分
4. 检查皮带张力
```

#### 5. 噪音大
```
优化:
1. 使用32细分或更高
2. 开启静音模式
3. 降低电流 (在扭矩允许范围内)
4. 添加电机减震垫
5. 检查机械部件润滑
```

### 错误代码

| 代码 | 含义 | 解决 |
|-----|------|------|
| E01 | 编码器错误 | 检查编码器连接 |
| E02 | 电机过流 | 降低电流或检查短路 |
| E03 | 电机过热 | 降低电流或加强散热 |
| E04 | 校准错误 | 重新校准 |
| E05 | 位置偏差大 | 检查机械阻力 |

## 性能优化

### 提高响应速度
```
1. 使用更高的细分 (32或64)
2. 优化脉冲频率
3. 减少机械阻力
4. 使用更轻的打印头
```

### 提高精度
```
1. 确保磁环安装精确
2. 使用合适的电流
3. 定期校准
4. 保持机械部件清洁
```

### 降低发热
```
1. 使用合适的电流 (不要过高)
2. 添加散热片
3. 改善通风
4. 降低待机电流
```

```

- S42B支持TMC2209仿真模式
- 开启StealthChop
- 几乎无声运行
```
