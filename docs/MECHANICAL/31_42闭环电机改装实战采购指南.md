## 二、安装步骤

### 2.1 准备工具

| 工具 | 用途 |
|------|------|
| 内六角扳手套装 | 螺丝拆卸 |
| 尖嘴钳 | 拉扣螺母 |
| 螺丝刀 | 接线端子 |
| 万用表 | 检测电机接线 |
| 焊接工具（可选）| 接线 |

### 2.2 拆卸原步进电机

1. **断电** — 关闭打印机电源
2. **记录接线** — 拍照记录原电机线序（A+/A-/B+/B-）
3. **拆卸固定螺丝** — 通常 4 颗 M3 螺丝
4. **移除电机** — 小心皮带不要崩飞

### 2.3 安装闭环电机

**步骤：**
1. 安装电机座到原位置
2. 固定闭环电机（注意方向，轴朝向皮带轮）
3. 安装同步轮到电机轴（使用联轴器过渡）
4. 挂上 GT2 皮带，调整张紧
5. 接线（参照原线序）

### 2.4 接线方式

**两相四线电机接线：**
```
电机线圈 A 相：红线 + 白线
电机线圈 B 相：绿线 + 黑线

接驱动板：
  A+ → 红线
  A- → 白线
  B+ → 绿线
  B- → 黑线
```

> ⚠️ 如果方向相反，交换 A+ 和 A-（或 B+ 和 B-）

## 三、Klipper 配置

> 详细配置参考：`docs/01-封箱方案/42闭环电机/42闭环步进电机改装指南.md`

### 3.1 基本配置

```ini
[stepper_x]
step_pin: PF0
dir_pin: PF1
enable_pin: !PD0
microsteps: 16
rotation_distance: 40
full_steps_per_rotation: 200
# 42闭环无需改变 full_steps_per_rotation
```

### 3.2 驱动电流调整

闭环电机电流通常 1.2A，修改 `printer.cfg`：

```ini
[stepper_x]
driver_TMC2209: ...
run_current: 1.0  # 根据电机规格调整，S42B V2.0 建议 1.0A
hold_current: 0.5
```

## 五、改装注意事项

### ⚠️ 重点提醒

1. **先测试再安装** — 单独测试电机转一圈，确认方向和电流
2. **注意轴径匹配** — 电机轴 5mm，同步轮内径要匹配 5mm
3. **皮带张紧适度** — 太紧增加电机负担，太松会跳齿
4. **散热注意** — 闭环电机发热略大，确保通风
5. **Z 轴一般不需要改** — Z 轴负载小，原步进够用

### ❌ 常见错误

- 接线顺序错误导致电机抖动 → 交换 A/B 相
- 皮带张紧不够 → 打印出现缺线
- 电流设置过大 → 电机发热严重
