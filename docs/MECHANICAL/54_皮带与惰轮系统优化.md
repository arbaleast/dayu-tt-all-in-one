## 一、GT2 皮带选型：6mm vs 9mm

### 1.1 GT2 皮带基础知识

GT2 皮带是一种**同步齿形带**（Timing Belt），采用 2mm 齿距（Pitch）的梯形齿形：

```
皮带参数：
  齿距：2mm（GT2 名称的由来）
  宽度：常见 6mm / 9mm
  材质：氯丁橡胶（Neoprene）+ 玻璃纤维/钢芯
  长度：以齿数计量（如 200GT2 = 400mm 长）

带轮参数：
  节圆直径（PD）= 齿数 × 2mm / π
  示例：20T GT2 → PD = 40mm → 每转 125.6mm
```

### 1.2 6mm vs 9mm 宽度对比

| 对比项 | GT2 6mm | GT2 9mm | 差异 |
|--------|---------|---------|------|
| 宽度 | 6mm | 9mm | +50% |
| 抗拉强度 | 基准 | +50% | 更适合高张力 |
| 刚度 | 较低 | **较高** | 高速时弹跳更少 |
| 适用机型 | V0/V1 小型机 | **V2.4/Trident 大型机** | — |
| 成本 | ¥20/米 | ¥35/米 | +75% |
| 推荐行程 | ≤250mm | **>250mm** | — |
| 惯性负载 | 较高 | **较低** | 9mm 皮带更稳定 |

### 1.3 选型建议

```
根据大鱼 TT 机型选择：

TT150 / TT235（行程 < 250mm）
  → GT2 6mm 皮带（成本低，够用）
  → 可选升级 GT2 9mm（性能提升明显）

TT310（行程 310mm）⭐ 强烈推荐 GT2 9mm
  → 原因一：310mm 行程皮带张力需求高
  → 原因二：高速打印（300mm/s+）需要更高刚度
  → 原因三：Voron Trident 标准配置为 9mm
  → 改装成本：约 ¥40 × 4 = ¥160
```

## 二、惰轮排列方式对比

### 2.1 惰轮基础

惰轮（Idler Pulley）是皮带传动中不与电机轴连接的皮带轮，主要作用：

```
惰轮的功能：
  1. 改变皮带走向（角度转折）
  2. 提供皮带支撑（跨距中间）
  3. 调节皮带张力（张紧轮）
  4. 增加皮带与带轮包角（提高传动精度）
```

### 2.2 常见惰轮排列方式

#### 方式 A：全光轴惰轮（纯支撑型）

```
排列：●────●────●────●
      惰轮 惰轮 惰轮 惰轮

特点：
  → 所有惰轮都是光轴轮（无齿）
  → 皮带与惰轮无啮合，靠摩擦传动
  → 结构简单，成本低
  → 缺点：摩擦力大，不适合高速
```

#### 方式 B：同步带轮 + 光轴（混合型）⭐推荐

```
排列：●═══⊕═══●
       惰轮 惰轮
       光轴 带齿

特点：
  → 电机侧用同步带轮（有齿，与电机轴连接）
  → 中间支撑用光轴惰轮（无齿，纯支撑）
  → 减少皮带振动节点
  → 降低啮合噪音
  → **Voron 官方推荐方式**
```

#### 方式 C：全同步带轮（高刚性型）

```
排列：⊕═══⊕═══⊕═══⊕
      带轮 带轮 带轮 带轮

特点：
  → 所有惰轮都是同步带轮（有齿）
  → 皮带全程与带轮啮合
  → 刚性最高，适合极限速度
  → 缺点：噪音大，成本高
```

### 2.3 惰轮排列对比表

| 排列方式 | 刚性 | 噪音 | 成本 | 适用场景 |
|---------|------|------|------|---------|
| 全光轴 | 低 | 低 | 低 | 低速/预算有限 |
| 混合型 ⭐ | 中 | 中 | 中 | **高速/大多数场景** |
| 全同步轮 | 高 | 高 | 高 | 极限速度/工业级 |

### 2.4 大鱼 TT 推荐排列

```
针对大鱼 TT 310mm 机型：

推荐：smooth → toothed → smooth
  即：光轴惰轮 → 同步带轮 → 光轴惰轮

具体实现：
  电机轴：20T GT2 同步带轮（带齿）
  皮带跨距中点：光轴惰轮（支撑）
  另一侧：张紧机构（光轴惰轮）

优势：
  → 消除皮带振动节点
  → 保持皮带与导轨平行
  → 减少皮带与带轮啮合噪音
```

### 2.5 惰轮轴承选型

**Voron 官方推荐：F695 法兰轴承**

```
F695 轴承规格：
  内径：5mm（或 6mm，选购时注意）
  外径：13mm
  厚度：4mm
  类型：法兰轴承（Flange Bearing）

对比普通轴承：
  F695 有法兰 → 安装更稳，轴向定位精准
  普通 625/626 → 需要额外法兰固定
```

| 轴承型号 | 内径 | 外径 | 法兰 | 推荐度 |
|---------|------|------|------|--------|
| F695 | 5mm | 13mm | 有 | ⭐⭐⭐⭐⭐ Voron 标准 |
| 625 | 5mm | 13mm | 无 | ⭐⭐⭐ |
| 626 | 6mm | 13mm | 无 | ⭐⭐⭐ |

## 三、皮带张力计算与测量

### 3.1 皮带张力的重要性

```
皮带张力的核心矛盾：

张力过低：
  → 皮带跳齿
  → 反向间隙（backlash）增大
  → 定位精度下降
  → 高速时皮带回弹

张力过高：
  → 轴承负荷增加，磨损加快
  → 电机扭矩消耗增大
  → 框架承受额外应力
  → 产生高频振动

最佳张力：刚好足够消除跳齿 + 一定的安全余量
```

### 3.2 张力计算公式

**皮带张力与振动频率的关系：**

```
T = 4 × m × L² × f² / π²

其中：
  T = 皮带张力（N）
  m = 皮带线密度（kg/m）
  L = 皮带跨距（m）
  f = 振动频率（Hz）
```

**GT2 皮带线密度参考：**

| 皮带类型 | 线密度 m |
|---------|---------|
| GT2 6mm | ~0.015 kg/m |
| GT2 9mm | ~0.022 kg/m |
| HTD 3M 9mm | ~0.025 kg/m |

**计算示例：**

```
条件：
  皮带类型：GT2 6mm
  跨距 L = 310mm = 0.31m
  目标频率 f = 100Hz

计算：
  T = 4 × 0.015 × 0.31² × 100² / π²
  T = 4 × 0.015 × 0.0961 × 10000 / 9.87
  T ≈ 58 N

结论：张紧至 100Hz 约需 58N 张力
```

### 3.3 频率测量方法

**工具：手机 APP（推荐）**

```
推荐 APP：
  1. iOS：FrequenSee（免费）
  2. Android：Vibration Analyzer（免费）
  3. 任意能显示频率的调音器 APP

测量步骤：
  1. 拨动皮带，使其振动发声
  2. 打开 APP，靠近皮带
  3. 读取振动频率
  4. 调节张紧器直至达到目标频率
```

**目标频率参考：**

| 皮带类型 | 最低频率 | 推荐频率 |
|---------|---------|---------|
| GT2 6mm | 80Hz | 100-120Hz |
| GT2 9mm | 100Hz | 120-150Hz |
| HTD 3M | 120Hz | 150-180Hz |

### 3.4 张力不均的检测与解决

**症状：** 打印件出现沿某一轴向的规律性波纹

**原因：** 同一轴的两根皮带张力不同

**检测方法：**

```bash
# 方法 1：听音法
# 拨动每根皮带，对比音调
# 音调相同 → 张力均匀
# 音调不同 → 存在张力差

# 方法 2：频率测量
# 逐根测量每根皮带频率
# 张力差应 < 5%
```

**解决方案：**

```
张力差 < 5%：
  → 使用张紧器微调

张力差 > 5%：
  → 检查惰轮是否偏心或磨损
  → 重新穿装皮带（可能局部扭曲）
  → 更换磨损的惰轮轴承
```

## 四、张力自动调节器设计

### 4.1 为什么需要自动调节

```
皮带系统的动态变化：

时间因素：
  → 皮带蠕变：长期运行逐渐伸长
  → 温度影响：热胀冷缩导致张力波动
  → 负载影响：高速加速时皮带动态伸长

后果：
  → 张力下降 → 跳齿风险增加
  → 张力不均 → 运动精度下降
  → 需要频繁手动调节

解决方案：弹簧自动张紧器
```

### 4.2 弹簧张紧器原理

```
     弹簧（提供恒定力 F）
          ↓
        ┌─┴─┐
        │   │  ← 张紧轮可在弹簧方向移动
        └─┬─┘
          │
        ──┴── 皮带
```

**恒定张力原理：**

```
弹簧力 F = k × Δx
其中：
  k = 弹簧刚度（N/mm）
  Δx = 弹簧压缩量（mm）

只要 Δx 在有效行程内，F 保持恒定
→ 皮带伸长时，弹簧自动补偿
→ 张力始终保持在设定值附近
```

### 4.3 弹簧参数计算

**设计目标：** GT2 6mm 皮带，张力 60N

```
所需参数：
  最大张力：T_max = 80N
  最小张力：T_min = 50N（皮带蠕变后）
  弹簧行程：Δx = 5mm（可调节余量）

弹簧刚度：
  k = (T_max - T_min) / Δx
  k = (80 - 50) / 5
  k = 6 N/mm

选择弹簧：
  线径 d = 0.8-1.0mm
  中径 D = 6-8mm
  自由长度 L0 = 25-35mm
  圈数 n = 10-15 圈
  刚度 k ≈ 5-7 N/mm（符合要求）
```

### 4.4 Voron 官方张紧器设计

**参考来源：** Voron-Hub GitHub 仓库

```
OpenSCAD 参数化设计：
  源文件：Voron-Hub/belt_tensioner.scad

可调参数：
  belt_width：6mm / 9mm
  spring_spec：弹簧规格
  mounting_holes：安装孔位
  tensioner_range：张紧行程

3D 打印材料：PETG 或 ABS
  → 耐温 > 60°C（电机附近温度）
  → 强度足够支撑张力
```

### 4.5 DIY 张紧器设计（低成本方案）

```
简易方案：Nylock 螺母自锁结构

原理：
  螺母拧紧到设定张力 → 螺纹自锁
  皮带的轻微伸长 → 螺母略微松动补偿
  Nylock 螺母的尼龙衬垫 → 防止进一步松动

成本：¥0（使用现有螺丝）
限制：只能补偿微小伸长，不适合长期运行
```

### 4.6 张紧器安装位置

```
推荐安装位置：张紧力臂末端

典型 CoreXY 皮带路径：
  电机A ──皮带A1──┬──皮带A2── 惰轮
                  │
                  ↓
              [张紧器]
                  │
                  ↑

原则：
  1. 张紧器应安装在皮带路径的外侧（方便调节）
  2. 张紧轮与皮带夹角约 30-45°（提供分力）
  3. 避免张紧轮直接压在皮带上（会产生额外摩擦）
```

## 六、日常维护

### 6.1 定期检查项目

```
每 500 打印小时：
  □ 检查皮带磨损（齿形是否完整）
  □ 检查惰轮转动是否顺畅
  □ 测量皮带张力频率（是否 > 最低值）
  □ 检查皮带是否偏斜（是否沿惰轮侧面跑偏）

每 1000 打印小时：
  □ 更换皮带（即使未明显磨损）
  □ 清洁皮带和惰轮（去除积灰）
  □ 检查电机轴带轮是否松动
  □ 添加润滑脂到惰轮轴承（可选）
```

### 6.2 皮带更换信号

```
需要更换皮带的迹象：
  1. 齿形磨损（与新皮带对比）
  2. 表面裂纹或分层
  3. 张力明显下降（频率 < 最低值）
  4. 打印精度持续下降
  5. 皮带有明显伸长
```

## 七、总结

皮带与惰轮系统是 CoreXY 打印机的**生命线**，其性能直接影响打印速度、精度和稳定性。

**核心优化方向：**

1. **皮带选型**：TT310 强烈推荐 GT2 9mm + Gates 原厂皮带
2. **惰轮排列**：混合型（同步轮 + 光轴）是最佳性价比选择
3. **F695 轴承**：Voron 社区验证的最佳惰轮轴承规格
4. **张力管理**：弹簧自动张紧器 + 频率法检测
5. **定期维护**：每 500 小时检查，1000 小时换皮带

**改装优先级：**

| 优先级 | 改装项 | 成本 | 效果 |
|--------|--------|------|------|
| P0 | GT2 9mm 皮带 | ¥140 | 速度 +50% |
| P0 | F695 轴承惰轮 | ¥30 | 摩擦力 -50% |
| P1 | 弹簧张紧器 | ¥20 | 张力稳定性 |
| P2 | Gates 原厂皮带 | ¥80 | 寿命 +100% |

> 皮带系统优化是高速打印的基础投资，合理的张力管理和高质量的惰轮轴承可以将皮带跳齿风险降低 90% 以上。
