## 一、Klicky 探针安装

### 概述
Klicky是可拆卸的磁性微动开关探针系统，通过舵机或手动方式装卸。

### 关键步骤

#### 1. 打印件准备
需要打印的部件：
- 探针主体
- 磁铁固定座
- 开关安装支架
- 舵机臂 (如使用自动)

**打印参数:**
- 材料: PETG 或 ABS
- 填充: 40%
- 壁厚: 3层

#### 2. 硬件组装
1. 将钕铁硼磁铁按统一极性插入探针和固定座
2. 安装微动开关，确保触发杆朝向正确
3. 连接杜邦线到开关COM和NC引脚
4. 测试磁吸力度是否适中

#### 3. 固件配置 (Klipper)

```ini
[probe]
pin: ^P1.27  # 根据实际接线修改
x_offset: 0
y_offset: 25.0
z_offset: 6.5  # 需校准
speed: 5.0
samples: 2
sample_retract_dist: 2.0

[servo klicky_servo]
pin: P2.0  # 舵机信号引脚
maximum_servo_angle: 180
minimum_pulse_width: 0.0006
maximum_pulse_width: 0.0024
```

#### 4. 校准流程
1. 手动测量探针偏移 (x_offset, y_offset)
2. 使用 `PROBE_CALIBRATE` 命令校准Z偏移
3. 测试重复精度: `PROBE_ACCURACY`
4. 保存配置: `SAVE_CONFIG`

### 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 探针无法吸附 | 磁铁极性相反 | 翻转磁铁或调换位置 |
| 触发不稳定 | 开关接触不良 | 检查焊接/接线 |
| Z偏移漂移 | 探针安装不牢 | 加固磁铁或检查开关 |

## 二、Klipper 固件刷写

### 准备工作
1. 准备上位机 (树莓派/香橙派等)
2. 准备SD卡 (32GB以上，Class 10)
3. 记录当前主板型号和引脚定义

### 安装步骤

#### 1. 上位机系统安装

**树莓派:**
```bash
# 下载 Raspberry Pi Imager
# 选择 Raspberry Pi OS Lite (64-bit)
# 开启SSH: 在boot分区创建空文件名为 "ssh"
```

**香橙派:**
```bash
# 下载 Armbian 或官方镜像
# 使用 BalenaEtcher 烧录
```

#### 2. 安装Klipper及相关组件

```bash
# 连接到上位机 (SSH)
ssh pi@raspberrypi.local  # 默认密码: raspberry

# 安装KIAUH (Klipper安装助手)
cd ~
git clone https://github.com/th33xitus/kiauh.git
./kiauh/kiauh.sh

# 在菜单中选择安装:
# - Klipper
# - Moonraker
# - Fluidd 或 Mainsail
```

#### 3. 编译主板固件

```bash
# 进入Klipper目录
cd ~/klipper

# 进入配置菜单
make menuconfig
```

**大鱼TT主板配置:**
- MCU: LPC1768
- Clock: 100MHz
- Bootloader: 16KiB
- Serial: USB

```bash
# 编译
make clean
make

# 生成的固件在: ~/klipper/out/klipper.bin
```

#### 4. 刷写固件到主板

**方法A: SD卡刷写**
1. 将 `klipper.bin` 复制到SD卡
2. 重命名为 `firmware.bin`
3. 插入主板SD卡槽
4. 重启主板 (会自动刷写)

**方法B: 直接刷写 (如主板支持)**
```bash
# 通过DFU模式或串口直接刷写
# 具体命令因主板而异
```

#### 5. 配置printer.cfg

```bash
# 复制示例配置
cd ~/printer_data/config
# 根据你的主板型号编辑 printer.cfg
```

**获取MCU串口:**
```bash
ls /dev/serial/by-id/
# 输出类似: usb-Klipper_lpc1768_12345-if00
```

#### 6. 启动Klipper

```bash
# 重启Klipper服务
sudo systemctl restart klipper

# 查看日志
journalctl -u klipper -f
```

### 验证安装

1. 访问 Fluidd/Mainsail 网页界面
2. 确认MCU连接状态为 "就绪"
3. 执行 `G28` 测试归位
4. 测试加热: `M104 S200` (热端) / `M140 S60` (热床)

## 三、挤出机升级安装

### 泰坦挤出机安装

#### 1. 拆卸原挤出机
1. 断电
2. 拆除原挤出机固定螺丝
3. 断开电机线和风扇线
4. 取出送料管

#### 2. 安装泰坦挤出机
1. 安装挤出机支架到原位置
2. 连接步进电机 (注意线序)
3. 安装送料齿轮 (清洁齿轮表面)
4. 调整齿轮间隙 (刚好夹住 filament)

#### 3. 校准 E-steps

**计算新e-steps:**
```
新e-steps = 原e-steps × (3/1)  # 泰坦是3:1减速
```

**校准流程:**
1. 标记 filament 100mm 和 120mm 位置
2. 发送 `G1 E100 F100` 挤出100mm
3. 测量实际挤出长度
4. 调整 rotation_distance:
```ini
rotation_distance = 原值 × (实际挤出/100)
```

## 四、PEI平台安装

### 安装步骤

1. **清洁热床**
   - 使用酒精清洁热床表面
   - 确保无油污和灰尘

2. **安装弹簧钢板**
   - 对齐热床和钢板
   - 用夹子或磁吸固定

3. **粘贴PEI片**
   - 撕下PEI背面保护膜
   - 从一端慢慢贴合，避免气泡
   - 用刮板挤出气泡

4. **调平**
   - 重新运行网床补偿校准
   - 调整Z偏移

### 使用技巧
- 打印前用酒精擦拭PEI
- 打印ABS/ASA时，PEI可加热到100-110°C
- 冷却后轻轻弯曲钢板即可取下模型

## 五、主板升级安装

### GEMINI/JOKER 主板安装

#### 1. 拆卸原主板
1. 拍照记录所有接线
2. 逐个拔下连接器
3. 拆卸固定螺丝

#### 2. 安装新主板
1. 用铜柱固定主板
2. 按照片连接各部件:
   - 步进电机
   - 热端加热棒、热敏电阻
   - 热床加热、热敏
   - 风扇
   - 限位开关
   - 探针 (如有)

#### 3. 配置固件
1. 根据新主板型号编译Klipper
2. 刷写固件
3. 更新 printer.cfg 引脚定义

## 六、电源升级 (12V→24V)

### 注意事项
⚠️ **24V和12V设备不能混接!**

### 升级清单
需要同步升级的部件:
- [ ] 热床加热棒 → 24V规格
- [ ] 热端加热棒 → 24V规格
- [ ] 风扇 → 24V规格 (或加装降压模块)
- [ ] 接近开关 → 24V规格 (或使用分压电路)

### 安装步骤

1. **断开原电源**
   - 关闭电源开关
   - 拔掉电源线
   - 等待电容放电

2. **拆除原电源**
   - 拆卸电源固定螺丝
   - 断开所有输出线
   - 记录各线电压和用途

3. **安装24V电源**
   - 固定新电源
   - 连接主电源线 (L/N/GND)
   - 按颜色/标签连接各输出

4. **更换12V设备**
   - 逐一更换为24V规格
   - 特别注意加热棒和风扇

5. **测试**
   - 用万用表测量输出电压
   - 逐步上电测试各功能

## 七、外壳安装

#### 材料清单
- 3mm亚克力板若干
- M3螺丝螺母

- 磁吸 (门闭合用)

1. 先安装底部框架
2. 安装侧板和背板
3. 顶部留散热口或安装风扇
4. 前门安装合页和磁吸
5. 在顶部或侧面安装活性炭过滤器 (打印ABS时)

## 八、故障排查速查

### 通用排查流程

```
1. 现象描述 → 2. 可能原因 → 3. 检查方法 → 4. 解决方案
```

### 常见问题

| 问题 | 检查项 | 解决 |
|------|--------|------|
| 无法归位 | 限位开关接线 | 检查开关LED是否触发 |
| 挤出异常 | 挤出机齿轮间隙 | 调整间隙或清理 |
| 温度波动 | PID参数 | 重新运行PID校准 |
| 打印层纹 | 皮带张紧度 | 调整皮带松紧 |
| 首层不粘 | 网床/Z偏移 | 重新校准网床和Z偏移 |
