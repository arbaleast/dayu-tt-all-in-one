# 大鱼TT 软件与固件配置

## 一、固件选择

### 1.1 Klipper (强烈推荐)

**为什么选择Klipper：**
- 输入整形 (Input Shaping) - 消除共振，提升速度
- 网床补偿 (Bed Mesh) - 完美首层
- 压力提前 (Pressure Advance) - 消除边角 blob
- 远程控制 - 通过网页界面操作
- 配置简单 - 修改配置只需重启，无需重新刷固件

**硬件要求：**
- 上位机: 树莓派/香橙派/旧笔记本等
- 下位机: 原装主板即可

### 1.2 Marlin
- 原厂固件，稳定
- 无需上位机
- 配置修改需要重新编译刷写

### 2.5 压力提前 (Pressure Advance)

```ini
[extruder]
# ... 其他配置 ...
pressure_advance: 0.04  # 需校准
pressure_advance_smooth_time: 0.040
```

**校准方法：**
1. 打印压力提前测试模型
2. 观察转角，找到最佳值
3. 或使用命令: `TUNING_TOWER COMMAND=SET_PRESSURE_ADVANCE PARAMETER=ADVANCE START=0 FACTOR=.005`

## 六、切片软件配置

### 6.1 PrusaSlicer / SuperSlicer 推荐设置

**打印机设置:**
- 床尺寸: 250x250x250 (根据实际)
- G-code 风格: Klipper
- 回抽长度: 2-3mm (近程) / 4-6mm (远程)
- 回抽速度: 30-40mm/s

**速度设置:**
- 外围: 60-80mm/s
- 填充: 100-150mm/s
- 空驶: 150-200mm/s

**Klipper 特定起始G-code:**
```gcode
START_PRINT EXTRUDER_TEMP=[first_layer_temperature] BED_TEMP=[first_layer_bed_temperature]
```

**结束G-code:**
```gcode
END_PRINT
```

### 6.2 Cura 配置
- 机器类型: 自定义 FFF
- G-code 风格: Marlin (Cura不原生支持Klipper)
- 在G-code中添加Klipper特定命令

## 七、常用Klipper命令

```bash
# 重启Klipper
RESTART

# 固件重启
FIRMWARE_RESTART

# 归位
G28

# 网床补偿
BED_MESH_CALIBRATE
BED_MESH_PROFILE SAVE=default

# 输入整形测试
SHAPER_CALIBRATE

# PID自整定
PID_CALIBRATE HEATER=extruder TARGET=200
PID_CALIBRATE HEATER=heater_bed TARGET=60

# 保存配置
SAVE_CONFIG
```

## 八、故障排除

### 8.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 启动报错 "mcu" | 串口配置错误 | 检查 `ls /dev/serial/by-id/` |
| 打印共振 | 输入整形未启用 | 启用并校准 input_shaper |
| 首层不粘 | 网床未校准 | 运行 BED_MESH_CALIBRATE |
| 边角 blob | 压力提前未设置 | 校准 pressure_advance |
| 挤出不足 | 温度过低或回抽过长 | 检查温度和回抽设置 |
