## 一、为什么高速打印需要振动控制

### 1.1 高速打印的核心矛盾

当打印机以 300mm/s 以上速度运行时，机械系统会面临严峻挑战：

```
高速打印的主要问题：
  1. 皮带弹性 → 高速变向时皮带弹跳
  2. 框架共振 → 铝型材框架固有频率被激发
  3. 滑块惯性 → 高速启停时惯性力传导
  4. 电机脉冲 → 步进电机产生的振动频率

结果：打印件出现"鬼影"（Ringing /ghosting）
  → 表面出现规律的波纹
  → 严重影响打印质量
```

### 1.2 鬼影的产生机理

```
打印头在拐角处快速减速 → 停止 → 反向加速

皮带系统：                          框架系统：
  皮带弹性储存能量                    框架惯性
       ↓                                  ↓
  能量释放 → 滑块过冲             ←    框架振动
       ↓                                  ↓
  实际轨迹：█ ← 过冲位置        框架：∿∿∿ 共振
       ↓                                  ↓
  打印表面：∿∿∿ 规律波纹             同样产生鬼影
```

### 1.3 振动频率测量

每台打印机都有其**固有频率**（Natural Frequency），当外界激励频率接近固有频率时，振幅会被放大（共振）。

```bash
# 使用 ADXL345 测量振动频率
TEST_RESONANCES axes=x
TEST_RESONANCES axes=y

# 输出示例：
# X axis: peak at 53.2 Hz
# Y axis: peak at 38.4 Hz
```

## 二、输入整形（Input Shaping）详解

### 2.1 Input Shaping 原理

**Input Shaping（输入整形）** 是一种**主动振动控制技术**，通过在控制信号（电机脉冲序列）中添加"整形波"，主动抵消系统共振。

```
传统控制信号：        Input Shaped 信号：
  ▲                   ▲
  │                   │
  │                   │    ╱╲
  │                   │   ╱  ╲      ╱╲
──┴──────────────→   ──┴──╱────╲────╱──╲─────→
                    ↑
              添加的反向脉冲
              抵消系统共振
```

**核心原理：**

```
1. 测量系统固有频率（f₀）
2. 设计一个"整形器"（Shaper）
3. 将控制信号分解为多个子脉冲
4. 子脉冲的振动相互抵消
5. 机械系统平稳响应，无超调
```

### 2.2 常用 Input Shaper 类型

| 整形器类型 | 适用场景 | 特性 |
|-----------|---------|------|
| **EI**（2-hump EI）| 高刚性机器 | 抑制效果最强，但响应最慢 |
| **MZV**（Multiple Zero Vibration）| 中等刚性（大多数 Voron）| 平衡效果好 |
| **ZV**（Zero Vibration）| 高刚性低阻尼系统 | 响应最快但抑制较弱 |
| **陷波滤波**（Notch Filter）| 已知共振频率 | 针对性抑制特定频率 |

### 2.3 Klipper 中的 Input Shaping 配置

```python
# printer.cfg 配置示例

[input_shaper]
# X 轴共振频率（Hz）
shaper_freq_x: 53.2
# Y 轴共振频率（Hz）
shaper_freq_y: 38.4
# 整形器类型
shaper_type_x: mzv
shaper_type_y: mzv

# 高级参数（通常不需要调整）
shaper_damping_x: 0.1
shaper_damping_y: 0.1
```

### 2.4 Input Shaping 效果实测

| 参数 | 无 Input Shaper | 有 Input Shaper |
|------|----------------|-----------------|
| 最大安全加速度 | 2000-3000 mm/s² | **8000-15000 mm/s²** |
| 鬼影幅度 | 明显 | 减少 80-95% |
| 打印速度 | 50-80mm/s | **150-300mm/s** |
| 拐角质量 | 过冲、圆角 | 清晰、锐利 |

## 三、Voron 官方推荐的振动测量流程

### 3.1 硬件准备

**必需硬件：**

| 配件 | 规格 | 用途 | 成本 |
|------|------|------|------|
| ADXL345 | 加速度计模块 | 测量振动频率 | ¥15 |
| 杜邦线 | 4-5 根 | 连接 SPI 接口 | ¥2 |
| 3D 打印安装座 | — | 固定 ADXL345 | ¥0 |

**ADXL345 引脚连接：**

```
ADXL345          →  树莓派 / MCU
─────────────────────────────────
VCC (3.3V)       →  3.3V
GND              →  GND
SDA (SDI)        →  I2C SDA（或 SPI MOSI）
SCL (SCK)        →  I2C SCL（或 SPI CLK）
CS (CSB)         →  SPI 片选（高电平 = I2C 模式，低电平 = SPI 模式）
```

> ⚠️ ADXL345 有两种模式：I2C（默认）和 SPI。Klipper 推荐 I2C 模式（更简单）。

### 3.2 安装 ADXL345

```
推荐安装位置：打印头滑块上（最接近振动源）
  ↓
Y 轴安装：ADXL345 水平放置，Z 轴垂直向下
X 轴安装：ADXL345 侧面安装，测量 X 方向振动
  ↓
安装方向记录：
  打印头移动正方向 = 测量正方向
  这对于 Klipper 正确计算频率至关重要
```

3D 打印安装座设计参考：Voron 官方 GitHub（ADXL345 mount for 2020 frame）

### 3.3 Klipper 配置

```python
# printer.cfg 添加

[mcu rpi]
# 树莓派作为 MCU（如果使用 RPi 作为辅助 MCU）
# 如果直接用主控板接 ADXL345，跳过此段

[adxl345]
# ADXL345 加速度计配置
# cs 是片选引脚（对于 I2C 模式使用 I2C 地址）
spi_bus: spidev0.0
# 或 I2C 模式：
# i2c_address: 83

[resonance_tester]
# 共振测试配置
accel_chip: adxl345
# 测试时移动的轴（x 或 y）
probe_points:
    150, 150, 20  # 打印床中心上方 20mm
```

### 3.4 测量流程

**Step 1：确认连接**

```bash
# 在 Klipper 控制台检查 ADXL345 是否识别
ACCELEROMETER_QUERY
# 期望输出：accel chips: adxl345, values: [x, y, z]
```

**Step 2：测试 X 轴共振**

```bash
# 在 Klipper 控制台执行
TEST_RESONANCES axes=x
# 等待约 30 秒
# 输出：X 轴频率响应图 + 推荐 shaper
```

**Step 3：测试 Y 轴共振**

```bash
TEST_RESONANCES axes=y
# 等待约 30 秒
# 输出：Y 轴频率响应图 + 推荐 shaper
```

**Step 4：应用校准结果**

```bash
# Klipper 会输出建议的配置参数，格式如下：
# 建议将以下内容添加到 printer.cfg：

[input_shaper]
shaper_freq_x: 53.2
shaper_type_x: mzv
shaper_freq_y: 38.4
shaper_type_y: mzv
```

**Step 5：验证效果**

```bash
# 测试打印一个 ringing 测试件
# 观察表面波纹是否明显减少

# 如果仍有振纹，尝试调整 shaper_type：
# 从 mzv 切换到 ei 或陷波滤波
```

## 四、ADXL345 加速度计深度使用

### 4.1 ADXL345 规格

```
型号：ADXL345
接口：I2C / SPI
量程：±2g / ±4g / ±8g / ±16g（可选）
分辨率：10-bit（默认）/ 13-bit（高分辨率模式）
采样率：100-3200 Hz（可配置）
功耗：23μA（正常工作）
工作电压：2.0-3.6V（逻辑电平 3.3V）
```

### 4.2 ADXL345 安装注意事项

**安装方向影响测量结果：**

| 安装方向 | 测量轴 | 典型频率范围 |
|---------|--------|------------|
| 水平放置 | X/Y 平面振动 | 30-100 Hz |
| 垂直放置 | Z 轴振动 | 50-150 Hz |

**安装位置影响测量结果：**

```
最佳位置：打印头滑块上
  → 最近振动源，信号最准确
  → 避免框架其他部位的振动干扰

次选位置：热床框架上
  → 信号衰减
  → 可能被框架共振频率污染
```

### 4.3 常见问题排查

**问题：ADXL345 无法识别**

```bash
# 检查接线
# I2C 模式：SDA + SCL + 3.3V + GND 必须都连接正确
# 检查是否有其他 I2C 设备冲突

# 检查 I2C 地址（ADXL345 默认 0x53）
# 在树莓派上执行：
i2cdetect -y 1
# 期望看到 53 地址
```

**问题：测量数据噪声大**

```bash
# 原因：采样率设置不当或电磁干扰
# 解决：
# 1. 降低 Klipper 的 [resonance_tester] 的 sample_count
# 2. 使用更短的杜邦线
# 3. 远离电机和电源线
```

## 五、皮带张力自动调节器

### 5.1 为什么要自动调节张力

```
皮带系统的核心问题：
  1. 皮带会蠕变（长时间运行伸长）
  2. 温度变化导致皮带热胀冷缩
  3. 张力不均 → 皮带两侧运动不同步
  4. 张力下降 → 皮带跳齿风险增加

传统方案：手动调节
  → 需要工具（内六角扳手）
  → 调节后需要重新校准 Input Shaper
  → 不适合长时间无人值守打印

自动调节器：恒定张力，补偿蠕变和热胀
```

### 5.2 弹簧自动张紧器

**原理：** 弹簧提供恒定的拉力，自动补偿皮带伸长：

```
     弹簧
      ↕（恒定力 F）
    ──┬──
      │  ← 滑块
      │
    皮带
      │
    ──┴──
     张紧轮
```

**参数计算：**

```
弹簧刚度选择：
  所需张力 T ≈ 15-25N（对于 GT2 6mm 皮带）
  弹簧压缩量 ΔL ≈ 5-10mm（允许行程）
  弹簧刚度 k = T / ΔL ≈ 2-5 N/mm

推荐弹簧规格：
  线径：0.8-1.0mm
  中径：6-8mm
  自由长度：20-30mm
  圈数：10-15 圈
```

### 5.3 Voron 官方 AWD 套件的张紧设计

LDO CNC AWD Kit 中包含的张紧机构：

```
螺纹微调 + 弹簧的双重机制：
  → 螺纹微调：粗调（首次安装）
  → 弹簧：自动补偿（运行时）

优势：
  1. 安装时容易设定初始张力
  2. 运行时自动保持张力恒定
  3. 配合 AWD 双皮带设计，效果加倍
```

### 5.4 张力测量方法

**频率法（Voron 标准）：**

```
原理：皮带张力 T 与振动频率 f 的关系
  T = 4 × m × L² × f² / g

其中：
  m = 皮带线密度（GT2 6mm ≈ 0.015 kg/m）
  L = 皮带跨距（mm）
  f = 振动频率（Hz）
  g = 重力加速度（9.81 m/s²）

实际操作：
  1. 拨动皮带，使其发出声音
  2. 使用手机 APP 测量振动频率
  3. 张紧皮带至 > 100Hz
```

**推荐频率：**

| 皮带类型 | 最低频率 | 推荐频率 |
|---------|---------|---------|
| GT2 6mm | 80Hz | 100-120Hz |
| GT2 9mm | 100Hz | 120-150Hz |
| HTD 3M | 120Hz | 150-180Hz |

## 六、综合调校流程

### 6.1 完整调校路线图

```
Step 1: 硬件检查
  □ 皮带张力充足（>100Hz）
  □ 惰轮转动顺畅，无卡顿
  □ 电机固定牢固，无松动
  □ 框架无明显晃动

Step 2: Input Shaper 校准
  □ 安装 ADXL345
  □ 配置 Klipper
  □ TEST_RESONANCES axes=x
  □ TEST_RESONANCES axes=y
  □ 应用校准结果

Step 3: 参数微调
  □ max_accel 逐步提升（每次 +1000 mm/s²）
  □ 打印 ringing 测试件
  □ 观察鬼影是否消除
  □ 找到临界加速度值

Step 4: 长期稳定性
  □ 长时间打印测试
  □ 监控电机温度
  □ 定期检查皮带张力
```

### 6.2 调校参数参考

```yaml
# printer.cfg 参考配置（TT310 AWD 改装后）

[printer]
max_velocity: 500        # 最大速度 mm/s
max_accel: 20000        # 最大加速度 mm/s²（可提升至 50000+）
max_accel_to_decel: 15000  # 最大减速度
square_corner_velocity: 5   # 拐角速度（影响拐角质量）

[input_shaper]
shaper_freq_x: 53       # 需要实际测量
shaper_type_x: mzv      # 需要实际校准
shaper_freq_y: 38       # 需要实际测量
shaper_type_y: mzv       # 需要实际校准
shaper_damping_x: 0.1    # 通常默认
shaper_damping_y: 0.1    # 通常默认
```

## 七、总结

高速打印的振动控制是**软硬件协同优化**的系统工程：

**硬件层面：**
- 皮带张力自动调节（弹簧张紧器）
- 框架刚性增强（2040 梁 + 对角撑）
- AWD 四驱（消除皮带侧向跳动）

**软件层面：**
- Input Shaping（主动振动抑制）
- Klippain Shaketune（自动化校准工具）
- ADXL345 实时振动监测

**预期效果（综合优化后）：**

| 参数 | 优化前 | 优化后 |
|------|--------|--------|
| 最高速度 | 150mm/s | 300-500mm/s |
| 最大加速度 | 3000mm/s² | 20000-50000mm/s² |
| 鬼影/振纹 | 明显 | 减少 80-95% |
| 打印时间节省 | — | 30-50% |

> 振动控制是高速打印的基础设施，建议在改装硬件之前先完成 Input Shaper 校准（成本仅 ¥15），以充分释放现有设备的性能。
