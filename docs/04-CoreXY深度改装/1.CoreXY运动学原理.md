## 一、CoreXY 工作原理

### 1.1 核心概念

CoreXY 是一种并联运动学（Parallel Kinematics）结构，由 MIT 在 1993 年提出。其核心思想是：**两个电机协同控制 X/Y 两个自由度**，而非每个轴各用一台电机。

### 1.2 双电机 + 同步带工作机制

```
传统 XY 台式：
  X 轴 → 1 个电机独立驱动
  Y 轴 → 1 个电机独立驱动

CoreXY：
  电机 A + 电机 B → 共同驱动 X 轴和 Y 轴
```

**运动学方程：**

```
X 实际位置 = ½(皮带A拉伸量 + 皮带B拉伸量)
Y 实际位置 = ½(皮带A拉伸量 - 皮带B拉伸量)
```

**具体运动分解：**

| 运动方向 | 电机 A (M1) | 电机 B (M2) |
|---------|------------|------------|
| 只走 X | 正转 | 反转 |
| 只走 Y | 正转 | 正转 |
| 对角线 X+Y | 电机 A 正转 | 静止 |

**皮带路径图（标准 CoreXY）：**

```
         电机A
           │
    ┌──────┴──────┐
    │  ●────────● │  ← 顶部惰轮
    │  │        │  │
    │  │   [滑块]│  │  ← 打印头滑块
    │  │        │  │
    │  ●────────● │  ← 底部惰轮
           │
         电机B
```

两根皮带（A 轨和 B 轨）在滑块处呈 90° 交叉，通过电机正反转的线性组合实现任意 XY 方向运动。

### 1.3 与 CoreXZ 的区别

| 特性 | CoreXY | CoreXZ |
|------|--------|--------|
| 运动平面 | XY 平面 | XZ 平面 |
| 典型应用 | 桌面级高速机 | 高速机（Voron Switchwire）|
| Z 轴 | 独立 Z 轴 | 随 X 动（悬臂结构）|

## 二、与笛卡尔打印机的对比

### 2.1 笛卡尔（Cartesian）结构

```
     Y 轴电机
        │
   ┌────┴────┐
   │ ● ─────● │  ← Y 轴同步带
   │ │        │
 X │ │        │
轴 │ [滑块]  │  ← 打印头
电 │ │        │
机 │ ● ─────● │
   └─────────┘
     X 轴
```

笛卡尔打印机的 X 轴和 Y 轴**完全独立**，每个轴由各自的电机驱动。

### 2.2 核心差异对比

| 对比项 | 笛卡尔 | CoreXY |
|--------|--------|--------|
| 电机数量 | 3-4 个（X/Y/Z/E）| 2 个（X/Y）|
| 移动质量 | X 轴带电机 | 只有滑块，无电机 |
| 加速度上限 | ~3000 mm/s² | ~20000+ mm/s² |
| 最高速度 | ~150mm/s | 600-1000mm/s |
| 机械复杂度 | 低 | 中 |
| 成本 | 低 | 中 |
| 皮带回弹/振动 | 较少 | 需要输入整形 |

### 2.3 为什么 CoreXY 更快？

**减少移动质量**是核心：

- 笛卡尔：X 轴滑块上绑着电机 + 散热块 + 热端 → 重量大
- CoreXY：X/Y 电机固定在框架上，不随打印头移动 → 滑块极轻

> "All Voron printers are built using CoreXY or CoreXZ configurations to reduce the amount of moving mass, allowing increased acceleration and speeds."
> — Voron Design 官方文档

## 三、大鱼 TT 当前 CoreXY 结构分析

### 3.1 推测规格

根据参考文档，大鱼 TT 当前配置推测如下：

| 项目 | 推测规格 | 备注 |
|------|---------|------|
| 运动系统 | CoreXY | 运动学原理验证 |
| 框架 | TT150/TT235：2020；TT310：2020/2040 | 槽宽 6mm |
| 皮带 | GT2 6mm | 可能是标准配置 |
| 线性导轨 | MGN12 或光轴 | 待确认 |
| 电机 | 42 步进电机 | 标准 NEMA17 规格 |
| 固件 | Klipper | 已确认 |

### 3.2 现有 CoreXY 结构特点

**优势：**
- 结构紧凑，继承 Voron CoreXY 轻量化理念
- Klipper 固件支持 CoreXY 运动学补偿
- 有改装空间，可升级到 AWD 四驱

**潜在瓶颈：**

```
皮带系统：GT2 6mm 皮带
  → 高速时皮带弹性明显
  → 反向间隙（backlash）影响精度
  → 建议升级 HTD 3M 或 Gates 9mm

框架刚性：2020 铝型材
  → X 梁弯曲刚性偏低
  → 建议替换 2040 或加对角撑

导轨：光轴 vs MGN 导轨
  → 光轴摩擦力大，高速响应差
  → 建议升级 MGN9C/MGN12
```

### 3.3 大鱼 TT vs Voron 标准规格

| 对比项 | 大鱼 TT（推测）| Voron Trident 标准 |
|--------|--------------|-------------------|
| 框架 | 2020 铝型材 | 2020 铝型材 |
| 皮带 | GT2 6mm | **GT2 9mm** |
| 导轨 | MGN12 或光轴 | MGN12H（Y）/ MGN9C（X）|
| 电机数 | 2 | 2（标准）/ 4（AWD）|

**结论：** 大鱼 TT 基础架构接近 Voron 设计，是合格的 CoreXY 平台。主要差距在皮带宽度、导轨精度和电机驱动方式。

## 四、运动学关键参数

### 4.1 皮带传动比

CoreXY 标准配置使用 16T 或 20T 同步带轮：

```
20T GT2 带轮 → 节圆直径 = 40mm
每转行程 = 40mm × π ≈ 125.6mm

步进角距 = 1.8°（标准 42 电机）
每毫米步数 = 200 脉冲 / 125.6mm ≈ 1.59 pulse/mm（1/16 微步）
```

### 4.2 加速度限制因素

```
加速度上限 ≈ 马达扭矩 / 移动质量

CoreXY 优势：
  - 移动质量只有滑块（~200g）
  - 马达固定在框架上，不参与往复运动
  - 所以加速度可以很高（20000-100000 mm/s²）
```

## 六、总结

CoreXY 运动学是当前桌面级高速 3D 打印的主流方案，其核心优势在于**将电机固定化、移动质量最小化**，从而实现极高的加速度和速度。大鱼 TT 采用 CoreXY 结构，理论上具备改装到 600-1000mm/s 高速打印的潜力。

关键改装方向：
1. **皮带系统**：6mm → 9mm 或 HTD 3M
2. **框架刚性**：2020 → 2040 + 对角撑
3. **运动控制**：Klipper Input Shaper + AWD 四驱
