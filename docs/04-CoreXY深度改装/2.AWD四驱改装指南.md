## 一、AWD（All-Wheel Drive）原理

### 1.1 标准 CoreXY 的问题

标准 CoreXY 使用**两台电机**分别驱动 X 轴和 Y 轴，电机固定在框架两侧，通过同步带控制滑块位置。这种设计存在以下固有缺陷：

```
问题一：皮带张力不均匀
  → 电机安装在框架两侧，皮带绕线路径不同
  → 长时间运行后，一侧皮带张力衰减更快
  → 导致打印精度逐渐下降

问题二：皮带跳齿
  → 高速加速时，皮带与带轮啮合不稳定
  → 极限加速度下容易跳齿，产生定位误差
  → 限制最高速度和加速度

问题三：皮带弹性反作用
  → GT2 皮带本身有弹性，高速变向时产生弹跳
  → 皮带回弹带动滑块位移，产生鬼影（ringing）
```

### 1.2 AWD 四驱如何解决

**AWD（All-Wheel Drive）** 将标准 CoreXY 的 **X 轴 1 电机 + Y 轴 1 电机** 升级为 **X 轴 2 电机 + Y 轴 2 电机**，共 4 个电机同步驱动：

```
标准 CoreXY：
  M1 ─┐
       ├─→ X 皮带轨
  M2 ─┘

AWD CoreXY：
  M1 ─┐     ┌─ M3
       ├─ X ─┤
  M2 ─┘     └─ M4
           (同步驱动)
```

**核心改进：**

| 改进项 | 标准 CoreXY | AWD CoreXY |
|--------|------------|------------|
| 电机数量 | 2 | **4** |
| 皮带数 | 2（X轨 + Y轨）| 4（X轨双皮带 + Y轨双皮带）|
| 皮带张力 | 单侧，易衰减 | **双侧对称，自动平衡** |
| 跳齿风险 | 高 | **极低** |
| 定位精度 | ±0.05mm | **±0.02mm** |

### 1.3 四电机同步原理

AWD 四驱使用**同步带均布设计**：每根运动轴由两条平行皮带驱动，两台电机通过机械同步（或电子同步）同时拉紧皮带。

```
          电机A1          电机A2
            │    皮带 A1   │
      ●─────┼──────────────┼─────●
      │ 惰轮│              │惰轮│
      ●─────┼──────────────┼─────●
            │    皮带 A2   │
            ▼              ▼
          ←───── X 轴运动方向 ─────→
```

两台电机（M1 + M3）通过同一根长皮带或电子同步驱动，对皮带的张力平衡有自我调节作用。

## 三、改装步骤（适配大鱼 TT 310mm 机型）

> ⚠️ 改装难度：中高级。建议有 Klipper 配置经验的用户操作。

### 3.1 改装前准备

**工具清单：**

| 工具 | 用途 |
|------|------|
| 内六角扳手套装 | 拆卸/安装螺丝 |
| 活动扳手 | 螺母紧固 |
| 尖嘴钳 | 弹簧安装 |
| 万用表 | 电机接线检测 |
| 磁铁/镊子 | 小零件拾取 |
| 3D 打印零件胶 | 固定螺丝胶 |

**配件检查清单：**

- [ ] LDO CNC AWD Kit × 1 套
- [ ] LDO 42 步进电机 × 2 个（额外购买）
- [ ] GT2 9mm 皮带（大鱼 TT 310mm 行程用 ~1米 × 4）
- [ ] F695 轴承 × 4 个（套件自带）
- [ ] M3/M4 螺丝胶

### 3.2 改装步骤

#### Step 1：拆卸原有 CoreXY 电机组件

```
⚠️ 操作前：断开打印机电源，移除电源线
```

1. 拆下 X 轴和 Y 轴电机固定座
2. 记录电机线序（通常：红、蓝、绿、黄 或 黑、绿、红、蓝）
3. 拍照记录皮带路径（避免安装时皮带走向错误）
4. 拆卸旧皮带和惰轮

#### Step 2：安装 AWD 电机座

**大鱼 TT 310mm 适配说明：**

大鱼 TT 310mm 使用 2020 铝型材框架，**LDO AWD 电机座专为 2020 型材设计，理论上兼容**。但需注意：

```
适配性检查点：
  1. 框架内宽 ≥ 310mm（需实测）
  2. 电机座安装孔距 = 31mm × 31mm（NEMA17 标准）
  3. 皮带走线路径是否与 TT310 框架干涉
```

安装方式：

1. 将电机座插入 2020 型材槽内
2. 使用 M5 T 型螺母 + 内六角螺丝固定
3. 确认电机座与框架垂直（误差 < 0.5°）

#### Step 3：安装惰轮和张紧机构

```
推荐惰轮排列：smooth → toothed → smooth
  即：光轴惰轮 → 同步带轮 → 光轴惰轮
```

1. 将 F695 轴承压入惰轮孔位（使用轴承压装工具或台钳）
2. 安装惰轮组件到电机座对应位置
3. 安装弹簧张紧器（提供恒定张力）

#### Step 4：安装新电机

1. 将 2 个新 LDO 42 电机安装到 AWD 电机座
2. 确认电机轴方向（电机轴朝向框架内侧）
3. 使用套件螺丝紧固（扭矩：0.3-0.4 N·m）

#### Step 5：穿装 GT2 9mm 皮带

```
皮带穿装顺序：
  1. 先穿 X 轴皮带（2条平行）
  2. 再穿 Y 轴皮带（2条平行）
  3. 确认皮带齿形与带轮啮合正确
```

**关键检查点：**
- 皮带齿形完全啮入带轮（无偏斜）
- 皮带张力：使用频率法测量，张紧至 >100Hz 音调
- 两根平行皮带张力差 < 5%

#### Step 6：电气接线

**电机并联方案（推荐）：**

AWD 四驱有**两种电机接线方案**：

**方案 A：机械同步（简单可靠）**
```
X 轴：M1 和 M3 并联接到 X 接口
Y 轴：M2 和 M4 并联接到 Y 接口
```
→ 两台电机同时收到相同脉冲，机械同步

**方案 B：电子同步（更精确）**
```
X 轴：M1 接 X 接口，M3 接 Y2 接口（第二驱动器）
Y 轴：M2 接 Y 接口，M4 接 E2 接口
```
→ Klipper 中配置多电机同步

**接线步骤：**

1. 确定 4 个电机的相序（使用万用表或参考旧电机接线）
2. 将 X 轴 2 个电机并联（红线接红线，蓝/绿接蓝/绿）
3. 将 Y 轴 2 个电机并联
4. 接入 Klipper 控制板

⚠️ **警告**：并联电机必须型号相同、相序一致，否则会损坏电机或驱动！

#### Step 7：Klipper 配置更新

```python
# printer.cfg 中更新电机配置

# X 轴：2 个电机（双电机同步）
[stepper_x]
step_pin = PF13
dir_pin = PF12
...
# 添加第二电机
[stepper_x]
...

# 如果使用方案 B（电子同步），需要在 config 中添加：
[stepper_x]
...
second_stepper = stepper_x1

[stepper_x1]
step_pin = ...
dir_pin = ...
```

执行 `FIRMWARE_RESTART` 后测试运动是否正常。

## 六、常见问题与解决

### 6.1 皮带张力不均

**症状：** 打印出现规律性波纹，沿某一轴向重复出现。

**原因：** 两根平行皮带张力差异大。

**解决：**
1. 使用频率法重新测量每根皮带张力
2. 调节张紧器螺丝，使张力差 < 5%
3. 检查惰轮是否磨损或偏心

### 6.2 电机发热严重

**症状：** 电机外壳烫手（>60°C）。

**原因：** 电机并联后电流分配不均。

**解决：**
1. 确认电机相序正确（错误相序会导致电流叠加）
2. 检查接线端子是否接触良好
3. 降低 Klipper 中的电机电流（通常设定为额定电流的 80%）

### 6.3 改装后噪音增大

**症状：** 打印时出现高频啸叫。

**原因：** 皮带张力过高或带轮偏心。

**解决：**
1. 适当降低皮带张力（不低于 80Hz 频率）
2. 检查带轮轴承是否润滑不足
3. 添加润滑脂到惰轮轴承

对于大鱼 TT 310mm 机型：
- **框架兼容性**：✅ 2020 型材框架理论上兼容 LDO AWD Kit
- **改装价值**：⭐⭐⭐⭐⭐（最高推荐）
- **改装难度**：⭐⭐⭐（中高级，需一定动手能力）
- **预期效果**：速度 3-5× 提升，加速度 5-10× 提升
- **总成本**：约 ¥350（使用现有电机可更低）

> ⚠️ **重要提示**：改装涉及电机电气连接，务必确认相序正确，否则可能损坏电机或驱动板。建议在有经验的用户指导下操作。
