## 一、2020 vs 2040 铝型材对比

### 1.1 规格参数

2020 和 2040 是最常见的 3D 打印机框架铝型材规格，名称来源于其截面尺寸：

```
命名规则：
  20xx = 20mm × xxmm（宽 × 高）

2020 铝型材：
  宽：20mm
  高：20mm
  槽宽：6mm（标准）

2040 铝型材：
  宽：20mm
  高：40mm
  槽宽：6mm（标准）
```

### 1.2 截面特性对比

| 特性 | 2020 | 2040 | 差异 |
|------|------|------|------|
| 截面积 | 314 mm² | 594 mm² | +89% |
| 理论重量 | ~0.86 kg/m | ~1.6 kg/m | +86% |
| 弯曲刚度 Ix | 28.3 × 10³ mm⁴ | 118.8 × 10³ mm⁴ | **+320%** |
| 扭转刚度 J | 12.5 × 10³ mm⁴ | 58.3 × 10³ mm⁴ | **+366%** |
| 成本 | ¥15/m | ¥30/m | +100% |

### 1.3 刚性提升的物理意义

**弯曲刚度提升 320% 意味着什么？**

```
负载 P 作用下的弯曲变形 δ：

δ = P × L³ / (3 × E × Ix)

其中：
  L = 型材长度
  E = 弹性模量（约 70 GPa，铝）
  Ix = 截面惯性矩

对于相同负载：
  2040 的 Ix 是 2020 的 4.2 倍
  → 2040 的弯曲变形只有 2020 的 1/4.2
  → 即刚度提升 320%
```

### 1.4 选型建议

```
大鱼 TT 各机型框架配置建议：

TT150（150mm 行程）
  → 2020 框架（够用）
  → 理由：打印幅面小，框架受力小
  → 可选升级：XY 轴用 2040（局部加强）

TT235（235mm 行程）
  → 2020 框架（基本够用）
  → 建议：对角斜撑（必做）
  → 可选升级：X 梁用 2040

TT310（310mm 行程）⭐ 强烈推荐 2040 X 梁
  → 理由一：X 轴跨度大，弯曲变形明显
  → 理由二：高速打印（300mm/s+）需要高刚性
  → 理由三：Voron Trident 标准配置
  → 改装：仅替换 X 梁为 2040（成本 ¥30）
```

## 二、角码加固方案

### 2.1 框架柔性分析

```
2020 铝型材框架的薄弱环节：

问题一：XY 平面内的剪切变形
  → 框架四角为直角连接
  → 长期振动导致角部微变形
  → 影响打印精度

问题二：对角线方向柔性
  → 框架在斜向力作用下容易变形
  → 打印头高速运动时产生斜向惯性力

问题三：垂直方向（Z 向）
  → 框架底部支撑不足
  → 打印机整体晃动
```

### 2.2 角码连接器类型

| 类型 | 特点 | 刚度 | 成本 | 推荐度 |
|------|------|------|------|--------|
| 入门级角码 | 塑料，简易插入 | 低 | ¥2 | ⭐ |
| 铝合金角码 | CNC 加工，强度中 | 中 | ¥10 | ⭐⭐⭐ |
| Voron 专用角码 | 原厂设计，高强度 | 高 | ¥25 | ⭐⭐⭐⭐⭐ |
| 3D 打印角码 | PETG/ABS，高可定制 | 中高 | ¥5 | ⭐⭐⭐⭐ |

### 2.3 对角斜撑（Diagonal Bracing）

**原理：** 利用三角形的稳定性，将框架的对角线连接起来：

```
框架俯视图：

不加斜撑：              加斜撑后：
  □ ─ ─ ─ □              □ ─ ─ ─ □
  │       │    →          ╱╲      │
  │       │              ╱  ╲     │
  │       │             ╱    ╲    │
  □ ─ ─ ─ □            □ ─ ─ ─ □

对角撑杆 ↑ 吸收斜向力，防止框架菱形变形
```

**斜撑安装方式：**

| 安装方式 | 刚度提升 | 成本 | 复杂度 |
|---------|---------|------|--------|
| 2020 撑杆 + 角码 | +30-40% | ¥50 | 低 |
| 2040 撑杆 + 角码 | +40-50% | ¥80 | 低 |
| 碳纤维撑杆 | +50-60% | ¥150 | 中 |
| 金属板加固（切割弯折）| +60%+ | ¥100 | 高 |

**安装步骤：**

```
Step 1：测量对角线长度
  → 测量框架两对角距离 A 和 B
  → A ≈ B（正方形框架）
  → 撑杆长度 = √(A² + B²)

Step 2：准备撑杆
  → 2020 撑杆：切割至计算长度
  → 两端安装 2020 角码

Step 3：安装
  → 先安装一侧角码
  → 将撑杆斜插入角码
  → 安装另一侧角码
  → 紧固螺丝

Step 4：检查
  → 确认框架对角线相等
  → 手动推拉测试刚性提升
```

## 三、底部框架加强

### 3.1 底部框架的受力分析

```
打印机底部的受力情况：

静态负载：
  → 热床 + 玻璃/碳纤维板（~5-10kg）
  → 框架自重（~15-20kg）
  → Z 轴电机和导轨

动态负载：
  → 打印头 Y 方向高速运动
  → 热床加速（热床雕刻打印时）
  → 产生的惯性力传递到框架底部

问题：
  → 底部框架刚性不足 → 整体晃动
  → 影响 Z 轴定位精度
  → 打印件底部出现层纹
```

### 3.2 底部加强方案

#### 方案 A：底板加固（低成本）

```
材料：铝板 / 碳纤维板 / 亚克力板

安装方式：
  1. 切割 3-5mm 铝板至框架内尺寸
  2. 放置于框架底部
  3. 使用 M5 螺丝 + T 型螺母固定

效果：
  → 框架刚性提升 20-30%
  → 减少框架"呼吸效应"
  → 成本：¥30-50
```

#### 方案 B：底部型材加固（中成本）

```
在现有框架底部添加横向型材：

结构：
  原本：□（4 根 2020）
  加固：目（底部 + 2 根横撑）

效果：
  → 刚性提升 40-50%
  → 彻底消除底部晃动
  → 成本：¥50（材料）
```

#### 方案 C：封闭式箱体结构（高成本高刚性）

```
将框架底部完全封闭：

结构：
  ┌─────────────────┐
  │  顶框（2020）   │
  │                 │
  │   侧框架（2020） │
  │                 │
  │  底框（2040）  ← 更换 2040
  │  + 底板封闭     │
  └─────────────────┘

效果：
  → 刚性提升 80%+
  → 理论接近 Voron 2.4 级别
  → 成本：¥150
```

### 3.3 底框加固实施指南

```
针对大鱼 TT TT310 机型：

Step 1：评估当前状态
  → 开机打印时框架是否晃动？
  → 热床移动时是否有共振？
  → 打印件底部层纹是否明显？

Step 2：选择加固方案
  → 预算有限：底板加固（¥30）
  → 中等预算：底部型材（¥50）
  → 追求极致：封闭式箱体（¥150）

Step 3：安装
  → 如果是底板：切割合适尺寸，固定到框架底部
  → 如果是型材：在底部槽内插入横向撑杆
  → 确认所有连接紧固

Step 4：验证
  → 手动摇晃框架，确认刚性提升
  → 打印测试件观察底部质量
```

## 四、打印机柜设计参考

### 4.1 为什么需要打印机柜

```
打印机柜的作用：

功能一：隔音
  → 步进电机高频啸叫
  → 风扇噪音
  → 关上门噪音 < 40dB

功能二：温控
  → 减少热端热量散失
  → 减少热床热量损失
  → 提升 PLA 打印质量
  → 减少 ABS 翘曲

功能三：防尘
  → 保护光学器件（BLTouch）
  → 保护皮带和惰轮
  → 减少维护频率

功能四：安全
  → 防止误触热端/热床
  → 减少宠物接触风险
  → 火灾隔离（可选）
```

### 4.2 打印机柜设计要点

| 设计要素 | 要求 | 说明 |
|---------|------|------|
| 材质 | 铝合金 + PVC 板 / 木板 | 框架稳定 + 隔音 |
| 尺寸 | 内腔 > 打印机外尺寸 | TT310 建议 400×400×500mm |
| 门 | 透明亚克力或玻璃 | 方便观察 |
| 通风 | 顶部排风扇 + 侧进风 | 主动散热 |
| 线缆 | 过线槽 / 硅胶密封圈 | 防尘密封 |

### 4.5 打印机柜线缆管理

```
关键要点：
  1. 所有线缆使用拖链（chain）
  2. 避免线缆与运动部件干涉
  3. 预留检修口
  4. 使用理线架固定

推荐线缆规格：
  电机线：22AWG 硅胶线
  热端线：18AWG 硅胶线（加热块）
  热床线：14AWG 硅胶线
  传感器线：杜邦线（24AWG）

注意：
  → 大电流线缆单独走线
  → 避免与信号线平行走线
  → 减少电磁干扰
```

## 六、日常维护与检查

### 6.1 框架检查周期

```
每 500 打印小时：
  □ 检查所有螺丝是否松动（重点：角码连接）
  □ 检查框架型材是否有变形
  □ 检查角码是否有裂纹
  □ 检查底部支撑是否稳定

每 1000 打印小时：
  □ 紧固所有螺丝（扭矩检查）
  □ 检查对角斜撑是否变形
  □ 检查 2040 X 梁是否有弯曲
  □ 添加螺丝胶到关键连接（可选）
```

### 6.2 常见问题与解决

**问题一：框架晃动**

```bash
原因：
  → 角码螺丝松动
  → 对角斜撑未安装
  → 底部支撑不稳

解决：
  → 紧固所有螺丝
  → 安装对角斜撑（最有效）
  → 添加底部支撑
```

**问题二：角部异响**

```bash
原因：
  → 角码内部有间隙
  → 型材槽口不匹配

解决：
  → 在角码与型材接触面添加垫片
  → 使用螺丝胶固定
  → 更换磨损的角码
```

## 七、总结

框架刚性是高速 3D 打印的**根基**，没有坚实的框架，任何先进的运动控制和 Input Shaping 都无法发挥作用。

**核心改装原则：**

```
改装优先级：
  1. 2040 X 梁（最大刚性提升 / 成本比）
  2. 对角斜撑（消除框架菱形变形）
  3. 底板加固（消除整体晃动）
  4. Voron 角码（专业级连接）

刚性到位后：
  → Input Shaper 校准效果更好
  → 皮带张力可以更高
  → 速度可以更快
  → 打印质量更稳定
```

**大鱼 TT TT310 改装建议总结：**

| 改装项 | 建议 | 成本 | 优先级 |
|--------|------|------|--------|
| X 梁 2040 | **强烈推荐** | ¥30 | P0 |
| 对角斜撑 | **强烈推荐** | ¥50 | P0 |
| 底板加固 | 推荐 | ¥40 | P1 |
| Voron 角码 | 推荐 | ¥100 | P1 |
| 打印机柜 | 可选 | ¥250 | P2 |

> 框架刚性增强是最"值"的改装之一：投入 ¥80-220，就能让整机的性能提升潜力释放出来。这是高速打印的必备基础设施。
