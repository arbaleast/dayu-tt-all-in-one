# 腔体温控与封箱散热指南

> 更新时间：2026-03-25（X补充）
> 适用：大鱼TT全封闭机箱打印ABS/ASA
> 来源：Voron Community Best Practices + X/Twitter 2026最新讨论

---

## 一、为什么腔室温控很重要？

ABS/ASA 等材料需要高温腔室（50-60°C）才能：
- 减少翘曲和开裂
- 提高层间粘合强度
- 保证尺寸精度

腔室温度过高（>60°C）会软化打印机上的 3D 打印件（用于固定支架、线缆夹等），造成结构失效。

---

## 二、腔室理想温度区间

| 用途 | 推荐温度 | 说明 |
|------|---------|------|
| ABS 打印 | 50-60°C | 最佳区间 |
| ASA 打印 | 55-65°C | 略高但别超65°C |
| PLA | 常温即可 | 无需封闭 |
| 安全上限 | **60°C** | 超过会软化打印件 |

## 三、排气扇自动控制（推荐）

### 原理
腔温超过 35°C 时自动开启排气扇，低于阈值时降低转速，保持热平衡。

### Klipper 配置示例

```ini
# 腔室温度传感器（建议安装在热端上方10cm处）
[thermistor]
name: chamber_temp
temperature1: B3950
pin: PG5  # 根据实际接线调整

[sensor_reader chamber]
sensor: thermistor chamber_temp
min_temp: 0
max_temp: 120

# 排气扇控制
[fan]
name: exhaust_fan
pin: PA2  # 根据实际接线调整
kick_start_time: 0.5
off_below: 0.10

# 腔温联动
[delayed_gcode chamber_exhaust]
initial_duration: 0
gcode:
  {% if printer.sensor_reader.chamber.temperature > 35 %}
    SET_FAN_SPEED FAN=exhaust_fan SPEED=1.0
  {% else %}
    SET_FAN_SPEED FAN=exhaust_fan SPEED=0.3
  {% endif %}
```

### 阈值参考
- **35°C 以下**：低速运转（10-30%），保持空气流通
- **35°C 以上**：全速排气
- **50°C 以上**：报警提示（可设置）

## 四、Nevermore 活性炭滤芯系统

Nevermore 是 Voron 社区开源的腔室空气过滤方案，适合封闭打印ABS时去除 VOC。

### 优点
- 无耗材（活性炭可重复使用）

- 与腔温控制兼容

## 五、大鱼TT封箱散热改装要点

### 热床热量利用
封箱后，热床（通常200W）工作时会将腔室加热到 40-50°C，在多数地区无需额外加热。寒冷环境（室温<15°C）可能需要：
- 增加保温层（PEI板贴在箱壁上）
- 外置加热器（如陶瓷加热灯）

### 散热通道设计
- **顶部排气孔**：建议开 2-4 个 50mm 直径孔
- **前进后出**：形成空气对流
- **可调节开度**：用 3D 打印盖板控制通风量

### 滤芯位置建议
- 置于箱体后部角落
- 避免正对热端，影响热端散热
- 进风口朝下，出风口朝上（热气上升）

## 八、故障排查

| 症状 | 可能原因 | 解决方法 |
|------|---------|---------|
| 腔温超过60°C | 排气不足/环境太热 | 增加排气孔，开盖降温 |
| 打印件底部翘曲 | 腔温太低/气流不稳定 | 提高热床温度，检查密封 |
| 有异味残留 | 通风不良/活性炭饱和 | 检查Nevermore，暴晒复用 |
| 层间粘合差 | 腔温波动大 | 减少开门次数，稳定控温 |

## 九、X/Twitter 2026 最新经验（🆕）

**来源：** X/Twitter 3D打印社区讨论（2026年3月）

### Bambu Lab P2S 的启示
Bambu Lab P2S（2026年发布）主打功能与封箱设计高度相关：
- **关舱打印不怕热积压**：封闭环境反而是优势，腔温稳定
- **Adaptive Airflow**：任意耗材自动匹配腔温，风扇转速动态调节
- **AI Monitoring**：实时检测打印异常，防止整夜报废
- 对大鱼TT的参考：**封箱不是缺点，封闭+智能散热才是优势**

### Voron 社区封箱经验
- **热床功率利用**：200W 热床封闭后可持续维持腔温 45-55°C（室温 20°C）
- **密封优先于排气**：先确保四面密封，再设计排气；密封不好排气只是白加热
- **开门打印 vs 关门打印**：ABS/ASA 必须关门打印，PLA 可以开门

### CNC Kitchen（Stefan）的忠告
> "封箱打印时，第一层打印完成后立即关门，中途不要开门，否则温差骤变会导致翘边。"

### 封箱散热黄金原则
1. **先密封后排气** — 密封做不好，排气全浪费
2. **热空气必须能出去** — 35°C 自动触发排气扇全速
3. **冷空气必须能进来** — 底部留进风口，形成对流
4. **Nevermore 过滤VOC** — ABS 打印必备，保护健康
