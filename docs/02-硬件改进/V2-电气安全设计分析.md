# 大鱼TT V2 — 电气安全设计分析

> 分析日期：2026-03-26
> 分类：02-硬件改进 / 电气安全

---

## 一、热失控（Thermal Runaway）— 最核心的火灾成因

### 什么是热失控？

当热端或热床的温度传感器（热敏电阻/热电偶）松脱或失效时，固件看到"温度读数偏低"，就会持续通电加热，最终导致温度失控、塑料融化燃烧。

### 大鱼TT的风险点

- 热床 270W、热端 50W — 功率大，热失控后升温极快
- 封箱后腔温本身已较高（27°C 强制排风后），热失控更难被察觉
- PLA 在 260°C 以上有自燃风险，封箱内无通风会加速蔓延

### 防护方案（三层）

| 层级 | 措施 | 大鱼TT建议状态 |
|---|---|---|
| **固件层** | Marlin/Klipper 的 Thermal Runaway Protection（TRP） | ⚠️ 必须确认固件已开启 |
| **硬件层** | 独立于主控的温度上限保险（热熔丝/热断路器） | ⚠️ 建议增加 |
| **物理层** | 热敏电阻可靠固定（焊死 + 胶固定） | ⚠️ 检查 TT310 原设计 |

### 固件配置要点（Marlin）

```
#define THERMAL_PROTECTION_HOTENDS    // 热端热失控保护
#define THERMAL_PROTECTION_BED         // 热床热失控保护
#define THERMAL_PROTECTION_CHAMBER     // 腔体温控保护 ← 大鱼TT封箱必须开启
#define THERMAL_PROTECTION_COOLER      // 冷却液温控（如果有）
```

## 二、MOSFET 失效与功率电路安全

### 核心问题

控制板上的 MOSFET 同时承受大电流（热床 10-15A）和高热量。失效模式通常是**短路**——MOSFET 击穿后变成导体，持续给负载通电。

### 大鱼TT热床功率估算

| 电压系统 | 工作电流 | MOSFET 压力 |
|---|---|---|
| 24V 热床（270W） | 270W / 24V ≈ **11.3A** | 中等，可用 IRLB8743 等 MOSFET |
| 12V 热床（270W） | 270W / 12V ≈ **22.5A** | 高，MOSFET 发热严重，不推荐 |

> **建议：大鱼TT用 24V 系统，热床回路单独 MOSFET 驱动**

### MOSFET 选型要求

- 额定电流 ≥ 2× 工作电流（留安全余量）
- RDS(ON) 要低（< 10mΩ），减少自身发热
- 必须配合散热片使用
- 推荐型号：IRLb8743（30V/56A）、AON6514（30V/34A）

### PCB 走线设计

- 1oz 铜箔，1mm 宽度 ≈ 1A 电流能力（经验值）
- 11A 电流需要至少 **1.5mm 走线宽度**（双面铺铜更佳）
- 过孔要足够分散大电流
- 焊盘要大，保证热传导到铺铜区

### 热床驱动独立模块（强烈建议）

主板自带 MOSFET 直接驱动大功率热床风险高，建议：
- 使用外置 MOSFET 驱动模块（如典型的 "Kill MOSFET" 板）
- 或使用 **固态继电器（SSR）** 控制 AC 热床（见下一节）

## 三、AC/220V 市电热床 — 高电压安全

### 风险说明

大功率热床（200W+）如果用 DC 低压，需要极粗的导线和巨大的 MOSFET。部分商用机器直接使用 **220V AC 硅胶加热板**，但引入了触电和火灾风险。

### AC 热床安全设计要点

| 问题 | 解决方案 |
|---|---|
| 220V 裸露带电 | 用 SSR（固态继电器）隔离，低压信号控制高压 |
| SSR 失效（短路） | SSR 前加独立热熔丝/断路器 |
| 触电风险 | 所有市电导线用热缩管/硅胶套管，焊接点灌胶密封 |
| 地线连接 | 热床金属外壳必须接地（PE），防止外壳带电 |
| 绝缘层 | 硅胶加热板和铝板之间用高温玻璃纤维绝缘垫 |

### ⚠️ 大鱼TT判断

- 如果用 **24V 热床**：安全性高，但加热慢（270W 在 24V 下约 11A）
- 如果追求快速升温用 **220V AC 热床**：必须 SSR + 独立保险 + 良好绝缘，成本约高 50 元但大幅提升安全性
- **建议**：继续用 24V 低压热床，通过增加保温层（底部铝箔隔热棉）来提升加热效率

## 四、过流保护

### 目的

导线短路时，电流急剧上升，保险丝/断路器在秒级内切断，防止导线过热起火。

### 保护层级

| 保护位置 | 器件 | 典型规格 |
|---|---|---|
| PSU 输入 | AC 保险丝/断路器 | 10A（220V 系统） |
| 12V/24V 主回路 | 自恢复 PPTC 或玻璃管保险丝 | 根据负载总电流 × 1.2 |
| 热床回路 | 玻璃管保险丝（快熔型） | **15A（24V 热床）** |
| 电机回路 | 步进驱动内置保护 | — |

### 保险丝选型原则

- 额定电流 = 正常工作电流 × 1.2 ~ 1.5
- **慢熔型（Slow-blow）**：用于电机、热床（启动电流大）
- **快熔型（Fast-blow）**：用于控制电路
- 常用型号：5×20mm 玻璃管，Bel Fuse 5ST 系列

## 五、过压/欠压/浪涌保护

| 保护类型 | 器件 | 作用 |
|---|---|---|
| TVS 二极管 | SMBJ15CA（双向） | 抑制电源尖峰，保护控制板 |
| 防反接 | 功率二极管或 P-MOSFET | 电源接反保护 |
| UPS 不间断电源 | 500VA+ UPS | 防止打印中断电导致工件报废 |
| 断电监测 | Klipper 内置 + 脚本 | 断电后安全停机 |

## 六、接地与漏电保护

### 框架带电问题

MOSFET 失效、绝缘老化、接地不良时，3D 打印机金属框架可能带电，人触碰会有麻电感，严重时致命。

### 解决方案

1. **保护接地（PE）**
   - 所有金属外壳、电机支架、控制板地都要连接到电源地
   - 接地电阻 < 0.1Ω（用万用表测量）

2. **RCD/GFCI 漏电保护**
   - 家用配电箱加装 **30mA 漏电保护器**
   - 3D 打印机使用单独回路
   - 原理：相线和零线电流差值 > 30mA 即跳闸

3. **定期检查**
   - 每 3 个月用万用表测量外壳对地电阻
   - 检查所有接插件紧固状态

## 七、电缆与接插件 — 机械疲劳风险

### 隐形杀手

3D 打印机运行时电缆反复弯曲（XY 轴移动），时间长了内部导线会断裂，导致：
- 间歇性接触不良 → 打火 → 升温 → 起火
- 断裂瞬间产生电弧（尤其大电流回路）

### 防护措施

| 方案 | 做法 |
|---|---|
| **拖链（Cable Chain）** | 保护活动区域的电缆（Voron 等高端机器标配） |
| **弹簧线缆** | 用弹簧线代替直导线，增加弯曲寿命 |
| **网状编织套管** | 包裹裸露导线束，提高机械强度 |
| **最小弯曲半径** | 电缆弯折半径不能小于线径的 6 倍 |
| **定期检查** | 每 3-6 个月检查一次活动电缆外观 |

## 八、功率预算与电源选型

### 大鱼TT整机电功率估算

| 组件 | 功率 | 备注 |
|---|---|---|
| 热床 | 270W | 主要功耗 |
| 热端（双挤出） | 100W | 2 × 50W |
| 步进电机（4轴） | 30-50W | 取决于电流设置 |
| 控制板 + 散热风扇 | 10-20W | |
| 腔体排风扇 | 5-10W | 120mm 风扇 |
| **满载合计** | **~450W** | |

### 电源选型公式

```
电源功率 = (组件最大功率之和) × 1.2 ~ 1.5
```

### 大鱼TT建议

- **24V / 500W 工业电源**（如明伟 LRS-500-24）
- 留 20% 以上余量，避免电源长期满载发热
- 不要用二手电源

## 九、封箱特殊风险

封箱后电气安全额外风险：

1. **散热条件变差** — 风扇堵住后 MOSFET/电源更容易过热
2. **腔内温度升高** — 加速电解电容老化（电解液沸点约 105°C）
3. **塑料燃烧风险** — PLA 260°C 以上自燃，封箱内没有通风

### 封箱电气安全设计要求

- ⚠️ **必须配置腔温传感器**（MAX6675 热电偶 + Adafruit MAX31855 模块），配合固件超温断电
- ⚠️ **MOSFET 和电源模块不能放在封箱顶部**（热量积累区），要放在通风处
- ⚠️ 选用 **高温电容**（105°C rated），不用 85°C 普通电容
- ⚠️ 箱内所有导线用 **阻燃套管**（UL94 V-0 / VW-1 等级）
- ⚠️ **独立烟雾报警器**（光学式，非离子式）：放置在封箱顶部角落

## 十一、大鱼TT电气安全检查清单

- [ ] 固件开启 Thermal Runaway Protection（热端 + 热床 + 腔温）
- [ ] 热敏电阻用高温胶固定，防止松脱
- [ ] 热床 MOSFET 加装散热片，验证工作温度在安全范围
- [ ] 主板到热床回路加玻璃管保险丝（15A）
- [ ] 所有金属外壳接地（保护接地 PE）
- [ ] 电源选型 ≥ 500W（24V），留足余量
- [ ] 腔内安装温度传感器（热电偶），超温联动断电
- [ ] 活动电缆走拖链或用弹簧线，定期检查
- [ ] 选用高温（105°C）电解电容
- [ ] 箱内导线用阻燃套管
- [ ] 独立烟雾报警器安装在封箱附近
