## 一、概述 / Overview

本文档详细定义了大鱼TT 310全封闭机箱底部支架系统的设计规格，包括底部支架、调整脚固定座和机箱脚垫的完整参数，用于指导FreeCAD模型的创建。

底部支架安装于2020框架底部四角，负责连接侧板、底板与框架，同时作为调整脚固定座的安装基座。

## 七、FreeCAD建模指南 / FreeCAD Modeling Guide

### 7.1 底部支架建模步骤

1. 创建Sketch: 40×30mm矩形，R3圆角外轮廓
2. Pad: 15mm高度
3. 创建顶面凹槽Sketch: 减材Pad 3mm深
4. 创建M3沉头孔: 4个角落，Sketch驱动的Pocket
5. 创建侧板连接耳: Sketch驱动的Pad

### 7.2 调整脚固定座建模步骤

1. 创建Sketch: Ø20mm圆，中心Ø12mm圆（减材）
2. Pad: 15mm高度
3. 创建法兰Sketch: Ø26mm圆环
4. Pad法兰: 2mm高度
5. 创建M3沉头孔: 3孔均布，Sketch驱动的Pocket

### 7.3 脚垫建模步骤

1. 创建Sketch: Ø25mm圆，中心Ø12mm圆（减材）
2. Pad: 5mm高度
