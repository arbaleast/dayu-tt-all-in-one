## 一、摄像头选型

### 1.1 推荐方案

| 型号 | 类型 | 特点 | 价格 | 推荐度 |
|------|------|------|------|--------|
| **Mintion Beagle V2** | USB 即插即用 | 专为3D打印机设计，远程监控+延时摄影 | ¥200-300 | ⭐⭐⭐⭐⭐ |
| **罗技 C270** | USB | 免驱、成像好、便宜 | ¥80-120 | ⭐⭐⭐⭐ |
| **小米/米家摄像头** | WiFi | 需破解，接入局域网 | ¥80-150 | ⭐⭐⭐ |
| **萤石 C2** | USB/WiFi | 夜视好，稳定 | ¥100-200 | ⭐⭐⭐ |
| **DIY USB 摄像头** | USB | 可用旧笔记本摄像头 | ¥8-50 | ⭐⭐⭐ |

### 1.2 免驱摄像头要求

- **免驱即插即用**：大多数 UVC (USB Video Class) 摄像头在 Linux 下无需额外驱动
- 推荐使用 **MJPG 格式**摄像头（大多数 USB 摄像头都支持）
- 确认命令：
  ```bash
  ls /dev/v4l/by-id/*
  v4l2-ctl --list-devices
  ```

### 1.3 注意事项

- **树莓派之外的大多数上位机不支持硬件解码**，用 CPU 软件解码，性能不足会卡滞
- 建议分辨率 **640x480@10fps**（Fluidd 默认），性能好的 CPU 可用 **800x600**
- 摄像头 **供电要稳定**，要求 5V@2A 以上
- 安装时确保**没有正在执行打印任务**，避免中断通讯

## 三、Klipper 摄像头配置教程

> 主要参考：
> - B站 `Klipper固件之摄像头配置指南`（BV1RC4y1E7tj）
> - B站 `给大鱼TT加了个摄像头，klipper的兼容性真不错`（BV1q94y177KN）

### 3.1 方式一：Crowsnest 插件（推荐）

Crowsnest 是 Mainsail/Fluidd 专用摄像头插件，推荐使用。

**安装步骤：**

1. 进入 SSH，运行 KIAUH：
   ```bash
   ./kiauh/kiauh.sh
   ```

2. 选择 `4) Advanced` → `Crowsnest` 安装

3. 安装完成后修改配置 `/home/pi/klipper_config/crowsnest.conf`：

   ```ini
   [cam 1]
   mode: mjpeg
   port: 8080
   device: /dev/v4l/by-id/usb-xxx-video-index0
   resolution: 640x480
   fps: 15
   ```

4. 重启服务：
   ```bash
   sudo systemctl restart crowsnest
   ```

### 3.2 方式二：KIAUH 安装 MJPG-Streamer

1. SSH 进入 KIAUH：
   ```bash
   ./kiauh/kiauh.sh
   ```

2. 删除旧 MJPG-Streamer，重新安装

3. 添加摄像头，设置配置：
   - **MJPEG Stream** — 传统方式，被动接收，**不推荐**
   - **MJPEG Adaptive** — Fluidd 主动拉取，推荐 ✅

4. 修改分辨率和帧率，编辑 `~/klipper_config/webcam.txt`：
   ```bash
   camera_usb_options="-r 640x480 -f 10 -y -d /dev/v4l/by-id/usb-Etron_Technology__Inc._USB2.0_Camera-video-index0"
   # -r: 分辨率  -f: 帧率
   ```
5. 重启：
   ```bash
   sudo systemctl restart webcamd
   ```

### 3.3 延时摄影配置（Moonraker-Timelapse）

Moonraker-Timelapse 可以自动截取图片生成延时视频。

**安装：**

通过 Mainsail OS 或 KIAUH 安装 `Moonraker` 和 `Timelapse` 组件。

**配置 `printer.cfg` 中启用：**
```ini
[timelapse]
output_path: /tmp/timelapse/
frame_rate: 30
count_per_layer: False
parkprinting: True
```

**宏命令触发：**
```bash
TIMELAPSE_TAKE_FRAME  # 手动拍照
```

**延时摄影设置注意：**
- 延时摄影宏**不支持密码访问的摄像头**
- 可用高分辨率低帧率：`1600x1200@7fps`

## 四、摄像头走线与固定

### 4.1 推荐安装位置

1. **框架顶部俯拍** — 视野最广，看全局打印状态
2. **打印头侧面** — 看挤出头和首层
3. **热床下方** — 看底部打印效果

### 4.2 支架方案

| 来源 | 链接 |
|------|------|
| B站 - 大鱼TT klicky CNC 支架与停靠坞 | https://www.bilibili.com/video/BV1He411o7Bm/ |

### 4.3 走线建议

- 使用 **尼龙扎带** 固定线材
- 避免线材接触热床、加热棒等发热部件
- 摄像头线束建议留一定余量，防止打印头移动时拉扯

## 五、大鱼TT 具体配置示例

### 5.1 确认摄像头设备

```bash
ls /dev/v4l/by-id/*
# 返回类似：usb-Generic_Webcam_20080401001-video-index0
```

### 5.2 webcam.txt 示例

```ini
[webcamd]
webcam_device=/dev/v4l/by-id/usb-Generic_Webcam_20080401001-video-index0
resolution=640x480
framerate=10
```

### 5.3 Fluidd/Mainsail 中设置

在 Webcam & Timelapse 设置中：
- Stream URL: `/webcam/?action=stream`
- Snapshot URL: `/webcam/?action=snapshot`
- 使用 MJPEG Adaptive 模式

## 八、常见问题

### Q1: 摄像头没有图像
- 检查 `v4l2-ctl --list-devices` 是否识别到设备
- 确认 webcam.txt 中设备路径正确
- 重启 webcamd 服务：`sudo systemctl restart webcamd`

### Q2: 画面卡滞
- 降低分辨率或帧率
- 减少同网段其他设备网络占用
- CPU 性能不足时换用 µStreamer

### Q3: Fluidd 中无法显示
- 确认用的是 MJPEG Adaptive 模式
- 检查 Stream URL 是否正确：`/webcam/?action=stream`

### Q4: 延时摄影无输出
- 检查输出路径是否有写入权限
- 确认 Moonraker 服务正常
- 摄像头不能有密码保护
