## 二、安装步骤

### 2.1 硬件连接

```
摄像头USB线 ──→ SBC（树莓派/小政K6等）USB口
```

- 优先接 **USB 2.0** 接口（蓝色或标注USB2.0）
- 确保 SBC 供电充足，打印时不要同时插其他大电流设备
- 摄像头位置建议：**框架顶部俯拍**，视野覆盖整个热床

### 2.2 确认设备识别

SSH 连接 SBC，执行：

```bash
# 查看识别到的摄像头设备
ls /dev/v4l/by-id/*

# 返回示例（记住这个路径）
# usb-Logitech_Logitech_USB_Headset_12345678-video-index0

# 查看设备详情
v4l2-ctl --list-devices

# 返回示例
# Logitech Webcam C270 (usb-1a1a00000000-a):
#   /dev/video0
```

## 三、软件配置

### 3.1 Crowsnest 配置（推荐，Mainsail/Fluidd）

Crowsnest 是 Mainsail 官方摄像头插件，配置最简单。

**① 修改配置文件**

```bash
sudo nano /home/pi/klipper_config/crowsnest.conf
```

**完整配置示例：**

```ini
[crowsnest]
log_path: /home/pi/logs/crowsnest
log_level: verbose
no_proxy: false

[cam 1]
enable_rtsp: false
mode: mjpeg
port: 8080
device: /dev/v4l/by-id/usb-Logitech_Logitech_USB_Headset_12345678-video-index0
resolution: 640x480
fps: 15
# C270 支持的分辨率：
# 640x480 @ 30fps（默认，推荐）
# 320x240 @ 30fps
# 160x120 @ 30fps
```

**② 重启服务**

```bash
sudo systemctl restart crowsnest
```

**③ 检查状态**

```bash
sudo systemctl status crowsnest
# 应该显示 active (running)
```

---

### 3.2 MJPG-Streamer 配置（备选，KIAUH安装）

如果不用 Crowsnest，用 KIAUH 安装 MJPG-Streamer：

**① 编辑 webcam.txt**

```bash
sudo nano /home/pi/klipper_config/webcam.txt
```

**内容：**

```ini
[webcamd]
webcam_device=/dev/v4l/by-id/usb-Logitech_Logitech_USB_Headset_12345678-video-index0
resolution=640x480
framerate=15
```

**② 重启服务**

```bash
sudo systemctl restart webcamd
```

## 四、Fluidd / Mainsail 网页端配置

### 4.1 Fluidd 设置

1. 打开 Fluidd 网页 → 右下角**设置图标**
2. 进入 **Webcam & Timelapse**
3. 配置如下：

| 设置项 | 值 |
|--------|-----|
| **Stream URL** | `/webcam/?action=stream` |
| **Snapshot URL** | `/webcam/?action=snapshot` |
| **Mode** | `MJPEG Adaptive` ✅ |
| **Rotated** | 根据实际安装方向选择 |

4. 点**保存**，回到主页看是否出图

### 4.2 Mainsail 设置

1. Settings → Webcam
2. 配置同上

## 五、printer.cfg 添加摄像头配置（可选）

如果你想把摄像头图片放进状态页面：

```ini
[board_pins]
aliases:
  ...

# 摄像头状态（可选）
[camera]
device: /dev/v4l/by-id/usb-Logitech_Logitech_USB_Headset_12345678-video-index0
```

> 大多数情况不需要改 printer.cfg，Fluidd/Mainsail 直接读取 webcamd 服务即可。

## 六、延时摄影配置（Moonraker-Timelapse）

### 6.1 安装

通过 KIAUH 安装 Moonraker-Timelapse：
```bash
./kiauh/kiauh.sh
# 选择 4 → 3 安装 Moonraker-Timelapse
```

### 6.2 配置 printer.cfg

```ini
[timelapse]
output_path: /home/pi/klipper_config/timelapse/
frame_rate: 30
count_per_layer: False
parkprinting: True
gcode_standby: TURN_OFF_FAN

[delayed_gcode timelapse_turn_on_fan]
initial_duration: 0
gcode:
  SET_FAN_SPEED FAN=cooling_fan SPEED=0.5
```

### 6.3 使用

打印时自动启用，结束后在 Fluidd → **Timelapse** 页面查看/下载视频。

## 七、摄像头位置建议

### 推荐位置：框架顶部居中，俯拍


## 九、常见问题

### ❌ 摄像头没有图像
```bash
# 检查设备是否识别
ls /dev/v4l/by-id/*

# 检查服务状态
sudo systemctl status crowsnest

# 查看日志
tail -30 /home/pi/logs/crowsnest/crowsnest.log
```

### ❌ 画面卡顿/延迟高
- 降低分辨率到 640x480
- 降低帧率到 10fps
- 检查 SBC 温度，过热会降频

### ❌ Fluidd 显示 "No signal"
- 确认 Stream URL 是 `/webcam/?action=stream`（不是 `/webcam2/`）
- 确认用了 **MJPEG Adaptive** 模式

### ❌ 延时摄影不工作
- 检查 `/home/pi/klipper_config/timelapse/` 目录是否有写入权限
- 摄像头不能有密码保护
