### 🚨 优先级 1：BottomBracketV2 减重优化

**当前状态**：104x 安全系数 → 材料浪费，PETG 耗材成本↑

**分析**：
- 截面 20×12mm，跨度 30mm，弯曲应力仅 **0.48 MPa**
- PETG 抗拉强度 50 MPa → 安全系数 104x，严重过度设计
- 整机 25kg，每角受力 6.2kgf，实际载荷极小

**优化方案**：

| 方案 | 改动 | 预期安全系数 | 减重效果 |
|------|------|------------|---------|
| A：减薄截面 | 20×12mm → **16×10mm** | ~60x（仍充足）| -30% 材料 |
| B：开减轻孔 | 底部开椭圆孔（间隔 8mm）| 维持高安全系数 | -25% 材料 |
| C：蜂格结构 | 内部三角蜂窝替代实心 | ~40x | -40% 材料 |

**推荐**：方案 A + B 结合，即减薄至 16×10mm 同时加减轻孔，兼顾强度与材料经济性。

**预期效果**：
- 打印时间减少 ~25%（每件）
- PETG 耗材节省 ~30%
- 安全系数仍维持 **60x 以上**，远超需求

### 🚨 优先级 2：TopFanMountV2 迷宫密封优化

**当前状态**：152 三角片，迷宫结构较复杂，可能存在密封死角

**分析**：
- 当前迷宫密封圈数不足（仅 1~2 圈）
- 120mm 风扇静压 150Pa → 需要更密集的迷宫齿数才能有效降噪
- 滤网框与风扇罩分离 → 多零件，增加装配复杂度

**优化方案**：

| 方案 | 改动 | 效果 |
|------|------|------|
| A：增加迷宫齿数 | 2 圈 → **4 圈** | 降噪效果 +50% |
| B：一体化滤网框 | 风扇罩与滤网框合并 | 零件数 -1，装配简化 |
| C：密封条直接粘接 | 海绵条直接粘在风扇罩法兰上 | 消除卡扣失效风险 |

**推荐**：方案 A + B 一体化设计，将迷宫齿数增加至 4 圈，同时将滤网框整合进风扇罩主体。

**预期效果**：
- 降噪提升 3~5 dB
- 零件数减少，装配更简单
- 密封更可靠

### 🟡 优先级 4：线缆密封入口 CableEntryRingV2 多合一

**当前状态**：61 三角片，单一线缆通道，缺乏分区

**问题**：
- 电源线（Ø8~10mm）、数据线（USB/网线 Ø5mm）、热端线（Ø3mm）共用一个 Ø20~25mm 大孔
- 大孔小线 = 密封不严，漏气率较高
- 热端风扇线/调平线/温度传感器线混杂

**优化方案**：

设计一个**三通道分隔密封入口**：

```
┌─────────────────────┐
│  [大孔 Ø14mm]       │  ← 电源线（配橡胶圈）
│  [中孔 Ø8mm]        │  ← USB/网线（配橡胶圈）
│  [小孔 Ø4mm × 2]    │  ← 热端线/调平线（泡棉密封）
│  [T-slot卡扣]       │  ← 固定在2020框架
└─────────────────────┘
```

| 通道 | 用途 | 密封方式 |
|------|------|---------|
| 大通道 Ø14mm | 电源线（主电源 3×1.5mm²）| 硅胶圈 + 锁紧螺母 |
| 中通道 Ø8mm | USB / RJ45 网线 | 泡棉塞 |
| 小通道 Ø4mm × 2 | 热端风扇线、NTC 传感器线 | 泡棉塞 |

**预期效果**：
- 密封效果提升，漏气率减少
- 多线缆分类管理，维护更清晰
- 与现有 T-slot 卡扣集成
