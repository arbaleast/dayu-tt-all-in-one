# 故障排查与维护手册

> 更新时间：2026-03-25
> 适用：大鱼TT日常维护与常见问题
> 来源：社区经验汇总

---

## 一、日常维护 Checklist

### 每次打印前
- [ ] 检查平台粘附（PEI 表面清洁）
- [ ] 确认喷嘴无堵塞（观察挤出不间断）
- [ ] 检查线材是否受潮（脆响 = 需烘干）
- [ ] 清理残料（ особенно 喷嘴周围）

### 每 50 小时打印后
- [ ] 清洁导轨和张紧器
- [ ] 检查皮带张力（拇指按压测试）
- [ ] 检查螺丝紧固（尤其 Z 轴和热端）
- [ ] 清理风扇叶片（灰尘堆积会影响散热）

### 每 200 小时打印后
- [ ] 更换喷嘴（黄铜喷嘴磨损会影响精度）
- [ ] 检查轴承润滑（油脂型润滑脂）
- [ ] 深度清洁散热片（压缩空气吹）
- [ ] 校准 E 轴（推进校准测试）

## 二、常见问题分类

### 🔴 打印质量问题

#### 底部翘边/边角翘起
| 可能原因 | 解决方法 |
|---------|---------|
| 平台未调平 | 运行 `BED_MESH_CALIBRATE` |
| 热床温度不够 | ABS 建议 100-110°C |
| 第一层太高 | 降低 Z 偏移（-0.02~0.05mm 微调） |
| 腔温太低 | 封闭打印或加温 |
| 材料受潮 | 60°C 烘干 4-6 小时 |

#### 拉丝（Stringing）
```yaml
# OrcaSlicer 调整
回抽距离：3-5mm（根据材料）
回抽速度：20-30mm/s
裙边类型： Brim（增加粘附）
温度：降低 5-10°C 测试
```

#### 层间错位（Shifting）
| 可能原因 | 解决方法 |
|---------|---------|
| 皮带太松 | 张紧器调整，或换新皮带 |
| 步进电机过热 | 检查驱动芯片温度，加散热片 |
| 喷嘴撞到打印件 | 检查模型底部是否需要支撑 |
| 加速太高 | OrcaSlicer 降低 `max_accel` |

#### 表面振纹（Ringing）
- **主要原因**：输入整形（Input Shaper）未校准
- **解决**：`INPUT_SHAPER_CALIBRATION` 重新校准
- 打印速度降至 80% 测试，确认是否是振动问题

#### 填充与外壳分离
- 降低填充密度（15-20% 足够）
- 检查 `wall order` 设置（外壳先打印）
- 确认喷嘴温度不过高

### 🔴 硬件问题

#### 热端不加热
```bash
# Klipper 检查
HEATER_BAW_TEMP heater_generic-hotend
# 检查温度是否上升

# 检查接线
- 热端加热丝红线 → 正极
- 热端热敏电阻 → 信号（GND）
- 检查接头是否氧化
```

#### 热床不加热
| 可能原因 | 检查方法 |
|---------|---------|
| 接线松动 | 检查 XT60 或 screw terminal |
| 热床线断裂 | 万用表测两端电阻（~1-2Ω 正常） |
| 固件配置错误 | `heater_bed` pin 是否正确 |

#### 步进电机异响/丢步
- 检查驱动电流：`STEPPER_BUZZ motor_name`
- 降低速度测试是否是物理卡顿
- 检查电机供电电压（24V 系统）

#### 风扇不转
```bash
# Klipper 测试
SET_FAN_SPEED FAN=hotend_fan SPEED=1.0
# 有声音 = 风扇OK，问题在控制
# 无声音 = 检查供电或风扇本身
```

#### 拖链卡顿/断裂（大鱼TT已知痛点 ⚠️）
| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 拖链卡顿 | 线缆弯折半径过小 | 增大拖链弯曲半径，检查内部线缆排列 |
| 拖链噪音大 | 拖链与框架碰撞 | 加装橡胶垫或重新布线 |
| 拖链断裂 | 长期使用疲劳 / 线缆过重 | 更换拖链；大鱼TT原版方案线材偏重，考虑加固型号 |
| "线太沉拖链顶不住" | 大鱼TT已知痛点 | 参考 B站「大鱼TT 原版工具头拖链方案 2.0」改装，或换用更耐重型拖链 |

**拖链改装参考：**
- 大鱼TT 原版工具头拖链方案 2.0（B站）
- 大鱼CC 3D打印机拖链安装教程（B站）
- Voron 风格加固型拖链（社区兼容）
```bash
# Klipper 测试
SET_FAN_SPEED FAN=hotend_fan SPEED=1.0
# 有声音 = 风扇OK，问题在控制
# 无声音 = 检查供电或风扇本身
```

### 🔴 软件/固件问题

#### Klipper 连接失败
```
排查顺序：
1. 重启树莓派/主控板
2. 检查树莓派 IP 和端口
3. 检查 moonraker 是否运行：systemctl status moonraker
4. 查看 Klipper 日志：tail -f ~/klipper/logs/klippy.log
```

#### 打印中途失联（Communication Error）
- **主要原因**：树莓派供电不足（特别是带屏幕扩展坞）
- **解决**：换用 5V 3A 以上电源，或从主控板取电
- **临时方案**：`max_extrude_cross_section` 降低防止堵料

#### 床面网格偏差大
```
常见原因：
1. 喷嘴与平台间距不均匀（需先调平）
2. 探针偏移未补偿（Z offset 偏差）
3. 平台本身不平（可用玻璃板改善）

排查命令：
BED_MESH_PROFILE load=default
BED_MESH_CALIBRATE
```

## 三、打印材料问题对照表

| 材料 | 热端温度 | 热床温度 | 腔温 | 注意事项 |
|------|---------|---------|------|---------|
| PLA | 200-220°C | 50-60°C | 常温 | 最容易，低收缩 |
| PETG | 235-250°C | 70-80°C | 30-40°C | 需防堵，回抽调高 |
| ABS | 250-270°C | 100-110°C | 50-60°C | 必须封闭，高收缩 |
| ASA | 250-260°C | 90-100°C | 50-60°C | 户外首选，需封闭 |
| TPU | 220-250°C | 40-60°C | 常温 | 低速（<30mm/s），软料 |

## 四、紧急情况处理

### 喷嘴堵塞
```
处理步骤（从易到难）：
1. 低温推丝：HEAT_NOZZLE 170°C → 手动推入 PLA
2. 快速热拔：260°C 预热 → 快速拧下喷嘴 → 清理
3. 细针通：0.3mm 钻头或喷嘴针疏通
4. 换喷嘴：以上无效则直接更换（建议备用2个）
```

### 平台粘死
- 打印件粘死在平台取不下来：
  1. 加热热床到 70°C（PEI 会软化）
  2. 水平滑出，不要硬掰
  3. 如果是玻璃板，敲击背面脱模

### 堵料识别
- 特征：挤出不连贯，有时出时不不出
- 先查温度，再查回抽，最后拆热端
- PETG 最容易堵（建议用 PTFE 管）

## 五、远程诊断

如遇到难以排查的问题，可以：
1. **截图 Klipper 日志**：`tail -100 ~/klipper/logs/klippy.log`
2. **导出配置**：`Klipper_config` 备份
3. **拍照/录像**：问题现场
4. **B站/论坛搜索**：`大鱼TT + 具体问题` 组合搜索
