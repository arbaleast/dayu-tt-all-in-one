# 大鱼TT 封箱散热 & BOSSAC 热床改装指南

> 更新时间：2026-03-24
> 适用：大鱼TT / 打印 ABS/ASA 必须改装

---

## 一、封箱的必要性

| 耗材 | 是否需要封箱 | 原因 |
|------|------------|------|
| PLA | ❌ 不需要 | 打印温度低，不挑环境 |
| PETG | ⚠️ 建议 | 需防风，防变形 |
| **ABS** | ✅ 必须 | 80°C+ 热床，温差大翘曲 |
| **ASA** | ✅ 必须 | 同 ABS，抗 UV |

**不封箱打印 ABS 的后果：**
- 底角翘曲脱落
- 模型开裂
- 层间粘合差

## 二、BOSSAC 热床改装

### 2.1 什么是 BOSSAC 热床？

**BOSSAC** = Board One Stupid Simple Advanced Controller

一种基于 **STM32** 的热床控制器，支持：
- 24V/12V 兼容
- 过温保护
- 温度传感器闭环控制
- 直接替换原配热床

### 2.2 热床选型对比

| 类型 | 电压 | 功率 | 加热速度 | 推荐度 |
|------|------|------|---------|--------|
| 硅胶加热板 | 24V | 150-300W | 中等 | ⭐⭐⭐⭐ |
| 磁吸软玻璃 | 24V | 200W+ | 快 | ⭐⭐⭐⭐⭐ |
| BOSSAC 热床 | 24V | 200W+ | 快 | ⭐⭐⭐⭐ |
| 玻纤板 + 加热片 | 24V | 200W | 快 | ⭐⭐⭐ |

### 2.3 热床尺寸选择

| 大鱼TT 尺寸 | 适用热床尺寸 |
|------------|-------------|
| 235mm 版 | 235×235mm（M6 螺丝孔位）|
| 310mm 版 | 310×310mm（M8 螺丝孔位）|

### 2.4 热床安装注意事项

⚠️ **螺丝孔位必须精确匹配**
- M6 版本：235×235mm，6mm 螺丝孔
- M8 版本：310×310mm，8mm 螺丝孔

⚠️ **电源要求**
- 24V 电源，电流 ≥ 10A
- 功率密度不要超过 0.8W/cm²（安全）

## 三、腔体散热方案

### 3.1 风扇配置原则

> "风只需要在打印的时候不发生跳动即为合格"

**核心思路：**
- 主动加热腔体（保温）
- 主动换气（排湿排味）
- 温控自动启停

### 3.2 风扇类型选择

| 类型 | 规格 | 适用场景 |
|------|------|---------|
| 5015 涡轮风扇 | 50×50×15mm | 腔体主动换气 |
| 4010 静音风扇 | 40×40×10mm | 小型腔体 |
| 6020 强排风扇 | 60×60×20mm | 大腔体高流量 |
| 80mm 机箱风扇 | 80×80×25mm | 过滤+换气 |

### 3.3 推荐方案：腔体过滤换气系统

**参考：** K2 Plus 封箱方案

| 组件 | 规格 | 作用 |
|------|------|------|
| 初效滤网 | G3 过滤棉 | 过滤大颗粒 |
| HEPA 滤网 | H12 级 | 过滤 PM2.5 |
| 活性炭滤芯 | 椰壳活性炭 | 吸附 ABS 气味 |
| 腔体排风扇 | 6020 24V | 主动换气 |
| 腔体循环风扇 | 4010 24V | 腔内热风循环 |

**温控逻辑（Klipper 宏）：**
```ini
[fan]
# 腔体风扇，温度超过设定值自动启动
# 建议腔体温度 > 45°C 时启动换气
```

### 3.4 腔体循环风扇安装

**作用：**
- 消除腔内温度分层
- 让加热器热量均匀分布
- 减少模型局部冷却导致的翘曲

**安装位置：** 热床下方或框架角落

**配置示例：**
```ini
[heater_fan chamber_fan]
pin: PC13
max_power: 1.0
kick_start_time: 0.5
heater: heater_bed
heater_temp: 45.0
fan_speed: 0.5
```

## 五、温控系统配置（Klipper）

### 5.1 腔体温度传感器

```ini
[temperature_sensor chamber]
sensor_type: NTC 100K
sensor_pin: arduino:analog 1
min_temp: 0
max_temp: 120
```

### 5.2 腔体风扇自动控制

```ini
[idle_timeout]
timeout: 600

[respond]
```

### 5.3 温控宏

```ini
[gcode_macro AUTO_CHAMBER]
gcode:
  {% if printer.heater_bed.temperature > 80 %}
  SET_FAN_SPEED FAN=chamber_fan SPEED=0.5
  {% endif %}
```

## 六、封箱打印注意事项

### 6.1 ABS 打印参数

```ini
热床温度: 100-110°C
腔体温度: 45-60°C
喷嘴温度: 250-270°C
冷却风扇: 20%（后期）
打印速度: 50-80mm/s
```

### 6.2 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 底角翘曲 | 温差大、热床温度不足 | 提升热床至110°C，加 Brim |
| 拉丝 | 腔体温度过高 | 降低腔温至50°C以下 |
| 有异味 | 换气不足 | 加装排风扇+活性炭 |
| 玻璃门起雾 | 内外温差大 | 使用防雾涂层玻璃 |

---

## 七、ABS 打印技巧

1. **先预热**：打印前 10 分钟开启热床和腔体加热
2. **Brim 加大**：ABS 建议 Brim 8mm 以上
3. **封闭环境**：确保腔体密封，减少气流扰动
4. **底部加固**：首层 3-5 层加粗
5. **顶部保湿**：打印完成后不要立即开箱，冷却后再取
