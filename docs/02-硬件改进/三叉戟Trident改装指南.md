# 大鱼TT → Voron Trident 改装指南

> 更新时间：2026-03-24
> 改装难度：⭐⭐⭐⭐（较复杂）
> 改装收益：支持多色打印、结构更稳固

---

## 一、什么是三叉戟（Trident）？

**Voron Trident** 是 Voron 0 的放大版，特点：
- **3Z 轴**（三根丝杆同步），打印更稳定
- 比 Voron 2.4 更容易组装
- 稳定性好、可靠性高
- 支持多色打印（升级到 4 色）

> 大鱼 TT 改装三叉戟 = 把原本的单 Z 轴改成三 Z 轴 + 换工具头

---

## 二、改三叉戟的好处

| 优点 | 说明 |
|------|------|
| 🔧 更稳定 | 三根丝杆同步，Z 轴抖动更小 |
| 🎨 多色打印 | 升级到 2-4 色 AMS/eratank 方案 |
| ⚡ 更快 | 三轴同步升降，减少等待时间 |
| 🔇 更静音 | 丝杆润滑后比原版更顺滑 |

## 五、大鱼TT 改装步骤（简化版）

### 第一步：改 X 轴

1. 换碳纤维 X 轴（更轻、更稳）
2. 安装 SB 挤出头
3. 布线走 CAN 总线

### 第二步：改 Z 轴

1. 安装三轴连接件（原 TT 框架需扩孔）
2. 安装 3 根 8mm 丝杆
3. 同步带轮连接

### 第三步：固件配置

更新 printer.cfg：

```ini
# 三 Z 轴配置
[stepper_z]
step_pin: PC6
dir_pin: PA14
enable_pin: !PA15
rotation_distance: 8

[stepper_z1]
step_pin: ...  # 第二Z轴
...

[stepper_z2]
step_pin: ...  # 第三Z轴
...
```

### 第四步：Klipper 调平

```bash
# 三轴独立调平
Z_ENDSTOP_CALIBRATE

# 网床校准
BED_MESH_CALIBRATE

# 共振测试
RESONANCES AXIS=Y
```

## 九、注意事项

⚠️ **框架尺寸改动注意**
- 改了铝型材尺寸会影响打印面积和高度
- 案例：310行程方案打印高度只剩260mm
- 建议先不改框架，只改 Z 轴传动

⚠️ **CAN 总线优先**
- 改装后设备多，CAN 总线能大幅简化接线
- 强烈建议同时升级 SB2040 CAN 板

⚠️ **固件调试复杂**
- 三轴同步需要单独调参
- 建议有 Klipper 经验后再尝试
