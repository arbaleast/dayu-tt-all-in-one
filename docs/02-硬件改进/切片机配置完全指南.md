## 二、大鱼TT 基础打印机参数

### 2.1 机器参数

| 参数 | 值 |
|------|-----|
| 结构 | CoreXY |
| 打印尺寸 | 235×235mm / 310×310mm |
| 热床尺寸 | 235×235mm（M6螺丝）/ 310×310mm（M8螺丝）|
| Z 轴 | 单轴 或 三轴（改装后）|
| 喷嘴直径 | 0.4mm（默认）|
| 最大加热温度 | 热端 300°C / 热床 120°C |

### 2.2 关键切片设置

```
打印机 → 机器设置：
- 床尺寸 X: 235 / Y: 235
- 打印高度: 250 / 310
- 起点 X: 0 / Y: 0
- 起点 Z: 0
- 喷嘴直径: 0.4
- 挤出头数: 1（单头）/ 2-4（多色改装后）
```

## 三、OrcaSlicer 打印机配置步骤

### 3.1 新建打印机

1. 打开 OrcaSlicer → 打印机 → 添加打印机
2. 选择 **Custom printer**（自定义打印机）
3. 命名：`大鱼TT-235` 或 `大鱼TT-310`

### 3.2 机器设置参数

**运动设置：**
```ini
- 旅行速度: 150 mm/s
- 打印速度参考值: 60 mm/s
```

**归位顺序：**
```ini
- X/Y归位: 左后角（min_x, min_y）
- Z归位: 顶部（max_z）
```

**搭梯：**
```ini
- 启用: 是
- 宽度: 3 mm
- 高度: 1 mm
```

## 五、速度与加速度参数（Klipper 优化）

### 5.1 OrcaSlicer 内核速度设置

```ini
[速度]
默认打印速度: 60 mm/s
默认移动速度: 150 mm/s
默认归位速度: 20 mm/s
Z抬升速度: 12 mm/s

[加速度]
默认打印加速度: 1500 mm/s²
默认旅行加速度: 3000 mm/s²
默认归位加速度: 1000 mm/s²

# 大鱼TT 建议加速度（带 Klipper）：
- 打印: 1500-2500 mm/s²
- 空程: 3000-4000 mm/s²
- 归位: 1000 mm/s²
```

### 5.2 针对高速打印的 OrcaSlicer 技巧

> 来源：B站 `OrcaSlicer 十大问题 偏好设置`

1. **"按层高度重新分段"** 不要随意勾选，会影响打印质量
2. **间隙填充** 启用，减少缩水痕
3. **压力平滑** 启用，Klipper 机器推荐

## 六、支撑与悬空结构

### 6.1 支撑设置

```ini
支撑类型:
- 自动: Tree（树形支撑），对复杂模型更好
- 手动: 画笔工具精确定位

支撑密度: 10-15%
支撑界面密度: 20-30%
支撑与模型间距: 0.2mm
```

### 6.2 悬空角度

```ini
- 悬空角度阈值: 50-55°（大于此角度自动加支撑）
- 顶层悬空: 使用更多底部层
```

## 七、网床补偿与首层

### 7.1 首层设置

```ini
首层:
- 打印速度: 50% 正常速度
- 首层高度: 0.2mm（或者层高的 200%）
- 首层收缩补偿: 是

热床粘附:
- 类型:  Brim（边缘）
- Brim 宽度: 5-8mm（ABS建议8mm）
- Raft: 仅大面积/ABS/ASA 使用
```

### 7.2 Klipper 网床补偿集成

在 OrcaSlicer 中：
- 切片时选择 **"使用网床补偿"**
- OrcaSlicer 会读取 `printer.cfg` 中的 `[bed_mesh]` 配置
- 确保 Klipper 已运行 `BED_MESH_CALIBRATE`

## 九、OrcaSlicer 十大技巧

> 来源：B站 `3D打印从菜鸟到专家 OrcaSlicer十大问题`

1. **"按层高度重新分段"** 不要勾选，会影响精度
2. **"间隙填充"** 建议开启，减少缩水痕
3. **压力平滑** 对 Klipper 机器很有用
4. **多色打印** OrcaSlicer 支持 Bambu AMS 格式
5. **有机支撑** 树形支撑比柱形更好用
6. **减少过挤** PLA 建议 -1% to -2%
7. **回抽优化** 近程挤出建议关闭或极小值
8. **封口层** 底部封口层数影响底板强度
9. **网床导入** OrcaSlicer 支持 Klipper 网床数据
10. **自定义 Gcode** 善用切片后处理脚本
