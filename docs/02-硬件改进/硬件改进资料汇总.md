## 一、结构件升级

### 1.1 重制版 XY 结构件（推荐 ⭐⭐⭐⭐⭐）
- **来源：** [MakerWorld - 大鱼TT重制版XY结构件](https://makerworld.com.cn/zh/models/1586964-da-yu-ttzhong-zhi-ban-xyjie-gou-jian)

## 三、探针系统

### 3.1 Klicky 探针（推荐 ⭐⭐⭐⭐⭐）
- **来源：** [GitHub - tanaes/whopping_Voron_mods](https://github.com/tanaes/whopping_Voron_mods/tree/main/pcb_klicky)
- **解决的问题：** 接近开关热飘严重，首层打印不稳定
- **注意：** 停靠坞使用 MG90s/SG90 舵机，**不要用 MG996**（电流太大，主板带不动）
- 舵机方案需要单独配置 Klipper

### 3.2 接近开关接线注意（八爪鱼主板）
- **来源：** [GitHub - rushairer/bigfish-tt-extentions](https://github.com/rushairer/bigfish-tt-extentions)
- 八爪鱼 1.1 主板：信号引脚和 GND 之间电压低，**NPN 接近开关不能直接用**
- 解决方案：
  1. 用 **PNP 接近开关**
  2. 或 NPN 接近开关信号引脚**接上拉电阻**
  3. 2023年10月更新：24V NPN 接近开关也可使用，接线方法：正负极接电源，信号PIN接 Probe 信号PIN即可
- **推荐：** 5V NPN 常闭接近开关，1.1 和 Pro 板都可用，Pin 前面加 `~`

### 4.2 主板选择
- **来源：** [rushairer/bigfish-tt-extentions](https://github.com/rushairer/bigfish-tt-extentions)

## 十、推荐改装路线（按优先级）

### 第一阶段：低成本高收益（¥300-500）
1. **Klicky 探针** - 自动调平，告别手动调床
2. **Klipper 固件** - 输入整形、网床补偿，打印质量飞跃
3. **PEI 平台** - 附着力好，取模方便
4. **风扇风道优化** - 改善悬垂和桥接

### 第二阶段：性能提升（¥600-1000）
1. **SB 头近程挤出** - 减少堵料，提升挤出精度
2. **主板升级** - 八爪鱼/SB2040 CAN板
3. **42 闭环电机** - 降低噪音、提高精度

### 第三阶段：完善优化（¥1000-2000）
1. **重制版 XY 结构件** - 双边固定+F695轴承，可靠性升级
2. **双 Z 轴** - 减少层纹
3. **全包外壳 + BOSSAC 热床** - ABS/ASA 打印必备
