## 一、皮带张紧器的重要性

皮带张紧不够会导致：
- **跳齿**：打印出现缺线、层纹
- **丢步**：打印尺寸偏小
- **噪音**：皮带抖动产生异响

皮带张紧合适的标准：
> "皮带只需要在打印的时候不发生跳动即为合格"

## 三、皮带张紧调节技巧

### 3.1 判断张紧程度

**方法一：** 用手指按压皮带中间
- 正常：下沉 1-2mm
- 太松：下沉 > 3mm，容易跳齿
- 太紧：几乎不下沉，加速轴承磨损

**方法二：** 听声音
- 张紧合适：打印时机箱安静
- 太松：打印时有"嗒嗒"声
- 太紧：有机率崩齿

### 3.2 调节步骤

1. 先松开张紧轮固定螺丝
2. 调整张紧螺丝（或弹簧）
3. 边调边按压测试，找到最佳点
4. 锁紧固定螺丝
5. 打印测试件验证

## 六、皮带选购指南

| 参数 | 大鱼TT 规格 |
|------|------------|
| 型号 | 2GT（2mm 齿距）|
| 宽度 | 6mm（XY 轴）|
| 长度 | 根据框架尺寸计算 |

**计算公式：**
```
皮带长度 ≈ 2 × (框架宽度 + 框架高度) + 10mm（余量）
```

**购买建议：**
- 选 **进口品牌**（ Gates 、Megadyne ）更耐用
- 国产性价比高（ process 、 CHC ）

## 七、大鱼TT 张紧器改装案例

### 7.1 Voron 皮带张紧器移植

来源：B站 `大鱼TT 改装` 系列

**步骤：**
1. 打印 Voron 0 张紧器支架
2. 购买 GT2 6mm 皮带轮（16-20齿）
3. 替换原版惰轮
4. 调节张紧螺丝

### 7.2 CNC 铝合金张紧器

**优点：**
- 更精确的张紧调节
- 免工具调节设计
- 颜值更高

**获取方式：**
- 购买淘宝 CNC 加工
- 群友共享加工件
