### 大鱼TT 简介

大鱼TT是一款开源的CoreXY结构3D打印机，由大鱼DIY设计。具有高速、高精度、可扩展性强等特点，是DIY 3D打印的热门选择。

**主要规格**:
- 打印尺寸: 235×235×250mm 或 310×310×310mm
- 结构: CoreXY
- 框架: 2020/2040铝型材
- 固件: Klipper/Marlin

## 📖 新手指南

### 推荐改装顺序

#### 第一阶段：低成本高收益 (预算 ¥300-500)
1. **[Klicky探针](docs/00-入门指南/1-硬件升级方案.md)** - 自动调平，告别手动调床
2. **[Klipper固件](docs/固件配置/Klipper基础配置.md)** - 输入整形、网床补偿，打印质量飞跃
3. **[PEI平台](docs/00-入门指南/1-硬件升级方案.md)** - 附着力好，取模方便
4. **[风扇风道优化](docs/00-入门指南/1-硬件升级方案.md)** - 改善悬垂和桥接质量

#### 第二阶段：性能提升 (预算 ¥600-1000)
1. **[挤出机升级](docs/00-入门指南/1-硬件升级方案.md)** - 泰坦/拓竹挤出机，减少滑丝
2. **[主板升级](docs/00-入门指南/1-硬件升级方案.md)** - 八爪鱼/GEMINI，更多功能
3. **[电源升级](docs/00-入门指南/1-硬件升级方案.md)** - 24V系统，大功率热床

#### 第三阶段：完善优化 (预算 ¥1000-2000)
1. **[全包外壳](docs/01-封箱方案/)** - 打印ABS/ASA必备
2. **[双Z轴改造](docs/00-入门指南/1-硬件升级方案.md)** - 减少层纹
3. **[更多细节优化](docs/00-入门指南/1-硬件升级方案.md)**

## ⚠️ 安全注意事项

- 改装前务必断电
- 12V和24V设备不能混接
- 大功率加热设备注意散热
- 定期检查接线是否松动
