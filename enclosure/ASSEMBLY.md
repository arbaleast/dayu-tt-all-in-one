# 大鱼TT 310 全封闭机箱 装配指南 / Assembly Guide

> 版本: 1.0.0
> 日期: 2026-03-25
> 预计装配时间: 3-5 小时

---

## 准备工作 / Preparation

### 工具清单

- [ ] 内六角扳手套装 (M2/M3/M4/M6)
- [ ] 十字螺丝刀
- [ ] 锉刀或砂纸 (去毛刺)
- [ ] 卷尺 / 卡尺
- [ ] 硅胶密封条安装工具 (或指甲)
- [ ] 清洁布

### 零件检查

按 `BOM.md` 清点所有零件，确认数量后再开始。

---

## 装配顺序 / Assembly Order

> 建议顺序：底部框架 → 侧板 → 底板 → 顶板 → 门板 → 电气连接

---

## 第一步：底部支架安装

### 1.1 安装底部支架 (B-01)

```
位置: 2020框架底部四角
数量: 4个
螺丝: M3×10 沉头 ×16 (每角4个)
```

**步骤：**
1. 清洁框架底部 T-slot
2. 将底部支架放入框架底部凹槽
3. 对准侧面固定孔
4. 拧入 M3×10 沉头螺丝（对角拧紧）
5. **扭矩**: 0.3-0.5 N·m，不要过紧

### 1.2 安装脚垫安装座 (B-02)

```
位置: 底部支架中央
数量: 4个
螺丝: M3×8 沉头 ×12 (每座3个)
```

**步骤：**
1. 将脚垫安装座放入底部支架凹槽
2. 对准 M3 螺丝孔
3. 拧入 M3×8 螺丝
4. 检查 Ø12 通孔是否居中

### 1.3 安装脚垫 (B-03)

```
位置: 脚垫安装座下方
数量: 4个
螺丝: M6×20 调整脚 ×4
```

**步骤：**
1. 将 TPU 脚垫放入安装座
2. 旋入 M6 调整脚
3. 调整高度（建议高出机箱底面 5-10mm）

---

## 第二步：侧板预装

### 2.1 安装侧板固定夹 (B-05)

```
数量: 16个 (每侧4个)
位置: 框架侧边 X4 / 前后 Y8
```

**步骤：**
1. 将 T 型螺母放入框架 T-slot
2. 将固定夹滑入 T-slot 顶部
3. 对准螺丝孔，拧紧 M3 螺丝

### 2.2 安装角码连接器 (B-06)

```
数量: 12个
位置: 框架节点
```

### 2.3 安装线缆入口圈 (B-14)

```
数量: 2个
位置: 侧板穿线孔
螺丝: M3×8 ×4
```

---

## 第三步：底板安装

```
尺寸: 510×490×5mm
螺丝: M3×10 沉头 ×8
密封条: P型 3×3mm
```

**步骤：**
1. 在底板边缘贴 P 型密封条（两侧）
2. 将底板放入框架底部（支架上方）
3. 对准 4 个脚垫安装孔
4. 对角拧紧固定螺丝
5. 检查密封条是否被均匀压紧

---

## 第四步：侧板安装

```
左/右侧板: 510×760×3mm
密封条: P型 3×3mm (底部)
```

**步骤：**
1. 在底板与侧板接缝处贴密封条
2. 将侧板小心放入框架（从底部向上）
3. 确认密封条不被挤压移位
4. 在顶部固定夹处拧紧螺丝
5. 检查板面垂直度

---

## 第五步：顶板安装

```
尺寸: 510×490×3mm
风扇孔: 120×120mm (排风口)
螺丝: M3×10 沉头 ×8
密封条: P型 3×3mm
```

**步骤：**
1. 安装顶部风扇罩 (B-04)
   - 将风扇放入罩内
   - 固定 M3 螺丝 ×4
2. 贴密封条（四周边缘）
3. 将顶板放入框架
4. 对角拧紧固定螺丝

---

## 第六步：后板安装

```
尺寸: 490×760×3mm
螺丝: M3×10 沉头 ×8
```

**步骤：**
1. 贴密封条
2. 从上向下卡入框架
3. 固定四周螺丝

---

## 第七步：前门板安装

```
尺寸: 490×740×5mm
铰链: 3对
磁铁: 6个
```

### 7.1 安装铰链 (H-06)

```
位置: 左侧板
铰链数量: 3个 (上/中/下)
螺丝: M3×8 ×12
```

**步骤：**
1. 将铰链座对准侧板标记位置
2. 拧入 M3 螺丝（不要完全紧固）
3. 挂上门板测试开合角度
4. 调整后最终拧紧

### 7.2 安装磁铁 (H-07)

```
位置: 门板右侧
数量: 6个 (3对上/中/下)
安装座: B-10 磁铁固定座
```

**步骤：**
1. 在门板右侧预钻位置放入磁铁固定座
2. 放入钕磁铁（极性一致，面向外）
3. 对应侧板位置放入铁片安装座
4. 轻轻合上门板测试吸附效果
5. 调整位置后最终固定

### 7.3 安装门把手 (B-11)

```
位置: 门板正面
螺丝: M3×8 ×2
```

---

## 第八步：密封条安装

### 密封条安装顺序

1. **P型密封条** (3×3mm) — 顶板/底板与侧板/后板接缝
2. **D型密封条** (5×5mm) — 门框四周

### 安装技巧

```
技巧1: 从角落开始，边压边装
技巧2: 用指甲或塑料刮刀压入槽中
技巧3: 留2-3mm 膨胀余量
技巧4: 转角处45度斜切拼接收尾
```

---

## 第九步：风扇安装

### 排风扇

```
位置: 顶板 (120mm 孔)
风扇: Noctua NF-A12×1 或等效
螺丝: M3×8 ×4
```

**步骤：**
1. 将风扇放入顶部风扇罩
2. 注意风向标记（应向机箱外排风）
3. 固定4角螺丝
4. 连接 PWM 电源线

### 过滤棉

```
位置: 风扇进风侧
规格: 120×120×3mm
```

---

## 第十步：电气连接

### 风扇电源

| 线路 | 连接 | 说明 |
|------|------|------|
| 红色 | 12V/24V | 电源正极 |
| 黑色 | GND | 电源负极 |
| 蓝色 | PWM | 调速信号 |
| 黄色 | TACH | 转速反馈（可选） |

### 温控配置

参考 `klipper-fan-config.md` 配置 Klipper 温控风扇。

## 装配后检查 / Post-Assembly Checklist

- [ ] 所有螺丝已拧紧（不要过紧）
- [ ] 密封条完整，无缝隙
- [ ] 门板开合顺畅
- [ ] 磁铁吸附可靠
- [ ] 风扇转向正确（向外排风）
- [ ] 线缆穿孔密封
- [ ] 清理内部异物
- [ ] 3D打印件无毛刺（安全门板开合）

---

## 调试运行 / Commissioning

1. **密封测试**: 关上门，运行 30 分钟，观察腔体温升
2. **风扇测试**: 测试各档位转速
3. **滤网检查**: 打印 1-2 小时后检查滤网积灰
4. **噪音检查**: 确认无共振噪音

---

## 维护 / Maintenance

| 项目 | 周期 | 操作 |
|------|------|------|
| 过滤棉清洁 | 每 50 小时 | 水洗晾干 |
| 密封条检查 | 每 3 个月 | 检查弹性 |
| 风扇清理 | 每 100 小时 | 毛刷清扫 |
| 螺丝紧固 | 每 6 个月 | 检查松动 |

---

## 安全注意

⚠️ **高温风险**: 打印时机箱内部温度可达 50-60°C，谨防烫伤
⚠️ **门板承重**: 亚克力板易碎，避免大力撞击
⚠️ **PETG 毒性**: 打印时确保通风或使用过滤系统
⚠️ **电气安全**: 风扇电源接线检查绝缘
