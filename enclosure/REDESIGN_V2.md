## 四、设计亮点

### 4.1 参数化代码

所有 3D 打印件由 Python 脚本生成（`stl_writer.py`），核心参数集中在一个 `P` 字典中：

```python
P = {
    'slot_w': 6.2,      # T-slot 槽宽
    'slot_d': 1.2,      # T-slot 槽深
    'bb_w': 30.0,       # 底部支架宽度
    'bb_h': 12.0,       # 底部支架高度
    'm3_drill': 2.9,    # M3 螺丝孔
    ...
}
```

**调整方法**: 修改 `P` 字典中的数值 → 重新运行脚本 → 生成新 STL

### 4.2 快拆化设计

| 连接方式 | 改进前 | V2 改进 |
|---------|--------|---------|
| 侧板固定 | 全部螺丝 | **卡扣 + 少量螺丝备份** |
| 门板固定 | 螺丝 | **磁吸快拆 + 3铰链** |
| 滤网更换 | 拆螺丝 | **磁吸框架** |
| 脚垫 | 螺丝固定 | **卡入支架** |
| 风扇罩 | 4×螺丝 | **卡扣 + 密封一体化** |

### 4.3 功能集成

- **BottomBracketV2**: 支架 + 线缆通道 + 密封 ledge 三合一
- **FootMountV2**: 安装座 + 减震腔 + 防呆凸台 三合一
