## 一、打印机本体关键尺寸 / Machine Critical Dimensions

### 1.1 打印规格 / Print Specifications

| 参数 | 数值 | 说明 |
|------|------|------|
| 成型尺寸 (X×Y×Z) | **310×310×310mm** | 厂家标称最大打印体积 |
| 层高范围 | 0.05–0.30mm | 推荐0.12–0.20mm |
| 喷嘴直径 | 0.4mm (标准) | 可选0.2/0.6/0.8mm |
| 耗材直径 | 1.75mm | PLA/ABS/PETG/ASA等 |
| 最大打印速度 | ~150mm/s | Klipper输入整形加持 |

### 1.2 机体外形尺寸 / Machine Outer Dimensions

> ⚠️ 以下为估算值，基于235版本等比例推算 + 2040型材框架结构  
> 实测数据待后续更新

| 尺寸方向 | 估算值 | 说明 |
|---------|--------|------|
| **X方向 (左右/幅宽)** | ~470mm | 沿型材框架横向，235版本~450mm |
| **Y方向 (前后/进深)** | ~450mm | 沿型材框架纵向，235版本~445mm |
| **Z方向 (高度)** | ~720mm | 底座~150mm + 打印区~310mm + 顶框架~260mm |

**X方向推算逻辑:**
- 235版本X方向型材长度 ≈ 390mm → 外形~450mm
- 打印幅面从235增至310，增加75mm
- X方向型材长度 = 390 + 75 = 465mm → 外形~470mm

**Y方向推算逻辑:**
- 235版本Y方向型材长度 ≈ 390mm → 外形~445mm  
- Y方向承载打印平台，型材略短
- Y方向型材长度 ≈ 380mm → 外形~450mm

**Z方向推算逻辑 (分三层):**

| Z轴层级 | 高度 | 说明 |
|--------|------|------|
| 底部框架 + 电子件 + 热床 | ~150mm | 包含脚垫、主板区域 |
| 打印有效区域 | 310mm | Z轴丝杆行程 |
| 顶部框架 + 龙门架 + 电机 + 同步轮 | ~260mm | CoreXY顶部结构 |
| **合计** | **~720mm** | 完整机身高度 |

| 参数 | 数值 |
|------|------|
| 型材规格 | **2020 (20mm×20mm, 槽宽6mm)** |
| X轴型材数量 | 2根 (前后框架横梁) |
| Y轴型材数量 | 2根 (左右框架横梁) |
| Z轴立柱数量 | 4根 (四角立柱) |

## 三、散热与通风 / Cooling & Ventilation

> 封箱后仓温可达40–50°C，适合ABS/ASA打印
